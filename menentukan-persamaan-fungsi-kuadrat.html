Jawab:

f(x) = (1/5)x² + (3/5)x + 2/5

Penjelasan dengan langkah-langkah:

f(x) = ax² + bx + c

a. (-1,0)

   0 = a(-1)² + b(-1) + c

   0 = a - b + c

b. (-2,0)

   0 = a(-2)² + b(-2) + c

   0 = 4a - 2b + c

c. (3,4)

   4 = a(3)² + b(3) + c

   4 = 9a + 3b + c

tinggal kita cari a, b, c

4a-2b+c = 0

 a-  b+c = 0  -

   3a - b = 0

9a + 3b + c = 4

 a -    b + c = 0 -

     8a + 4b = 4

       2a + b = 1

3a - b = 0

2a + b = 1  +

     5a = 1

       a = 1/5

       b = 1 - 2/5 = 3/5

       c = b - a = 3/5 - 1/5 = 2/5

jadi persamaannya adalah

f(x) = (1/5)x² + (3/5)x + 2/5

Penyelesaian:

Tipot sumbu x:

x²-4x

metode faktorisasi:

(x-4)(x+0)

Maka tipot nya adalah:

(4,0) dan (0,0)

----------------------------------------

Tipot sumbu y:

Untuk mencari nya harus x=0

Maka:

f(0)=x²-4x

f(0)=(0)²-4(0)

f(0)=0

Maka tipot sumbu y nya adalah:

(0,0)

----------------------------------------

Sumbu simetri:

x=-b/2a

x=-(-4)/2(1)

x=4/2

x=2

----------------------------------------

Nilai optimum:

b²-4ac/-4a

=4²-4(1)(0)/-4

=16-0/-4

=16/-4

=-4

----------------------------------------

Maka koordinat titik puncak nya adalah (2,-4)

36a + 6b = -3

28a = 14b

b = 2a

36a + 6(2a) = -3

36a + 12 a = -3

48a = -3

a = -1/16

b = -1/8

fungsi kuadrat melalui titik

(-1, 1), (0, 4), (1,5)

4 = a(0)² + b(0) + c

c = 4........................(1)

y = ax² + bx + c ......melalui titik (-1, 1)

1 = a(-1)² + b(-1) + c

a - b + c = 1

a - b + 4 = 1

a - b = 1 - 4

a - b = - 3

b = a + 3.................(2)

y = ax² + bx + c ......melalui titik (1, 5)

5 = a(1)² + b(1) + c

a + b + c = 5..........(3)

a + (a + 3) + 4 = 5

2a + 7 = 5

2a = 5 - 7 = - 2

a = - 1

b = a + 3 = - 1 + 3

b = 2

jadi : a = -1, b = 2, c = 4

Penjelasan dengan langkah-langkah:

Sehingga titik potong dengan sumbu y terletak pada titik (0, c).
