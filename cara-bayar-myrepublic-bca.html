23. hitunglah besar sudut BCAdengan caranya!​


Jawab:

75°

Penjelasan dengan langkah-langkah:

DAC= ACD
35= ACD

180= DAC+ACD+ ADC
180= 35+35+ADC
180= 70+ADC
ADC= 110

BDC= 180-110
BDC= 70
CBD= 70

DCB= 180-(70+70)
DCB= 180-140
DCB= 40

BCA= DCB+ ACD
BCA= 40+35
BCA= 75

Materi : Garis dan Sudut

Sudut ACD

Sudut ini adalah kaki segitiga sama kaki ACD yakni besar sudutnya sama dengan besar sudut dibawah kaki satunya [ ACD = 35° ]

Sudut ADC

ADC = 180° - 35° - 35°

ADC = 180° - 70°

ADC = 110°

Sudut BDC

BDC = 180° - 110°

BDC = 70°

Sudut CBD

Sudut ini adalah kaki segitiga sama kaki BCD yakni besar sudutnya sama dengan besar sudut dibawah kaki satunya [ CBD = 70° ]

Sudut BCD

BCD = 180° - 70° - 70°

BCD = 180° - 140°

BCD = 40°

Langsung Saja

BCA = BCD + ACD

BCA = 40° + 35°

{ BCA = 75° }

28. besar BCA adalahpake cara ya!!!​


Jawab:

70°

Penjelasan dengan langkah-langkah:

Diketahui:

BAC = 42°

CBD = 112°

Ditanya: berapa besar BCA?

CBA = 180° - CBD = 180° - 112° = 68°

*sudut total segitiga = 180° maka:

BCA = 180° - BAC - CBA

BCA = 180° - 42° - 68° = 70°

Jadi, besar BCA adalah 70°
