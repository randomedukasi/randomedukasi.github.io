Logaritma adalah operasi matematika yang merupakan kebalikan dari eksponen atau pemangkatan. Contoh logaritma bentuk eksponen aᵇ = c bila dinyatakan dengan notasi logaritma adalah ᵃlogc = b  

Untuk menjawab soal di atas maka harus dilakukan transformasi bentuk logaritma menggunakan sifat-sifat logaritma.  Sifat - sifat logaritma diantaranya adalah:

Jika a>0, a ≠ 1, m ≠ 1, b>0 dan c>0, maka berlaku :

1. ᵃlog a = 1

2. ᵃlog 1 = 0

3. ᵃ^ⁿlogᵇ^m = [tex]\frac{m}{n}[/tex] ᵃlog b

4. ᵃ^ⁿlogᵇ^ⁿ = ᵃlog b

5. ᵃlog b = [tex]\frac{1}{^blog a}[/tex]

6. ᵃlog b = [tex]\frac{^mlog b}{^mlog a}[/tex]

7. a ^(alog b) = b

8. ᵃlog b + ᵇlog c = ᵃlog (bc)

Pembahasan

= ³log 81 + log 0,001 - ⁵log 125 + ²log 8

= ³log 3⁴ + ¹⁰log 10⁻³ - ⁵log 5³ + ²log 2³

= 4 - 3 - 3 + 3

= 1


Jadi Nilai dari 3log 81 + log 0,001 - 5log 125+ 2log 8 adalah 1
