1. bagas membeli rumah seharga 650.000.000, dgn DP 50.000.000, jika dicicil selama 20 bulan, berapa cicilan perbulan nya?​


diketahui : •harga rumah = 650.000.000

•DP = 50.000.000

•selebihnya dicicil 20 bulan

ditanya : berapa cicilan per bulanya....?

jawab :

harga rumah dikurangi DP :

=> (650.000.000 - 50.000.000)

=> 600.000.000

jika dicicil selama 20 bulan maka :

=> 600.000.000 / 20 bulan

=> 30.000.000 / bulan

Jawaban:

30.000.000

Penjelasan dengan langkah-langkah:

dik: harga rumah= 650.000.000

DP= 50.000.000

dicicil selama 20 bulan

dit: berapa cicilan perbulan?

jawab:650.000.000-50.000.000

=600.000.000

600.000.000÷20

=30.000.000

jadi, cicilan perbulannya adalah 30.000.000

uang awal = 200 juta -5 juta = 195 juta

a. cicilan per bulan
bunga per tahun = 0,5%
besar bunga untuk 5 tahun = 195 juta x 0,5% x 5 tahun
                                       = 4.875.000
besar bunga per buln = 4.875.000 / (5x12) = 81.250
besar uang pokok per bulan = 195.000.000/ (12x5) = 3.250.000
besar cicilan per bulan = 3.250.000 + 81.250 = 3.331.250

b. besar bunga per bulan = 81.250

c. jumlah harga seluruhnya
 195.000.000 + 4.875.000 = 199.875.000

d. harga rumah setelah selesai
  200 juta + 4.875.000 = 204.875.000

9. Bani Akan Membeli Rumah Seharga Rp. 346.000.000.Bani akan Membayar secara Cicilan KPR Dengan DP=Rp.28.000.000 dan Akan Dilunasi Selama 10 Tahun Dengan Bunga Tiap bulan 17% Berapa Cicilan Tiap Bulan Yang Di Bayar Bani? ​


penyelesaian dengan langkah-langkah :

Bani beli rumah 346.000.000-28.000.000(DP)=318.000.000

1 tahun = 12 bulan

10 tahun = 12 x 10 = 120 bulan

tiap bulan bunganya 17%

318.000.000 : 120 = 2.650.000

17% dari 2.650.000 = 450.500

jadi perbulan dia harus membayar 2.650.000 + 450.500 = 3.100.500 perbulan sudah terkena bunga 17%
