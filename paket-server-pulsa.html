2. Boss bisa cara paket pulsa


Jawaban:

Telkomsel (Simpati & AS)

- Kode USSD *888#

Ini merupakan cara paling mudah untuk cek pulsa Telkomsel. Pengguna cukup membuka layar panggilan dan menekan *888# lalu tekan OK atau Call, kemudian sisa pulsa akan muncul, lengkap dengan sisa masa aktif.

- Telepon ke 888 atau 155

Seringkali menggunakan kode USSD gagal karena gangguan operator atau masalah lain. Pelanggan dapat menggunakan cara lain dengan menelpon nomor 888 atau 155, kemudian sistem akan langsung menyebut sisa pulsa dan masa aktif yang tersisa. Tenang saja, telepon ke 888 atau 155 bebas pulsa.

- Telepon ke 188

Selain itu terdapat 188 juga yang dapat anda hubungi untuk mengecek pulsa. Namun, sayangnya layanan ini dikenakan tarif Rp300/panggilan.

- Kunjungi Situs MyTelkomsel

Pelanggan juga dapat mengunjungi laman resmi MyTelkomsel, pelanggan cukup memasukkan nomor ponsel yang digunakan dan akan mendapat SMS verifikasi. Setelah itu situs Telkomsel akan menampilkan jumlah pulsa anda, beserta masa aktifnya.

- Aplikasi MyTelkomsel

Pengguna smartphone dapat menggunakan aplikasi melalui aplikasi MyTelkomsel yang diunduh melalui Google Play Store atau Apple App Store.

Tak sekedar untuk cek pulsa saja, di dalam aplikasi MyTelkomsel terdapat banyak fungsi, seperti membeli paket data, cek poin, cek saldo Tcash, dan lain-lain.

Cara Cek Pulsa XL Axiata

- Kode USSD *123#

sama halnya dengan Telkomsel, XL juga menyediakan kode USSD. Pengguna hanya perlu menekankan *123# dilayar hp, kemudian tekan ok atau call. Saldo pulsa beserta masa aktifnya akan tertera dilayar telepon.

- Kunjungi Situs MyXL

Pengguna dapat mengunjungi situs MyXL di my.xl.co.id, masukkan nomor XL pengguna, XL akan memberikan kode konfirmasi rahasia yang harus anda masukkan dalam situs MyXL, setelah anda berhasil login, pilihlah pilihan cek pulsa kemudia anda akan mengetahui saldo pulsa yang tersisa beserta masa aktifnya.

- Melalui aplikasi myXL

Bagi pengguna smartphone, pengguna cukup mengunduh aplikasi ini di Google Play Store atau Apple App Store, Login, kemudian anda dapat memilih pilihan cek pulsa yang akan menampilkan saldo pulsa yang tersisa dan juga masa aktifnya, tak hanya itu terdapat fitur-fitur lain dalam aplikasi myXL seperti cek kuota, beli paket internet, mengetahui promo XL, dan lain-lain.

Indosat Ooredoo (IM3 & Mentari)

- Melalui kode USSD *123#

Masukan kode pada menu panggilan, lalu tekan ok atau call, tunggu beberapa saat hingga muncul pop notifikasi yang berisi informasi jumlah pulsa beserta masa aktifnya.

- Melalui aplikasi MyIM3

Bagi pengguna smartphone, anda dapat mengunggah aplikasi MyIM3 di Google App Store atau Apple App Store, kemudian login dengan menggunakan nomor indosat yang terdaftar. Dalam aplikasi anda dapat mengakses semua informasi mengenai Indosat, termasuk seputar pulsa hingga paket data.

Cara Cek Pulsa Smartfren

- Melalui SMS ke 99

Pengguna cukup mengirim SMS, dengan cara ketik CEK kemudian kirimkan ke 999, pengguna akan mendapatkan balasan berupa saldo pulsa yang tersisa beserta masa aktifnya.

- Telepon ke *999

Cara ini merupakan yang paling banyak digunakan karena mudah. Pengguna hanya perlu menekan nomor *999 pada layar panggilan telepon, lalu tekan Ok atau Call, setelah itu pengguna akan mendapatkan balasan sisa pulsa smartfren anda, beserta masa aktifnya. Tentunya layanan ini bebas pulsa.

- Telepon nomor 999

Anda hanya perlu mengikuti intruksi operator untuk mengetahui sisa pulsa, beserta masa aktifnya.

- Mengunduh aplikasi Smartfren

Bagi pengguna android atau apple, anda dapat menggunggah aplikasi smartfren pada Google Play Store atau Apple App Store. Bukan hanya untuk mengecek pulsa. dalam aplikasi ini tersedia informasi mengenai sisa kuota, masa aktif kartu, nomor smartfren pengguna, dan lain-lain.

14. Common network services pada server telah ditentukan diantaranya: Web Server, Database Server, DNS Server, dan DHCP Server. Jika diimplementasikan pada SO Linux/Debian maka paket apa saja yang cocok diinstall untuk layanan diatas. Jelaskan!


Penjelasan:

Untuk mengimplementasikan layanan Web Server pada sistem operasi Linux/Debian, Anda dapat menginstall paket Apache. Apache adalah salah satu web server yang paling populer dan banyak digunakan di seluruh dunia.

Untuk mengimplementasikan layanan Database Server, Anda dapat menginstall paket MySQL atau MariaDB. Kedua paket ini merupakan sistem manajemen database yang populer dan banyak digunakan pada sistem operasi Linux/Debian.

Untuk mengimplementasikan layanan DNS Server, Anda dapat menginstall paket BIND. BIND adalah salah satu DNS Server yang paling populer dan banyak digunakan pada sistem operasi Linux/Debian.

Untuk mengimplementasikan layanan DHCP Server, Anda dapat menginstall paket ISC DHCP Server. ISC DHCP Server adalah salah satu DHCP Server yang paling populer dan banyak digunakan pada sistem operasi Linux/Debian.
