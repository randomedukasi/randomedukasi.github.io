--------------------------------------------------

Mapel: Matematika

Tingkat: SMP dan SMA

Materi: Persamaan Linear Dua Variabel

--------------------------------------------------


Misal:

Helm jenis A = x

Helm jenis B = y

→

x + y = 50 (kalikan dengan 12) (diperoleh dari maksimal tempat 50)

12x + 9y = 540 ( kalian dengan 1) (diperoleh dari modal)

→

12x + 12y = 600

12x + 9y = 540

________________-

3y = 60

y= 20 ........... (i)

---------------------------------

x + y = 50

x + 20 = 50  → x = 30 ........... (ii)


Penyediaan Helm A = 30 buah dan Helm B = 20 buah

Keuntungan:

30x + 25y = ....

30(30) + 25(20) = 900 + 500 = 1400


Jadi, keuntungan maksimumnya adalah Rp. 1.400.000,-

30. seorang penjual helm menambah stock dagangan tidak lebih dari 50 helm,harga pembelian helm INK adalah 120.000 dan helm KYT adalah 90.000 . dari penjualan elm tersebut di peroleh keuntungan 30.000 dan 25.000 untuk helm KYT. jika modal pedagang 5.400.000 tentukan keuntungan maksimum


Penyelesaian:

Misal: x = helm INK
y = helm KYT

didapat model persamaan matematika dari data yg diberikan soal:

persamaan 1 : 120,000x + 90,000y ≤ 5,400,000  (titik potong: (45,0) dan (0,60)
persamaan 2 : x + y ≤ 50            (titik potong: (50,0) dan (0,50)

eliminasi persamaan 1 dan persamaan 2, didapat:
x = 30 ; y = 20

substitusi titik potong yang paling kecil dari pers 1, pers 2, dan dari hasil eliminasi pers 1 dan pers 2: 
f(x,y) = 30,000x + 25,000y
(0,50)                 1,250,000
(45,0)                 1,350,000
(30,20)               1,400,000       ----MAKSIMUM---
