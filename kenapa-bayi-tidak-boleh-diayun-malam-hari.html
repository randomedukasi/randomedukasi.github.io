8. Seorang bayi yang sedang demam memiliki suhu pada siang hari sebesar 37°C. Pada menjelang malam hari suhu bayi tersebut meningkat menjadi 38,5°C. Berapakah peningkatan suhu bayi tersebut jika diukur menggunakan termometer Fahrenheit?


dik: seorang bayi demam dengan suhu siang hari 37°c dan malam meningkat menjadi 38,5°c
dit: berapakah peningkatan suhu jika diukur dengan thermometwr fahrenheit?
jawab:
suhu siang hari = t°f= (9/5.37) +32
= (333/5) + 32
= 333/5 + 32
= 365/5
=73°f

suhu malam hari = t°f (9/5.38,5) + 32
= (346,5/5) +32
= 378,5/5
= 75,7°f

jadi peningkatan suhu =
suhu siang - suhu malam
=73°-75,7°
=2,7° f

jadi peningkatan suhu nya adlh 2,7°f

= 4000 N/m
m = 3kg
a = 10 m/s^2

F = kx
x = F /k = 3 x 10 / 8000

9 cm

6/8 x 120 cm = 9 cmjawab

deret tak hingga

panjang lintasan pertama = p1 = 120 cm
r = 6/8 = 3/4
panjang lintasan kedua = p2 = 3/4 x 120 = 90 cm

Panjang Lintasan sampai berhenti = P~
P ~ =  p1  +  2 {  p2 / (1 - r)}
P~ = 120 + 2 { (90) /(1 - 3/4)}
P~ = 120 + 2 (90 /(1/4))
P~ = 120 + 8(90)
P~ = 120 + 720
P~ = 840 cm

atau cara lain
p1= 120
r = 6/8 = 3/4 
P~ =  (4+ 3) /(4 - 3) x 120
P~ =  7/1 x 120
P~ = 840
