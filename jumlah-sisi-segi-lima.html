17. Berapakah jumlah sisi tegak prisma segi lima


bentuk alas dan tutup kan segi lima ada lima,.. Prisma tegak segi n beraturan jumlah sisinya = n + 2. Kalau jumlah rusuknya = 3n. Jadi kalau prisma segilima tegak beraturan jumlah sisinya = 5 + 2 = 7, yang terdiri dari : 1 sisi alas, 1 sisi atas ( tutup ), dan 5 sisi tegak.

24. Jumlah sisi yang dimiliki Prisma segi lima adalah​


Penjelasan:

Bangun datar segitiga memiliki jumlah sisi tiga, jumlah sudut tiga , dan jumlah titik sudut tiga. Bangun datar segi lima: Bangun datar segi lima memiliki jumlah sisi lima, jumlah sudut lima , dan jumlah titik sudut lima, Bangun datar segi enam: Bangun datar segi enam memiliki jumlah sisi enam.

Jawaban: Prisma segilima mempunyai ..... sisi

Pembahasan :

prisma adalah salah satu bangun ruang sisi datar yang bentuk alas dan tutupnya sama

prisma banyak macamnya, diantaranya

prisma segitiga → prisma dengan alas segi tiga

prisma segi empat → prisma dengan alas segi empat

prisma segi lima → prisma dengan alas segi lima

prisma segi enam → prisma dengan alas segi enam

prisma segi tujuh → prisma dengan alas segi tujuh

prisma segi delapan → prisma dengan alas segi delapan

prisma segi sembilan → prisma dengan alas segi sembilan  

prisma segi sepuluh → prisma dengan alas segi sepuluh

dan seterusnya sampai prisma segi-n

untuk menghitung jumlah sisi, titik sudut dan rusuk pada prisma dapat kita gunakan rumus:

→ banyak sisi prisma segi-n = n + 2

→ banyak rusuk prisma segi-n = 3n

→ banyak titik sudut prisma segi-n = 2n

sekarang mari kita kerjakan soal

Prisma segilima mempunyai .....sisi,....rusuk,dan....titik sudut

prisma segi 5

→ banyak sisinya prisma segi lima = 5 + 2 = 7
