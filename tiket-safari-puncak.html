18. Tersedia 800 tiket untuk sebuah konser musik diakhir tahun, tiket tersebut terdiri dari tiket VIP dan festival, dan banyak tiket festival 4 kali lebih banyak dari tiket VIP. Jika harga tiket festival Rp75.000,00 sedangkan VIP Rp280.000,00, maka hasil penjualan semua tiket jika tiket habis terjual adalah …


Jawab:

101 juta

Penjelasan dengan langkah-langkah:

tiket tersedia = 800 tiket

tiket vip : tiket festival = 1 : 5

tiket festival = Rp. 75rb

tiket VIP = Rp. 280rb

tiket VIP = 1/5 x 800 = 160 tiket

tiket festival = 4/5 x 800 = 640 tiket

tiket vip = 160 x 280rb = 44,800,000

tiket festival = 640 x 75rb = 48,000,000

hasil penjualan = 44,800,000 + 48,000,000 = 92,800,000
