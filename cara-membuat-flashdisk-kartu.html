Beberapa hal yang patut dicurigai ketika UFD (USB Flash Disk) yang kita miliki tidak terdeteksi saat dicolok ke PC, antara lain :

– Port USB Rusak

– Setting bermasalah ( Bios, Windows)

– Driver bermasalah

– Rusak secara Fisik

– dll

untuk mencoba mengatasinya maka dapat dilakukan beberapa hal berikut ini :

1. Coba colok USB kita pada Port USB yang lain /pindah-pindah, jika belum berhasil coba colokan ke PC lain, apabila berhasil maka USB kita tidak bermasalah dan kecurigaan 100% kepada PC.

2. Atau dilakukan sebaliknya, coba colokan USB lain di PC kita (jika ada)/ atau perangkat lain yang menggunakan usb port. Dengan asumsi, jika alat yang kita hubungkan terdeteksi, maka port USB tidak bermasalah.

3. Cek di settingan BIOS barangkali disable sengaja atau tidak, hal ini bisa saja terjadi perubahan setting dengan tidak kita sadari manakala baterai MotherBoard kita udah soak / lemah.

4. Kemungkinan yang lainnya adalah PC terjangkit Virus, maka harus dibersihkan dari virus dengan menggunakan anti virus

5.Jika mencurigai sebuah software yang baru diinstall sebagai penyebabnya maka coba uninstall dahulu software tersebut.

6. Coba gunakan software utiliti untuk find dan mount partisi ( mudah2an bisa).

Berikut ini cara membuatnya:

1. Buat pola pada karton duplek dengan ukuran 24 x 8 cm, kemudian gunting.

2. Untuk membuat satu kartu, kamu harus merekatkan dua pola karton yang sudah digunting menggunakan lem, sehingga ukuran kartunya tebal.

5. Rapikan setiap sudut kartu agar tidak terlalu lancip menggunakan gunting.

6. Ulangi langkah itu, sebanyak kartu yang ingin kamu buat.

17. Cara membuat multi installer windows 10 di flashdisk rufus


Jawaban:

Cara Memasukkan Beberapa Installer ke Flashdisk (Multiboot USB) Beranda Windows Cara Memasukkan Beberapa Installer ke Flashdisk (Multiboot USB)

Cara membuat multiboot usb installer pada Flashdisk atau Hardisk

Pembuatan sebuah bootable Flashdisk installer merupakan salah satu langkah penting jika hendak melakukan install ulang sebuah sistem operasi. Ketimbang menggunakan CD / DVD untuk install OS, maka Flashdisk akan membuat proses install menjadi semakin cepat. Selain itu, sangat mudah memasukkan sebuah file ISO sistem operasi ke dalam Flashdisk agar menjadi USB Bootable. Anda bisa memasukkan OS apa saja, seperti Windows, Linux, dll.

Namun, yang menjadi masalah adalah ketika anda hanya bisa memasukkan satu file ISO ke dalam satu Flashdisk. Artinya, satu Flashdisk hanya bisa anda jadikan sebagai satu installer OS saja. Dengan demikian ketika anda hendak menginstall OS lain, maka harus memasukkan ulang file ISO OS yang lain dan menghapus yang lama. Hal ini tentu saja akan sangat merepotkan, terlebih lagi jika kapasitas Flashdisk / Hardisk anda sebenarnya bisa menampung beberapa file ISO sistem operasi sekaligus.

Ada banyak jenis sistem operasi yang bisa anda install pada setiap perangkat Komputer atau Laptop, dan semuanya memiliki beberapa versi dan fitur unggulan tersendiri. Sedangkan, tidak mungkin anda harus menyedikan Flashdisk sebanyak jumlah sistem operasi yang akan anda buatkan installer. Akan lebih efektif jika anda memasukkan semua file ISO sistem operasi tersebut ke dalam satu Flashdisk atau membuat sebuah Multiboot USB. Jadi, anda bisa hemat dana karena tidak perlu beli banyak Flashdisk dan saat ingin install OS mudah menemukannya karena tersimpan dalam satu Flashdisk.

24. flashdisk terbuat dari...


Flash Disk intinya dibuat dari Controller dan Flash. Mirip sekali dengan SSD jadi misal ada Sandforce Controller dan NAND Flash. Itu semua dari bahan carbon yang digabung dalam satu board.

Kalau ada pembuat Flas disk atau SSD bilang ada die shrink berarti partikel Flash yang dibuat lebih kecil yang berdampak kapasitas naik tapi reabilitas turun.

25. Cara membuat dua installer windows di flashdisk ventory


Penjelasan:

Membuat Bootable Windows 10 dengan Ventoy

Download dan jalankan Ventoy.Pilih perangkat USB yang kamu inginkan. Pastikan memiliki drive yang sesuai.Klik opsi Install dan klik Yes ketika muncul konfirmasi. Pada tahap ini Ventoy akan menghapus semua data yang ada pada flashdisk tersebut, jadi pastikan kamu sudah membackupnya jika terdapat data.Done.Setelah proses selesai, flashdisk akan memiliki dua partisi yaitu Ventoy dan VTOYEFI.

Partisi VTOYEFI berisikan semua komponen yang diperlukan agar dapat melakukan mount ISO dan bootable lewat flashdisk. Sedangkan partisi Ventoy sendiri bisa kamu isikan file ISO yang diinginkan, kamu bisa copy file ISO secara utuh tanpa perlu mengekstrak isinya terlebih dahulu.

Pada saat melakukan installasi, kamu akan mendapati layar opsi yang berisikan semua file ISO yang ada pada flashdisk. Selanjutnya kamu bisa mengikuti instruksi dilayar untuk melakukan installasi seperti biasa.
