21. Buatlah dialog menggunakan expression "Getting one attention and checking ones understanding "​


Jawaban:

Dialog Getting Attention

Daisy : Look what I’ve got here! 

Bella : What is that? 

Daisy : A ticket for Ariana Grande concert! (

Bella : Lucky you! I also want to go to the concert, but I can’t afford a ticket. 

Daisy : Oh don’t be sad, because look at this! I bought one ticket for you too! 

Bella : No way! Are you kidding, aren’t you? 

Daisy : No, I’m not. Here, see the ticket yourself if you don’t believe.

Bella :  Oh my God! It’s real.  I’m going to see Ariana Grande! Thank you so much! 

Daisy : You’re welcome. 

Dialog Checking ones understanding

Rina : Hi Sindy. 

Sindy : Hi Rina. 

Rina : Have you finished the assignment from Ms. Jenny? 

Sindy : Yes, I’ve just finished it. How about you?

Rina : Not yet, I’m still confused in answering some questions especially changing the verbs of active sentence into passive or vice versa.

Sindy : Oh, that one. Before you change the sentence into passive or active, please make sure what kind of sentence it is, an active or a passive. If it’s an active sentence, you need to change the verb 1 into verb 3 or vice versa. Do you get my point?

Rina : I’m not sure I get your point. It’s confusing.

Sindy : That’s okay Rina, I will teach you and give some examples so you can understand.

Rina : Sure Sindy, if you don’t mind.

Sindy : Okay, let’s study together. 

Rina : Okay Sindy. 

Penjelasan:

Terjemahan :

Getting one Attention

Daisy : Lihat apa yang aku punya!

Bella : Apakah itu?

Daisy : Satu tiket konser Ariana Grande

Bella : Kamu beruntung! Aku juga ingin pergi ke konsernya, tetapi aku tidak mampu membeli tiketnya.

Daisy : Oh jangan sedih, karena lihat ini! Aku juga membelikan satu tiket untukmu!

Bella : Tidak mungkin! Kamu bercanda, kan?

Daisy : Tidak. Ini, lihat sendiri tiketnya jika kamu tidak percaya

Bella : Ya Tuhan! Ini asli. Aku akan bertemu Ariana Grande! Terima kasih banyak!

Daisy : Sama-sama

Checking ones understanding

Rina : Hai Sindy

Sindy : Hai Rina

Rina : Sudahkah kamu menyelesaikan tugas dari Ms. Jenny?

Sindy : Ya, aku baru saja menyelesaikannya. Bagaimana denganmu?

Rina : Belum, aku masih bingung dalam menjawab beberapa pertanyaan terutama merubah kata kerja dari kalimat aktif menjadi pasif atau sebaliknya.

Sindy : Oh, yang itu. Sebelum kamu merubah kalimatnya menjadi pasif atau aktif, pastikan dulu jenis kalimatnya, kalimat aktif atau pasif. Jika itu merupakan sebuah kalimat aktif, kamu perlu merubah kata kerja bentuk 1 menjadi kata kerja bentuk 3 atau sebaliknya. Apakah kamu paham maksudku?

Rina : Aku tidak yakin aku mengerti maksudmu. Itu membingungkan.

Sindy : Tidak masalah Rina, aku akan mengajarmu dan memberimu beberapa contoh jadi kamu bisa mengerti

Rina : Tentu Sindy, jika kamu tidak keberatan.

Sindy : Baik, ayo kita belajar bersama.

Rina : Oke Sindy.
