Seorang pedagang paling sedikit menyewa 25 kendaraan jenis truk dan colt untuk mengangkut paling sedikit 224 karung. Sebuah truk dapat mengankut sebanyak 14 karung dan colt 8 karung. Ongkos sewa truk adalah Rp 400.000,- sekali jalan dan colt Rp 350.000,- sekali jalan. Banyaknya truk dan colt yang harus di sewa agar biaya sewanya minimal adalah ...

4 truk dan 21 colt.

Pembahasan :

Untuk mencari Banyaknya truk dan colt yang harus di sewa agar biaya sewanya minimal, maka dibuat sistim pertidaksamaan.

Diketahui:

Seorang pedagang paling sedikit menyewa 25 kendaraan jenis truk dan colt untuk mengangkut paling sedikit 224 karung. Sebuah truk dapat mengankut sebanyak 14 karung dan colt 8 karung. Ongkos sewa truk adalah Rp 400.000,- sekali jalan dan colt Rp 350.000,- sekali jalan.

Ditanya:

Banyaknya truk dan colt yang harus di sewa agar biaya sewanya minimal adalah ?

Dijawab :

colt = c

truk = t

Sistim pertidaksamaan:

c + t ≥ 25

14t + 8c ≥ 224

400000t + 350000c minimal =?

14t + 8c ≥ 224

7t + 4c ≥ 112

mencari t dan v:

c + v = 25

c = 25 - v

7t + 4c = 112

7t + 4(25 - t) = 112

7t + 100 - 4t = 112

3t = 12

t = 4

c + t = 25

c + 4 = 25

c = 21

misal sumbu x = t

y = c

persamaannya menjadi:

x + y = 25

7x + 4y = 112

buat grafik dengan titik potong:

x + y = 25

(0,25) dan (25,0)

7x + 4y = 112

(0,28) dan (16, 0)

titik potong kedua garis(4,21)

Dari sistim pertidaksamaan:

yang diuji titik

(25,0), (4,21), (0,28)

400000t + 350000c minimal =?

400000x + 350000y =

(4,21)

400000(4) + 350000(21) = 8950000

(25,0)

400000(25) + 350000(0) = 10000000

(0,28)

400000(0) + 350000(28) = 9800000

minimal = 8950000

x = 4

y = 21

t = 4

c = 21

Jadi Banyaknya truk dan colt yang harus di sewa agar biaya sewanya minimal adalah :

4 truk dan 21 colt

Truk dapat mengangkut tidak lebih dari 14 karung dan colt 8 karung. Ongkos sewa truk Rp500.000,00 dan colt Rp300.000,00. Jika x menyatakan banyaknya truk dan y menyatakan banyaknya colt, maka model matematika dari permasalahan di atas adalah​


Jawaban:

Dalam soal dimisalkan

x = banyaknya truk ⇒ x ≥ 0

y = banyaknya colt ⇒ y ≥ 0

14x + 8y ≤ 272

kedua ruas dibagi 2

f(x, y) = 500.000x + 300.000y

x + y ≥ 28

7x + 4y ≤ 136

y ≥ 0

f(x, y) = 500.000x + 300.000y
