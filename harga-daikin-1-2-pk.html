9. Pk yang akar akarnya 4 dan -1/2 adalah...​


Jawaban:

[tex]2 {x}^{2} - 7x - 4 = 0[/tex]

Penjelasan dengan langkah-langkah:

Karena akar-akarnya adalah 4 dan -1/2, maka:

[tex](x - 4)(x - ( - \frac{1}{2} )) = 0 [/tex]

[tex](x - 4)(x + \frac{1}{2}) = 0 [/tex]

[tex] {x}^{2} - 4x + \frac{1}{2} x - 2 = 0[/tex]

[tex]{x}^{2} - \frac{7}{2} x - 2 = 0[/tex]

Kedua sisi dikalikan 2, akan menghasilkan

[tex]2{x}^{2} - 7x - 4 = 0[/tex]

29. jika x1 dan x2 merupakan akar2 pk x^2+3x+1=0, maka pk yg akar2nya 1-x1 dan 1-x2 adalah


[tex](x+1)^{2}+3(x+1).(-1)^{2}+1.(-1)=0\\x^{2}+2x+1+3x+3-1=0\\x^{2}+5x+3=0[/tex]

x² + 3x + 1 = 0

x² - ( -3)x + 1 = 0

x1 + x2 = -3
→ 1 - x1 + 1 - x2 = -3
→ x1 + x2 = -3 - 2
→ x1 + x2 = -5

x1.x2 = 1
→ (1 - x1)(1 - x2) = 1
→ 1 - x1 - x2 - x1x2 = 1
→ -(x1 + x2) - x1x2 = 1 - 1
→ -x1x2 = 0 + x1 + x2
→ x1x2 = - ( x1 + x2 )
→ x1x2 = -(-5)
→ x1x2 = 5

persamaan kuadrat akar²nya 1 - x1 dan 1 - x2
adalah :

x² - (-5)x + 5 = 0

x² + 5x + 5 = 0
