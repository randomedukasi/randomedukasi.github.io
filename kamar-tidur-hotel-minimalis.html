8. hitunglah selisih luas kamar tidur anak dan kamar tidur pembantu sebenarnya!


Kamar tidur anak =>

Luas sebenarnya = Luas pada peta : Skala

~ panjang sebenarnya ~

=> 4 cm : 1/150

=> 4 cm × 150/1

=> 600 cm/1

=> 6 m

~ Lebar sebenarnya ~

=> 3 cm : 1/150

=> 3 cm × 150/1

=> 450 cm

=> 4,5 m

Luas sebenarnya kamar tidur anak

=> p x l

=> 6 m x 4,5 m

=> 27 m^2

**************************************************

Kamar tidur pembantu =>

Luas sebenarnya = Luas pada peta : Skala

~ panjang sebenarnya ~

=> 3 cm : 1/150

=> 3 cm × 150/1

=> 450 cm

=> 4,5 m

~ Lebar sebenarnya ~

=> 2,5 cm : 1/150

=> 2,5 × 150/1

=> 375 cm

=> 3,75 m

Luas sebenarnya kamar tidur pembantu

=> p x l

=> 4,5 m x 3,75 m

=> 16,875 m^2

Selisih luas kamar tidur anak dengan kamar tidur pembantu sebenarnya

=> 27 m^2 - 16,875 m^2

=> 10,125 m^2

= 3 × 4

= 1.800 cm

= 2,5 × 3

= 7,5 cm

= 7,5 × 150

= 1.125 cm

= 11,25 m

12. suhu di dalam kamar sebuah hotel 22°C derajat Celcius sedangkan suhu di luar hotel 27 °C lebih rendah dari suhu di dalam kamar hotel Berapa suhu di luar hotel tersebut​


Jawab:
Suhu luar = -5°C

Penjelasan:
Suhu dalam kamar (t₀) = 22°C
→ rendah = minus (-)
Perubahan suhu (Δt) = -27°C
Ditanya suhu luar (t₁)

t₁ = t₀ + Δt
t₁ = 22 + (-27)
t₁ = -(27-22)
t₁ = -5

Suhu luar = -5°C
(Xcvi)

~Bilangan Bulat

Suhu di dalam kamar sebuah hotel 22°C sedangkan suhu di luar hotel 27 °C lebih rendah dari suhu di dalam kamar hotel. Suhu di luar hotel tersebut adalah - 5°C

Pembahasan

Di soal tertulis "suhu di luar hotel 27°C lebih rendah dari suhu di dalam hotel", karena lebih rendah berarti bernilai negatif ( - 27°C )

Misal:

suhu di luar hotel = xsuhu di dalam hotel = y

Maka:

x = - 27°C + y

x = - 27°C + 22°C

x = - 5°C

Untuk menjawab soal ini, siswa perlu memahami dan menerapkan pengetahuan tentang operasi bilangan bulat sederhana dalam kehidupan sehari-hari. Tipe soal ini merupaka tipe pengaplikasian. Diaman siswa harus memahami teori dan mampu menerapkannya.

Sebelumnya kita artika terlebuh dahulu bilanagn bulat.

Pembagian pada dasarnya merupakan pengurangan secara berulang.
