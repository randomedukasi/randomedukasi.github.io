Jawaban:

50 km/jam

Penjelasan:

Waktu tempuh kereta ekonomi

t = 400/40 = 10 jam

Waktu tempuh kereta eksekutif

t = 10 - 2 = 8 jam

Kelajuan kereta eksekutif

V = 400/8 = 50 km/jam

Jawaban:

Kereta Api Ekonomi

S = 400 KM

V = 40 KM/Jam

T = S÷V

= 400 ÷ 40 = 10 Jam

Kereta Api Exekutif

T = 10 - 2 = 8 Jam

S = 400 KM

V = S ÷ T

= 400 ÷ 8 = 50 Km/Jam

5. kereta api sancaka kelas eksekutif berangkat dari yogyakarta pukul 23.45 dan tiba di surabaya pukul 04.35. kereta api ekonomi berangkat dari yogyakarta pukul 06.30 dan tiba di surabaya pukul 11.27. selisih waktu tempuh kedua kereta api tersebut adalah?​


Kereta api sancaka kelas eksekutif berangkat dari yogyakarta pukul 23.45 dan tiba di surabaya pukul 04.35. kereta api ekonomi berangkat dari yogyakarta pukul 06.30 dan tiba di surabaya pukul 11.27. selisih waktu tempuh kedua kereta api tersebut adalah?

Penyelsaian  

Waktu merupakan interval atau jarak antara dua atau lebih kejadian atau keaadaan, atau juga bias dikatakan sebagai lamanya suatu kejadian atau keadaan berlangsung.

Diketahui KA Sancaka eksekutif brangkat pukul = 23.45 dan sampai 04.35 sehingga kereta tersebut menghabiskan waktu tempuh perjalanan sebesar 3 jam 50 menit. sedangkan kereta KA Sancaka ekonomi brangkat pukul = 06.30 dan sampai 11.27 sehingga kereta tersebut menghabiskan waktu tempuh perjalanan sebesar 4 jam 57 menit.

Dengan demikian selisish waktu tempuh kedua kereta tersebut adalah  3 jam 50 menit -  4 jam 57 menit =  1 jam 07 menit.

• Satuan waktu

Berdasarkan pada satuan internasioanal (SI), satuan standar yang dimiliki oleh waktu yaitu detik atau sekon second.

• Konversi satuan waktu  

1 jam = 60 menit

1 menit = 60 detik

1 detik = 1000 milidetik

Kursi di kelas eksekutif = 30%

Dicari :

Kursi di kelas eksekutif = jumlah kursi × %kursi di kelas eksekutif

Kursi di kelas eksekutif = 200 × 30%

Kursi di kelas eksekutif = 60 kursi

Jadi, kursi di kelas eksekutif ada 60 kursi

Kereta itu hanya dapat membawa bagasi maksimum 1800 kg. Jika harga tiket kereta untuk kelas eksekutif Rp 30000 dan kelas ekonomi Rp 15000. Maka tentukanlah model matematikanya agar pendapatan maksimum.


misal x adalah kelas eksekutif
dan y adalah kelas ekonomi
maka persamaannya

x + y </= 50
60x + 30y </ 1.800

fungsi tujuannya
f(x,y) = 30.000x + 15.000y

cari titik potongnya dan gambar grafiknya, namti ketauan mana titik yang memenuhi syarat
pers 1:
x=0 y=50 ---> (0,50)
x=50 y=0 ---> (50,0)
pers 2:
x=0 y=60 ---> (0,60)
x=30 y=0 ---> (30,0)

cari nilai x dan y lewat eliminasi
x + y = 50
60x + 30y = 1800 --> dibagi 30
jadi
x + y = 50
2x + y = 60
dikurangi, jadi
-x= -10
x= 10
cari y lewat substitusi
(10) + y = 50
y= 50 - 10
y= 40
titik potongnya (10,40)

substitusi titik2 ke fungsi tujuan
f(0,50) = 30.000(0) + 15.000(50)
= 750.000
f(10,40) = 30.000(10) + 15.000(40)
= 300.000 + 600.000
= 900.000
dan substitusi titik2 lainnya, hasil terbesar adalah keuntungan maksimum
