1. cara mengaktifkan/menonaktifkan windows update pada windows 10


untuk menonaktifkan nya dengan cara:
1. ketik "services.msc" pada Run lalu enter
2. cari Windows Update lalu klik kanan dan pilih Properties
3. pilih Disabled pada startup type dan pilih Apply
4. lalu Restart komputernya

2. Cara mematikan windows update windows 10 permanen adalah .....


Jawaban:

Cara-Cara

1. Buka Control Panel dengan cara mengetikkan “Control Panel” di kolom pencarian di samping Start Menu.

2. Kemudian, pilih System and Security dan tekan menu Administrative Tools.

3. Akan muncul halaman baru yang berisikan berbagam macam fitur dengan fungsi yang berbeda-beda. Di sini, Anda pilih Services saja.

4. Scroll ke bawah sampai Anda menemukan Windows Update, dan klik dua kali. Muncul halaman Windows Update Properties.

5. Sekarang, ganti opsi pada Startup Type menjadi Disabled, dan klik tombol Stop pada Service Status. Kemudian tekan Ok.

7. carilah cara menonaktifkan sistem windows 7,8,9,10​


Jawaban:

Untuk menonaktifkan sistem Windows, Anda dapat melakukan hal-hal berikut:

Melalui Control Panel: Buka Control Panel dan pilih "System and Security." Kemudian, pilih "Power Options" dan pilih "Choose what the power buttons do." Di sini, Anda dapat mengatur tombol daya untuk menonaktifkan sistem saat ditekan.

Melalui Command Prompt: Buka Command Prompt dengan hak akses administrator dan ketik "shutdown /s /t 0" tanpa tanda petik. Ini akan menonaktifkan sistem secara instan.

Melalui Shortcut Keyboard: Anda dapat membuat shortcut keyboard untuk menonaktifkan sistem Windows. Caranya adalah dengan membuat shortcut baru di desktop dan menambahkan "shutdown /s /t 0" tanpa tanda petik sebagai target. Kemudian Anda tinggal menekan tombol shortcut yang telah anda buat.

Perlu diingat bahwa menonaktifkan sistem secara tiba-tiba tanpa menyimpan data Anda dapat menyebabkan kerusakan pada data Anda.

15. Jelaskan cara menonaktifkan komputer berbasis Windows


1.Menekan tombol power. Ini adalah cara tercepat untuk mematikan komputer. Tombol power di laptop atau PC jika ditekan akan segera terjadi penutupan beragam aplikasi secara instan, lalu mati. Hanya saja cara ini sebaiknya digunakan ketika benar-benar dalam posisi terpaksa, seperti komputer hang. Pasalnya, langkah ini memungkinkan terjadinya kerusakan sistem file, sistem operasi, hingga program yang saat itu masih terbuka.

2.Menerapkan shortcut combo. Ini dilakukan dengan memencet beberapa tombol yang ada di keyboard. Bahkan, Anda tidak perlu menggunakan campur tangan mouse untuk urusan ini. Pada sistem operasi Windows 7, penerapannya sangat sederhana. Tinggal tekan tombol Windows, tekan panah kanan, lalu tekan tombol Enter. Sementara pada Windows 10 hanya dengan menekan tombol Windows, lalu tekan X,u,u.

3.Membuat shortcut taskbar di laman desktop. Bisa pula mematikan komputer dengan mengeklik shortcut buatan sendiri yang diletakkan di desktop. Cara seperti ini:

Klik kanan pada desktop, tekan New lalu Shortcut.Untuk isian di kota Location, tuliskan perintah ini: “%windir%system32shutdown.exe -s -t 0” (tanpa tanda “)Klik Next, lalau namai shortcut sesuai keinginan.Shortcut yang sudah dibuat dapat diklik saat Anda ingin mematikan komputer.

16. Bagaimana cara menginstall Windows 10 Fall Creator Update di Windows 10 Anniversary Update x64-bit?


Langkah 1: Kunjungi halaman ini dan download Media Creation Tool versi terbaru. Pada halaman tersebut, klik Download tool now untuk memulai proses unduh Media Creation Tool.

Download Media Creation Tool

Download Media Creation Tool

Langkah 2: Jalankan aplikasi tersebut di komputer Windows 10 yang ingin Anda upgrade ke Creators Update. Klik tombol Accept untuk menerima license agreement.

License terms Windows 10 Media Creation Tool

License terms Windows 10 Media Creation Tool

Langkah 3: Selanjutnya, aplikasi akan menampilkan halaman "What do you want to do?". Disini, pilih opsi Upgrade this PC now lalu klik tombol Next.

Windows 10 Media Creation Tool

26. ada yang tau ga cara berhentiin windows update? kuota terkuras:"


. Langkah yang pertama, masuk ke control panel pada windows.
2. Langkah yang kedua, klik pada pilihan system and security.


3. Langkah selanjutnya, klik pada menu windows update.


4. Keempat, setelah anda masuk ke menu windows update, klik change settings pada sidebar bagian kiri untuk mematikan windows update.


5. Pada important update, klik dan pilih never check for updates, kemudian klik OK pada tombol bagian bawah.

Cara nonaktifkan Windows Update yg update otomatis di Windows 10:
1. Defer Update (membatalkan update sampai dapat koneksi internet yg bagus):
- Masuk ke Setting dengan klik Start - Setting atau ke Notification, lalu klik All Setting
- Kemudian masuk ke Update & Security, dan pilih Windows Update
- Kalau sudah masuk ke menu Windows Update, klik Advance Option dan klik/centang pilihan Defer Update

2. Set as Metered Connection (memberi "cap" pada wifi yg digunakan sebagai jaringan berkuota sehingga update tidak jalan otomatis):
- Masuk ke Setting, lalu pilih Network & Internet, dan pilih menu Wifi
- Kemudian, klik Advance Option, lalu klik/centang Set as Metered Connection

3. Disable Service (Cara paksa dengan menatikan service/programnya langsung. Cara ini dilakukan jika cara 1 & 2 ga ampuh)
- Masuk ke Service dengan ke Search, lalu ketik Service dan enter, atau buka Run (tombol windows+R) lalu ketik service.msc dan enter
- Kalau sudah terbuka jendela Service, cari Windows Update. Kalau sudah, klik kanan lalu klik Properties
- Kemudian, berhentikan dulu servicenya dengan klik Stop dan pada pilihan Startup Type, pilih Disable, dan selesai (update tidak akan berjalan lagi, namun beberapa aplikasi seperti Store dan aplikasi2 lain yg berbasis metro (dari Store) tidak bisa dijalankan dengan normal/sering no internet access)

28. bagaimana cara update windows 7 ke windows 11


Jawaban:

hanya punya satu pilihan yaitu dengan cara clean install atau sering juga disebut dengan install ulang. Berbeda dengan upgrade Windows 7 ke Windows 10 yang bisa dilakukan tanpa install ulang, khusus untuk sistem operasi terbaru ini pengguna tidak diberikan pilihan tersebut.

Bagi Anda yang tertarik ingin upgrade dari Windows 7 naik ke Windows 11 maka harus melakukan install ulang. Tapi sebelum melakukannya kami sarankan untuk memastikan bahwa komputer Anda memenuhi syarat minimum agar bisa menjalankan Windows 11.

Berikut ini adalah spesifikasi minimal PC dan laptop yang bisa upgrade ke Windows 11:

Prosesor: 1GHz, 2 Cores, 64-bit compatible

RAM: 4GB

Graphics: Compatible with DirectX 12 or later with WDDM 2.0 driver

Display: 720p

Storage: 64GB

Firmware: UEFI

Mendukung fitur TPM 2.0 dan Secure Boot
