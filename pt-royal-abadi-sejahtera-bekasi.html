10. perhatikan tabel skor tes penerimaan karyawan pt royal abadi berikut skor 5, 10, 15, 20, 25, 30. frekuensi 8, 6, 10, 5, 4, 7. i. mran = 16,5 ii. modus = 5 iii. kuartil atas = 25,5 iv. persentil ke 20 = 6​


Penjelasan dengan langkah-langkah:

i.Mean= 16.5 ✅

Mean (nilai tengah) dari data tunggal

Mean = Sigma Fi.Xi /n

n= jumlah frekuensi

F1.X1= 5x8=40

F2.X2= 10x6=60

F3.X3= 15x10=150

F4.X4= 20x5=100

F5.X5= 25x4=100

F6.X6= 30x7=210

sigma Fi.Xi= F1.X1+F2.X2+F3.X3+F4.X4+F5.X5+F6.X6

= 40+60+150+100+100+210= 660

n= 8+6+10+5+4+7= 40

Mean= 660/40 = 16,5 (benar)

ii.Modus=5 ❌

Modus (paling sering muncul) dari data tunggal

Fi terbanyak adalah 10 yaitu pada skor

Jadi, Modus data di atas = 15

iii. Kuartil Atas = 25,5

Kuartil Atas (Q3) pada data tunggal = 25,5 ❌

Fe(jumlah frekuensi)=40

Rumus Kuartil pad data tunggal

Letak Qi= Data ke- i(Fe+1)/4

Qi = XQi(bilangan bulat)+Qi(bilangan desimal/pecahan) x ( (XQi+1)-XQi)

X= data ke- (bilangan bulat)

Kuartil dari data di atas

Letak Q3= Data ke- 3(40+1)/4

= Data ke- 123/4

= Data ke 30,75

X---> 30. Qi desimal/pecahan= 0,75

X30=25. X31=25

jadi Q3= X30 + 0,75 (X31-X30)

= 25 + 0,75 ( 25-25)

Q3 = 25 + 0 = 25

Jadi, Kuartil Atas data di atas = 25

iv. Persentil ke-20 = 6 ✅

Persentil pada data tunggal

Letak Pi = Data ke- i (Fe+4)/100

Pi = XPi(bilangan bulat)+Pi(bilangan desimal/pecahan) x ((XPi+1)-XPi)

Persentil dari data di atas

Letak P20= Data ke- 20(40+1)/100

= Data ke- 820/100

= Data ke- 8,2

X---> 8. Pi desimal/pecahan= 0,2

X8 = 5. X9=10

Jadi, P20= X8 + 0,2(X9-X8)

= 5 + 0,2(10-5)

= 5 + 0,2(5)

= 5+1

P20= 6

Jadi, P20 = X8 + 0,2 ( X9-X8)
