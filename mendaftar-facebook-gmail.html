22. 1. langkah membuka gmail2. cara masuk akun gmail3. cara mendaftarkan aku baru4. cara mengirim file lewat Emali


Misalkan membuka gmail memakai komputer/ laptop:
1. yang pertamakali dilakukan adalah membuka program web browser (misalnya Mozilla Firefox) lalu ketikkan www.gmail.com pada address bar.

2. saat tampil jendela situs www.gmail.com klik tombol Masuk, yang mana nantinya kita diminta mengisikan user name dan password email.

3. untuk mendaftarkan akun baru, klik tombol buat akun (berwarna merah), yang mana nantinya kita diminta mengisikan biodata diri secara lengkap.

4. untuk mengirim file lewat email, dapat memakai tombol Lampirkan file saat menulis email.
