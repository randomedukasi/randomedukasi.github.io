Rp.14.400.000

Harga beli sofa II : (10% x Rp.12.000.000 ) + Rp.12.000.000 =
Rp. 1.200.000 + Rp.12.000.000 =
Rp. 13.200.000
Maka harga jualnya agar keuntung annya sama dengan sofa I dalam persen : (20% x Rp.13.200.000) + Rp.13.200.000 =
Rp.2.640.000 + Rp.13.200.000 =
Rp.15.840.000 -->> kalau yg dimaksud keuntungan sama persennya

Rp. 15.600.000

18,18 % atau jika dibulatkan jadi 18,2%

Mo = 12.000.000 dan n% = 20%
Mo = M + M(n%) = M(1 + 20/100)
12.000.000 = M(6/5)
M = 12.000.000(5/6) = Rp. 10.000.000 (modal)
M1 = M + M(10%) = M(1 + 10/100)
= 10.000.000(11/10)
= Rp.11.000.000 (sofa baru)
keuntungan yg diinginkan sama 20%
M2 = M1 + M1(20%) = M1(1 + 20/100)
= 11.000.000(6/5)
= Rp.13.200.000

 9.600.000 + 960.000 = Rp.10.560.000,00 ⇒ MODAL KEDUA

Untung pada penjualan 1 = Rp.2.000.000
