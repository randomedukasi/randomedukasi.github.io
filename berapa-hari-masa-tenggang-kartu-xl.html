Penjelasan dengan langkah-langkah:

Diketahui

42 pekerja = 50 hari

n = 35 hari

Ditanya tambahan pekerja

Jawab

42 pekerja = 50 hari

n = 35 hari

42 x 50 = n x 35

2100 = 35 n

n = 2100 : 35

n = 60 pekerja

maka tambahan pekerjanya adalah 60 - 42 = 18 pekerja
