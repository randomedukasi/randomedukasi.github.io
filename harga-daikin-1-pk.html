4x² - 2x - 1 = 0
a = 4; b = -2 dan c = 1

x1 + x2 = -b/a = -(-2)/4 = 1/2
x1 . x2 = c/a = -1/4

PK Baru:
x² - [(x1+x2) + x1.x2]x + (x1+x2)(x1.x2) = 0
x² -[1/2 + (-1/4)]x + (1/2)(-1/4) = 0
x² -(1/4)x - 1/8 = 0                     (dikalikan 8)
8x² - 2x - 1 = 0

13. jika x1 dan x2 merupakan akar2 pk x^2+3x+1=0, maka pk yg akar2nya 1-x1 dan 1-x2 adalah


[tex](x+1)^{2}+3(x+1).(-1)^{2}+1.(-1)=0\\x^{2}+2x+1+3x+3-1=0\\x^{2}+5x+3=0[/tex]

x² + 3x + 1 = 0

x² - ( -3)x + 1 = 0

x1 + x2 = -3
→ 1 - x1 + 1 - x2 = -3
→ x1 + x2 = -3 - 2
→ x1 + x2 = -5

x1.x2 = 1
→ (1 - x1)(1 - x2) = 1
→ 1 - x1 - x2 - x1x2 = 1
→ -(x1 + x2) - x1x2 = 1 - 1
→ -x1x2 = 0 + x1 + x2
→ x1x2 = - ( x1 + x2 )
→ x1x2 = -(-5)
→ x1x2 = 5

persamaan kuadrat akar²nya 1 - x1 dan 1 - x2
adalah :

x² - (-5)x + 5 = 0

x² + 5x + 5 = 0

22. Jika alfaβ akar dari pk 3x2-5x+7=0 tenukan pk baru yg akar akarnya berkebalikan dengan pk yg diketahui 1/alfa&1 /beta


kalo berkebalikan rumus cepetnya a sama c
dibalik

jadi kalo ada ax² + bx + c = 0
persamaan barunya jadi cx² + bx + a = 0

jadi jawabannya 7x² - 5x + 3 = 0

tapi kalo mau pake cara manual juga boleh
3x² - 5x + 7 = 0, akar akarnya A sama B
A+B = -b/a = -(-5/3) = 5/3
A.B = c/a = 7/3

persamaan baru
X² - (X1 + X2)X + X1.X2 = 0

X1 = 1/A, X2 = 1/B

X1 + X2 = 1/A + 1/B
= (A + B)/AB
= (5/3)/(7/3)
= 5/7

X1.X2 = 1/A × 1/B
= 1/(A.B)
= 1/(7/3)
= 3/7


Maka persamaan kuadrat barunya-
x² - (5/7)x + 3/7 = 0
---------------------------×7
7x² - 5x + 3 = 0
