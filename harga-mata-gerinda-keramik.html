7. Sebuah gerinda berputar 240 putaran tiap 5 menit, maka fekuensi gerinda tersebut adalah....​


GMB

▶ Frekuensi

n = 240

t = 5 menit = 300 s

f = __ ?

frekuensi gerinda

f = n / t

f = 240 / 300

f = 0,8 Hz ✔

Frekuensi gerinda adalah [tex]\boxed{\bold{0,8 \: Hz}}[/tex]

Pembahasan

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi.

[tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode.

[tex]\bold{n}[/tex] = banyak getaran.

[tex]\bold{t}[/tex] = waktu.

Berarti :

[tex]\boxed{\bold{T= \frac{1}{f}}}[/tex]

Langkah Penyelesaian

Diketahui :

Banyak getaran = 240 kali.

Waktu = 5 menit = 300 detik.

Ditanya :

Frekuensi gerinda adalah?

Dijawab :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{240}{300}}[/tex]

[tex]\bold{f = 0,8 \: Hz}[/tex]

Kesimpulan : Jadi frekuensi gerinda adalah [tex]\boxed{\bold{0,8 \: Hz}}[/tex]

22. Sebuah kamar mandi berukuran 2m x 1,4m yang dindingnya akan ditutup keramik setinggi 1 meter. Jika ukuran keramik 20cm x 20cm dan harganya Rp12.000/buah, maka harga keramik seluruhnya adalah?


Luas permukaan balok adalah :

= 2 x [ ( panjang x lebar ) + ( panjang x tinggi ) + ( lebar x tinggi ) ]

Sedagkan untuk mencari volume balok adalah :

= luas alas x tinggi  

= panjang x lebar x tinggi

Pembahasan

Perhatikan gambar terlampir.

Untuk menghitung jumlah keramik yg dibutuhkan, maka terlebih dahulu hitung luas permukaan balok hanya pada bagian dinding nya saja.

Luas permukaan balok pada bagian dinding sebelah kanan sama dengan luas permukaan balok pada bagian dinding sebelah kiri. Luas permukaan kedua dinding ( kanan dan kiri ), yaitu :

= 2 x panjang x lebar

= 2 x 1,4 m x 1 m  

= 2,8 m²

Luas permukaan balok pada bagian dinding sebelah depan sama dengan luas permukaan balok pada bagian dinding sebelah belakang. Luas permukaan kedua dinding ( depan dan belakang ), yaitu :

= 2 x panjang x lebar

= 2 x 2 m x 1 m  

= 4 m²

Sehingga luas dinding seluruhnya yaitu :

= 2,8 m² + 4 m²

= 6,8 m²

Hitung luas keramik, yaitu :

= sisi x sisi

= 20 cm x 20 cm

= 400 cm²

> Ubah ke dalam satuan m²

> 1 m² = 10.000 cm²

= ( 400 : 10.000 ) m²

= 0,04 m²

Menghitung banyak keramik yg di butuhkan untuk di pasang di dinding kamar mandi yaitu :

= luas permukaan dinding : luas keramik

= ( 6,8 : 0,04 ) x 1 ubin

= ([tex] \frac {68} {10}[/tex] : [tex] \frac {4} {100}[/tex] ) x 1 ubin

= ([tex] \frac {68} {10}[/tex] x [tex] \frac {100} {4}[/tex] ) x 1 ubin

= ([tex] \frac {68} {10}[/tex] x 25 ) x 1 ubin

= [tex] \frac {68 x 25} {10}[/tex] ubin

= [tex] \frac {1.700} {10}[/tex] ubin

= 170 ubin

Jika harga ubin per buah adalah Rp 12.000 maka harga keramik seluruhnya adalah :

= 170 ubin x Rp 12.000

= Rp 2.040.000
