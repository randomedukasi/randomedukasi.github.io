Soal :
Selama 8 jam kereta argo murah dapat menempuh jarak 320 km. dengan kecepatan yang sama, jarak tempuh kereta argo murah selama 14 jam adalah 

Di Ketahui :
* Jarak = 320 Km 
* Waktu = 8 Jam
* K = 14 Jam

Di Tanya :
Jarak Tempuh Kereta Argo Murah selama 14 Jam => .. ?

** Rumus Mencari Jarak Tempuh Kereta Selama 14 Jam.
Yaitu :
{{ J => K × W }}

Pembahasan :

-> Langkah 1 
{{ J     => K × W }}
{{ 320 => K x 8 Jam }}
{{ K     => 320 / 8 Jam }}
{{ K     => 40 Km/Jam }}

-> Langkah 2 
{{ J     => K x W }}
{{ J     => 40Km/Jam x 14Jam }}
{{ J     => 560Km }}

Jadi, Jarak Tempuh Kereta Argo Murah selama 14 Jam => 560 Km
