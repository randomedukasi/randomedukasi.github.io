4. persamaan dan perbedaan dariExcel 2003 ,Excel 2008 ,dan Excel 2007????


Perbedaan ketiganya terdapat dalam penyimpanan ukuran file yang telah dibuat:
- Ms. Excel 2003: hanya menyediakan pilihan ekstensi xls (ukuran filenya besar)
- Ms. Excel 2007: menyediakan pilihan ekstensi xlsx (ukuran filenya kecil)
- Ms. Excel 2008: menyediakan pilihan ekstensi xlsx (lebih kecil ukurannya dari Ms. Excel 2007)

Persamaannya adalah ketiga program office tersebut dibuat oleh Microsoft untuk melakukan pengolahan data angka dengan mudah.

15. apa maksud dari excel binary workbook dalam excel


Excel Binary Workbook atau XLSB adalah salah satu file format/jenis file dari Excel untuk menyimpan Workbook/Buku kerja Excel yang berisi Macro dengan bentuk biner. Fungsinya agar file Workbook dengan Macro tersebut cepat diakses/dibukanya dan ukurannya lebih kecil ketimbang menyimpan dengan format lain

16. berapa persentase compress file jika datanya 300mb ke 100mb?​


(300 - 100) / 300 x 100% = 66.67%

Jadi, persentase kompresi dari file 300MB menjadi 100MB adalah 66.67%.

26. Pengertian baris pada excel Pengertian kolom pada excel Pengertian cell pada excel Pengertian range pada excel Pengertian sheet pada excel Pengertian marge pada excel Pengertian warp pada excel


Berikut nana-nama istiah dalam Microsoft Office Excel.
-   Baris adalah bagian kotak pada semua dokumen secara horizontal.
-   Kolom adalah bagian kotak pada semua dokumen secara vertikal.
-   Cell adalah salah satu kotak-kotak pada dokumen
-   Range adalah Jumlah kotak yang dipilih secara sekaligus.
-   Sheet adalah suatu lmbar kerja pada satu buku dokumen.
-   Merge adalah beberapa kotak/cel yang digabungkan menjadi satu.
-   Warp adalah pemberian baris baru pada salah satu kotak/cell.
