9. ~Susunan kata dari - The - Feels


Jawaban:

~ Permutation

_____________

THE

Total unsur = 3! Unsur ganda = -

P = 3.2.1

= 6 susunan ☑

FEELS

Total unsur = 5! Unsur ganda = 2!

P = 5.4.3.2.1/2.1

= 120/2

= 60 susunan ☑

SOLUTION :

Formula:

nPr =n!  / (n1! n2! . . . nk!)

____________

"THE"

n = 3Subsets : T = 1; H = 1; E = 1;n1(T) = 1, n2(H) = 1, n3(E) = 1

nPr formula

=3!  / (1! 1! 1! )

= 1 x 2 x 3  / {(1) (1) (1)}  

= 6 / 1

= 6

In 6 distinct ways, the letters of word "THE" can be arranged.

THE HTE ETH TEH HET EHT

_____________

"FEELS"

n = 5Subsets : F = 1; E = 2; L = 1; S = 1;n1(F) = 1, n2(E) = 2, n3(L) = 1, n4(S) = 1

nPr formula

= 5!  / (1! 2! 1! 1! )

= 1 x 2 x 3 x 4 x 5 / {(1) (1 x 2) (1) (1)}

= 120  / 2

= 60
