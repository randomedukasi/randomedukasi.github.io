4. besar sudut bca pada gambar adalah​


Sudut

180° + 70° - (2x - 30)° - (x + 40)° = 0

250° - 2x + 30° - x - 40° = 0

240° - 3x = 0

x = 80°

Besar sudut BCA

= 180° - (2x - 30)°

= 180° - (2.80° - 30°)

= 180° - 130°

= 50°

8. cara menautkan dana/ATM BCA siapp store. tolong dong kaa butuh bnget :(​


Jawaban:

Kamu dapat mengisi Saldo DANA melalui Bank BCA menggunakan 4 metode berbeda:

Melalui ATM BCA:

Masukkan kartu BCA dan PIN.

Pilih menu Transaksi Lainnya > Transfer > BCA Virtual Account.

Masukkan nomor Virtual Account DANA kamu (contoh: 390108981234567).

Masukkan jumlah pengisian.

Ikuti petunjuk selanjutnya untuk menyelesaikan transaksi.

Catatan:

Biaya admin akan dikenakan untuk kartu non-BCA sesuai ketentuan masing-masing bank. Minimum pengisian sebesar Rp10.000.

Melalui KlikBCA:

Login ke website KlikBCA

Pilih menu Transfer > Transfer ke BCA Virtual Account.

Masukkan nomor Virtual Account DANA kamu (contoh: 390108981234567).

Masukkan jumlah pengisian.

Ikuti petunjuk selanjutnya untuk menyelesaikan transaksi.

Catatan:

Minimum pengisian adalah Rp 10.000

Melalui aplikasi m-BCA:

Login ke aplikasi m-BCA

Pilih menu m-Transfer > BCA Virtual Account.

Masukkan nomor Virtual Account DANA kamu (contoh: 390108981234567).

Masukkan jumlah pengisian.

Ikuti petunjuk selanjutnya untuk menyelesaikan transaksi.

Catatan:

Minimum transaksi topup adalah Rp 10.000.

Via aplikasi m-BCA (Sim Tool Kit):

Pilih m-BCA pada Simcard.

Pilih menu m-Payment > others.

Ketik “TVA" lalu klik “ok”.

Masukkan nomor Virtual Account DANA kamu (contoh: 390108981234567).

Masukkan jumlah pengisian.

Masukkan PIN BCA kamu.

Kamu akan menerima SMS konfirmasi.

Catatan:

Minimum pengisian sebesar Rp10.000.

10. besar <BCA pada gambar ∆ABC dibawah adalah


Kelas : 7
Mapel : Matematika
Kategori : Garis dan Sudut

◼ Jumlah Sudut ∆ = 180°

2x + 3x + 20° = 180°
5x + 20° = 180°
5x = 180° - 20°
5x = 160°
x = 160°/5
x = 32°


Maka, Besar Sudut BCA
= 3x
= 3(32°)
= 96°


Opsi D
[tex] < a + < b + < c = 180 \\ 2x + 3x + 20 = 180 \\ 5x + 20 = 180 \\ 5x = 180 - 20 \\ 5x = 160 \\ x = 160 \div 5 \\ x = 32 \\ < c = 3x = 3 \times 32 = 96[/tex]

jawabannya yang D.96

< ABC = 200 - 3x

< BCA = 180 - x - 50
< BCA = 130 - x

jumlah sudut segitiga = 180

70 + ( 200 - 3x ) + ( 130 - x ) = 180
70 + 200 - 3x + 130 - x = 180
400 - 4x = 180
400 - 180 = 4x
220 = 4x

x = 55

besar sudut < BCA
< BCA = 130 - x
