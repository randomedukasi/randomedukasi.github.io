4. Diva menyimpan uang di bank "mandiri" sebesar Rp 800.000,- dengan bunga 12% pertahun.hitunglah bunga tabungan diva setelah 6 bulan!​


Dik :

Modal ( M ) = Rp 800.000

Persentase Bunga ( b ) = 12% = 12/100

Jumlah Bulan ( N ) = 6 Bulan

Dit :

Jumlah Bunga ( B ) = ?

Jawab :

B = n/12 × b × m

B = 6/12 × 12/100 × 800.000

B = 6 × 12 × 8.000

B = 72 × 8.000

B = 576.000
