Jawaban:

kartu A = 12.000

kartu B = 8.500

Penjelasan dengan langkah-langkah:

x = kartu A

y = kartu B

eliminasi ( mencari x )

3x + 2y = 53.000 | x1 | 3x + 2y = 53.000

2x + 1y = 32.500 | x2 | 4x + 2y = 65.000 -

-1x = -12.000

x = -12.000/-1

x = 12.000

subtitusi ( mencari y )

3x + 2y = 53.000

3(12.000) + 2y = 53.000

36.000 + 2y = 53.000

2y = 53.000 - 36.000

2y = 17.000

y = 17.000/2 = 8.500
