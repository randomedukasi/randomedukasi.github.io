28. satu aja gppp tp itu tiga tiganya blm bener semua tp sebisa nya aja​


1. Volume = 2574466 m³

2. Luas permukaan prisma = 108cm²

3. Volume = 8064cm³

Penjelasan dengan langkah-langkah:

1. Volume = ⅓ luas alas x tinggi

= ⅓ (230m)² x 146m

= 2574466 m³

2. Luas = 2 x luas △+ luas selimut prisma

* luas selimut = (AB+BC+AC) x tinggi

* AB = √(BC² + AC²) = 5cm

Luas = 2(½x3cmx4cm)+(3+4+5)cm x 8cm

= 12cm² + 96cm²

= 108cm²

3. Volume = ⅓ luas alas x tinggi

= ⅓ 32cm x 18cm x 42cm

= 8064cm³
