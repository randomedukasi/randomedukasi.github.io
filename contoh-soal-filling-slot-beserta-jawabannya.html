Bilangan eksponen adalah berapa kali sebuah bilangan dikali

Contoh soal:

Hasil dari 2³ × 2² berapa?

-----> 2³ × 2² = 2³ ÷ 2⁻² = 2³⁺² = 2⁵ = 32

Hasil dari 2³ ÷ 2² berapa?

-----> 2³ ÷ 2² = 2³ × 2⁻² = 2³⁻² = 2¹ = 2

Penjelasan dengan langkah-langkah:

Pendahuluan:

Bilangan eksponen adalah berapa kali sebuah bilangan (basis/pokok) dikali. Ada beberapa cara untuk menyederhanakan bentuk eksponen, tentu jika bilangan basisnya sama:

Aturan eksponen adalah sebagai berikut, jika:

--> nᵃ × nᵇ = nᵃ ÷ n⁻ᵇ = nᵃ⁺ᵇ

Contoh 2³ × 2² = 2³ ÷ 2⁻² = 2³⁺² = 2⁵ = 32

--> nᵃ ÷ nᵇ = nᵃ × n⁻ᵇ = nᵃ⁻ᵇ

Contoh 2³ ÷ 2² = 2³ × 2⁻² = 2³⁻² = 2¹ = 2

--> (nᵃ)ᵇ = (nᵇ)ᵃ = nᵃᵇ

Contoh (2³)² = (2²)³ = 2⁶ = 64

--> nᵃ/ᵇ = ᵇ√nᵃ (diubah ke bentuk akar)

Contoh 2[tex]^{\frac{3}{2}}[/tex] = ²√2³ = √2³ = √8 = √4√2 = 2√2

--> aⁿ × bⁿ = (a×b)ⁿ

Contoh 2³ × 4³ = (2×4)³ = 8³
