mesin B =y

2x+3y= 135.000 ..(pers 1)
x+2y= 80.000(pers 2)

2x+3y=135.000

= -y= -35.000
y= 35.000
*substitusi (pers 2)
x+2y= 80.000
x+2(35000) = 80000
x= 80000-70000
x=10000

ditanya
3x+y?

30.000+35.0000
= 75.000

21. gajih beni 115% dari gajih toni sedangkan gaji meri 90% dari gajih toni jumlah gaji ketiganya 6100.000.00 maka gajih meri adalah?


Gaji Toni dimisalkan x = 100/100 x

Gaji Beni = 115/100 x
Gaji Meri = 90/100 x

6.100.000 = 100/100 x + 115/100 x + 90/100 x
6.100.000 = 305/ 100 x
x = 6.100.000 . 100/305
x = 2000.000

maka gaji meri adalah 90/100 . 2000.000 = 1800.000Misal :
a = Gaji Beni
b = Gaji Toni
c = Gaji Meri

Dik :
a = 115% b
c = 90% b
a + b + c = Rp6.100.000,00

Dit : c

Jwb :
a + b + c = 6.100.000
115% b + b + 90% b = 6.100.000
115/100 b + b + 90/100 b = 6.100.000
115/100 b + 100/100b + 90/100 b = 6.100.000
(115+100+90)/100 b = 6.100.000
305/100 b = 6.100.000
61/20 b = 6.100.000
b = (6.100.000) : 61/20
b = (6.100.000) × 20/61
b = 2.000.000

c = 90% b
c = 90% (2.000.000)
c = 90/100 × 2.000.000
c = 1.800.000

Maka, gaji Meri adalah Rp1.800.000
