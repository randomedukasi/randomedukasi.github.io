8. 6. Suku ke-6 dan ke-20 barisan aritmatika berturut-turut adalah 21 dan 77. Suku ke-35 barisantersebut adalahA37C. 117D. 151​


Jawaban:

Suku ke-35 adalah 137

Penjelasan dengan langkah-langkah:

Dik : U6 = 21

U20 = 77

Dit : U35 = ?

Jawab :

U6 = a + 5b = 21

U20 = a + 19b = 77 _

-14b = -56

b = -56 / -14

b = 4

Karena b = 4, maka

a + 5b = U6

a + 5(4) = 21

a + 20 = 21

a = 21 - 20

a = 1

Un = a + (n - 1)b

U35 = 1 + (35 - 1)(4)

= 1 + (34)(4)

= 1 + 136

U35 = 137
