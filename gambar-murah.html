Jawaban:

B = S-600

2B +2S = 8.800

2(S-600) + 2S = 8.800

2S-1200 + 2S = 8.800

4S-1200 = 8.800

4S = 10.000

S = 2500

B = S- 600

B = 2500 - 600

B = 1.900
