Bab : Clothes

Jawaban:

baju : shirt  

Penjelasan:

------------------------CLOTHES---------------------------------MATERI         : CLOTHES PELAJARAN : BAHASA INGGRISKELAS           :  2 SD

Di dalam materi ini terdapat jenis-jenis pakaian untuk laki-laki juga pakaian untuk perempuan. Nama-nama pakaian sebaiknya kalian hafalkan supaya kosakata kalian semakin bertambah dan tentunya kalian akan semakin terampil dalam Bahasa Inggris.  

CLOTHES FOR BOY / MAN (PAKAIAN UNTUK LAKI-LAKI)

1. shirt : kemeja

2. tie : dasi

3. belt : sabuk

4. short : celana pendek

5. shoes : sepatu

6. socks : kaus kaki

7. sandals: sandal

8. hat : topi

9. T-shirt : kaos oblong

10. pants : celana panjang

11. pajamas : piyama / baju tidur

Contoh kalimat :

1. Dimas is wearing hat.

   (Dimas memakai topi.)

2. Mr.Agus is wearing tie.

   (Pak Agus memakai dasi.)

3. Danil is wearing belt.

   ( Danil memakai sabuk.)

CLOTHES FOR GIRL / WOMAN (PAKAIAN UNTUK PEREMPUAN)

1. gown : gaun

2. skirt : rok

3. sandals : sandal

4. shoes : sepatu

5. socks : kaus kaki

6. blouse : blus

7.pajamas : baju tidur / piyama

Contoh kalimat:

1. Anna  is wearing a gown.

    (Anna memakai gaun.)

2.  Diana is wearing a skirt.

   (Diana memakai rok.)

3. Riska is wearing blouse.

   (Riska memakai blus.)
