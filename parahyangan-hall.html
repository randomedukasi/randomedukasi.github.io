9. jarak Jakarta ke Bandung 180 km ditempuh dengan kereta api Parahyangan selama 3 jam kelajuan rata-rata kereta api Parahyangan itu adalah​


Untuk mencari kecepatan rata-rata (v) kereta api Parahyangan, kita dapat menggunakan rumus kecepatan:

v = jarak / waktu

Di mana:

jarak = 180 km (jarak dari Jakarta ke Bandung)

waktu = 3 jam (waktu yang ditempuh)

Dengan mengganti nilai jarak dan waktu ke dalam rumus, kita dapat menghitung kecepatan rata-rata kereta api Parahyangan, yaitu:

v = 180 km / 3 jam

v = 60 km/jam

Jadi, kecepatan rata-rata kereta api Parahyangan adalah 60 km/jam.

Penjelasan dengan langkah-langkah:

kelajuan rata-rata:

[tex] \sf \: = \frac{jarak \: total}{waktu \: tempuh} [/tex]

[tex] \sf \: = \frac{180}{3} [/tex]

[tex] \sf \small \: = 60 \: km/jam[/tex]
