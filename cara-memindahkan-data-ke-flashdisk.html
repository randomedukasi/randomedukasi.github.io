16. jabarkan tata cara untuk meng-copy atau memindahkan data atau dokumen dari flashdisk ke hardisk​


Jawaban:

Ada cara untuk memindahkan file ke flashdisk

Penjelasan:

1. Nyalakan komputer MU

2. Setelah booting selesai, kamu tekan file Explorer yang berada di taskbar atau ikon yang bernama "This PC"

3. Pilih Drive flashdisk yang diinginkan (misal : flashdisk F)

4. Klik folder yang tersimpan jpg, dokumen, lagu dll

5. Setelah itu, kamu pindahkan satu satu filemu ke harddisk, jika ingin cepat bisa menekan tombol CTRL di keyboard dan pilih file (tombol jangan dilepas)

6. Lalu, klik kanan dan klik Copy

7. Setelah itu beralih ke harddisk (misal : kita letakkan pada drive d)

8. Pilih tempat folder yang diinginkan

9. Pada folder kosong, klik kanan pada mouse atau touchpad dan klik Paste

10. Tunggu hingga prosesnya selesai

Semoga membantu


17. jelaskan tata cara untuk mengopi/ Memindahkan data/dokumen dari flashdisk ke harddisk! tolong di bantu waktu nya sebentar lagi ​


Jawaban:

Misalkan memakai komputer dg sistem operasi Windows:

1. masuk ke dalam Windows Explorer hingga muncul tampilan keterangan harddisknya

2. colokkan/ pasang flashdisk ke komputer, hingga muncul tampilan keterangan bahwa flashdisk sudah terdeteksi di Windows Explorer.

3. dobel klik flashdisk yg terdeteksi tadi untuk masuk & mencari file yg diinginkan.

4. klik kanan file yg diinginkan dan klik Copy

5. selanjutnya dobel klik bagian harddisk (contohnya C)

6. klik kanan area kosong harddisk dan klik Paste.

Penjelasan:

Amati contohnya gambar lampiran:

di sebelah kiri, ialah tampilan harddisk di Windows Explorer (Ada C, D, E)

di sebelah kanan, ialah tampilan flashdisk setelah dicolokkan komputer (terdeteksi sebagai F)

26. kenapa flashdisk dan flashdrive sudah jarang digunakan untuk memindahkan data dalam jumlah besar​


Jawaban :

Salah satu alasan mengapa flashdisk dan flashdrive sudah jarang digunakan untuk memindahkan data dalam jumlah besar adalah karena keterbatasan kapasitas penyimpanan dibandingkan dengan media penyimpanan lain seperti hard disk eksternal atau cloud storage. Flashdisk dan flashdrive umumnya memiliki kapasitas penyimpanan terbatas, misalnya hanya beberapa gigabyte atau puluhan gigabyte saja. Sedangkan hard disk eksternal dan cloud storage bisa memiliki kapasitas penyimpanan yang jauh lebih besar.

Selain itu, keterbatasan kecepatan transfer data juga menjadi alasan mengapa flashdisk dan flashdrive kurang disukai untuk memindahkan data dalam jumlah besar. Transfer data melalui flashdisk dan flashdrive bisa memakan waktu yang cukup lama jika jumlah data yang dipindahkan besar. Sementara itu, hard disk eksternal dan cloud storage bisa mentransfer data dengan lebih cepat.

Meskipun begitu, flashdisk dan flashdrive masih digunakan untuk keperluan yang lebih sederhana dan praktis, misalnya untuk menyimpan data pribadi seperti dokumen, foto, atau video kecil.

29. Cara menghapus data flashdisk yang tidak bisa dihapus ?


Saya biasa pake cara CMD windows langsung bro, caranya sebagai berikut :

1. Cari dan buka menu Command Prompt. Klik 'Start', lalu ketik Command Prompt di fitur pencarian atau bisa buka lewat RUN dengan menekan tombol "Win+R" lalu ketik "CMD"
2. Setelah Command Prompt muncul di daftar pencarian, klik kanan dan pilih 'Run as administrator'.
3. Masukkan perintah rmdir /s /q diikuti dengan path folder/data yang ingin dihapus. Contohnya, rmdir /s /q E:\test.
