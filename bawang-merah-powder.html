B. Putih : B. Merah = 5 : 7 (×2)

B. Merah : B. Bombay = 2 : 3 (×7)

B. Putih : B. Merah : B. Bombay

10 : 14 : 21

B. Bombay - B. Merah = Rp4200

B. Merah + B. Putih = ...?

= 14+10 / 21-14 × Rp4200

= 24/7 × 4200

= Rp14.400

14. perbandingan harga bawang putih dan bawang merah adalah 5:7,sementara itu perbandingan harga bawang bombay dan bawang merah 3:2.selisin harga bawang bombay dengan bawang merah Rp.200.000,00.berapa jumlah harga bawang merah dan bawang putih brainly


Bawang putih = a
bawang merah = b
bawang bombay = c

a : b = 5 : 7
c : b = 3 : 2
c - b = 2000

Jumlah harga bawang bombay = c / (c-b)  x 2000 = 3 / (3-2)  x 2000 = 3000
Jumlah harga bawang merah = b/ (c-b) x 2000 = 2 / (3-2) x 2000 = 4000
Jumlah harga bawang putih = a / b  x 4000 = 5 / 7 x 4000 = 2857,14

.......BM : BB = 2 : 3

= 24 x 600
= Rp14.400,00

Mata Pelajaran = Matematik.
Kelas = VI.
Bab = Perbandingan.

Penyelesaiannya adalah.

Bawang Putih : Bawang Merah
5 : 7
Bawang Merah : Bawang Bombay
2 : 3

Bawang Putih = 10
Bawang Merah = 14
Bawang Bombay = 21

Bawang Putih : Bawang Merah :Bawang Bombay = 10 : 14 : 21

= (10 + 14 )/( 21 - 14 ) × 4.200
= 24/7 × 4.200
24 × 4.200 ÷ 7 = 14.400
