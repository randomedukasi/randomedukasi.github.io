11. Sebuah museum memberlakukan tiket masuk untuk dewasa dan anak-anak. Pada suatu hari terjual tiket masuk sebanyak 884 lembar. Perbandingan Antara banyak tiket masuk pengunjung dewasa dan tiket masuk anak-anak adalah 1 : 3. Jika x menyatakan banyak tiket dewasa yang terjual dan y menyatakan banyak tiket masuk anak-anak yang terjual, persamaan matriks dari permasalahan tersebut adalah.


Persamaan matriks dari permasalahan tersebut adalah:

[tex]\left[\begin{array}{cc}1&1\\3&-1\end{array}\right] \left[\begin{array}{c}x\\y\end{array}\right]=\left[\begin{array}{c}884\\0\end{array}\right][/tex]

Apabila dilakukan perhitungan maka akan diperoleh nilai x (tiket untuk dewasa) berjumlah 221, sedangkan nilai y (tiket untuk anak-anak) adalah 663 tiket.

Penjelasan dengan langkah-langkah

Matriks merupakan suatu bentuk penyusunan bilangan atau persamaan dalam bentuk baris dan kolom. Apabila diketahui dua persamaan linier:

ax + by = c
dx + ey = f
Di mana a, b, c, d, e, dan f adalah suatu bilangan.

Maka bentuk persamaan matriksnya adalah:

[tex]\left[\begin{array}{cc}a&b\\d&e\end{array}\right] \left[\begin{array}{c}x\\y\end{array}\right]=\left[\begin{array}{c}c\\f\end{array}\right][/tex]

Diketahui:

Jumlah tiket masuk untuk dewasa = x.Jumlah tiket masuk untuk anak-anak = y.Jumalh tiket masuk total = 884.Perbandingan tiket masuk dewasa : tiket masuk anak- anak sama dengan 1 : 3.

Ditanyakan:

Persamaan matriks = ?

Penyelesaian:

Persamaan pertama:

Dilihat dari jumlah tiket total:
x + y = 884 ...(1)Dilihat dari perbandingan:
x : y = 1 : 3
3x = y
3x - y = 0 ...(2)Maka nilai:
a = 1
b = 1
c = 88
d = 3
e = -1
f = 0

Bentuk matriks dari soal:

[tex]\left[\begin{array}{cc}1&1\\3&-1\end{array}\right] \left[\begin{array}{c}x\\y\end{array}\right]=\left[\begin{array}{c}884\\0\end{array}\right][/tex]

Wangsulan

Ing ngisor iki sing kalebu jinise ater-ater yaiku (kajaba) ...

➡️ B. Ater-ater la-/

PenjelasanKajaba tegese kecuali.

{Ater-ater}

→ yaiku wuwuhan kang manggon ing sangarepe tembung.

1. Ater-ater anuswara (n, m, ny, ng). Ater-ater iki gunane ngubah tembung lingga dadi tembung kriya.

» Tuladhane :

ny + sapu → nyapun + tulis → nulisng + langi → nglangim + pacul → macul

2. Ater-ater tripurusa (dak, ko/kok, di). Ater-ater tripurusa gunane kanggo tembung sesulih aku (dak) kowe (ko) lan dheweke (di).

» Tuladhane :

dak+tulis → daktuliskok+pangan → kokpangandi+pancing → dipancing

3. Ater-ater liyane kayata sa, pa, pi, ka, kuma, kami, a, ma, lan sapanunggalane.

» Tuladhane :

satus → sa + atuspanulis → pa + nulispituduh → pi + tuduhkapidek → ka + pidekkumawani → kuma + wanikamitenggeng → kami + tenggengmanepi → ma + nepi
