18. Apa isi fristboot ketika ingin instal Windows 10​


Jawaban:

"Boot bersih" memulai Windows dengan satu set driver minimal dan program startup, sehingga Anda dapat menentukan apakah program latar belakang mengganggu game atau program Anda. Hal ini mirip dengan memulai Windows dalam mode Aman, tetapi memberikan Anda lebih banyak kontrol atas layanan dan program mana yang berjalan pada startup untuk membantu Anda mengisolasi penyebab masalah.

Penjelasan:

Cara untuk melakukan boot bersih

Sekilas langkah-langkah ini mungkin terlihat rumit, namun mengikuti secara berurutan, langkah demi langkah, akan membantu mengembalikan Anda ke jalur yang tepat.

Gunakan langkah-langkah berikut untuk melakukan boot bersih di Windows 10. Melakukan boot bersih untuk Windows 7, Windows 8, dan Windows 8.1

Masuk ke komputer sebagai administrator. Jika Anda tidak memiliki akun administrator, Anda dapat membuatnya. Membuat pengguna lokal atau akun administrator di Windows 10.

Di kotak pencarian pada taskbar, ketik msconfig kemudian pilih Konfigurasi Sistem dari daftar hasil.

Search result - System Configuration

Penting Jika komputer terhubung ke jaringan, pengaturan kebijakan jaringan mungkin mencegah Anda melakukan langkah-langkah berikut. Hanya gunakan utilitas Konfigurasi Sistem untuk mengubah opsi boot lanjut di komputer dengan panduan dari teknisi dukungan Microsoft. Menggunakan utilitas Konfigurasi Sistem mungkin membuat komputer tidak dapat digunakan.

1. Pada tab Layanan dari Konfigurasi Sistem, pilih Sembunyikan semua layanan Microsoft, kemudian pilih Nonaktifkan semua.

System Configuration - Services tab - Hide all Microsoft services check box checked - Disable all

2. Pada tab Memulai dari Konfigurasi Sistem, pilih Buka Pengelola Tugas.

System Configuration - Startup tab - Open Task Manager button

3. Di bawah Startup pada Pengelola Tugas, untuk setiap item startup, pilih item kemudian pilih Nonaktifkan.

4. Task Manager - Startup tab

5. Tutup Manajer Tugas.

6. Pada tab Startup di Konfigurasi Sistem, pilih OK. Saat Anda memulai ulang komputer, Anda akan berada di lingkungan boot bersih. Ikuti langkah pemecahan masalah untuk menginstal, mencopot instalan, atau menjalankan aplikasi Anda. Komputer Anda mungkin akan kehilangan beberapa fungsionalitas sementara di lingkungan boot bersih. Mengatur ulang komputer untuk memulai secara normal setelah pemecahan masalah boot bersih dan melanjutkan fungsionalitas

21. Bagaimana cara menganti motherboard tanpa instal ulang Windows 10?


Jawaban:

Bisa

Penjelasan:

1.) Hidupkan PC.

2.) Setelah powering up komputer Anda, tekan "F8" berulang kali. Muncul pilihan pilih aja "Safe Mode", di antara pilihan lain.

3.) Pilih "Safe Mode".

4.) Ketika komputer Anda telah boot ke layar desktop, layar Anda akan memiliki pemberitahuan bahwa Anda berada dalam safe mode. Jika Anda mendapatkan pesan yang memungkinkan Anda untuk "... klik OK untuk melanjutkan", klik ok.

5.) Berikutnya, anda perlu masuk ke "Device Manager", dan uninstall semua driver / perangkat lunak yang berhubun den motherboard lama Anda.

6.) Klik pada menu "Start"

7.) Klik "Control Panel" (Jika dalam "View Category", klik "Performance and Maintenance", kemudian klik "System" icon. Jika dalam "Classic View", double klik "System" icon.)

8.) Klik pada "Hardware" tab.

9.) Klik pada "Device Manager".

11.) Klik kanan semua sub kategori untuk perangkat keras yang terdaftar di bawah, dan klik "Uninstall". Ketika "Uninstall" tidak bisa, pilih "Disable". Kategori-kategori yang harus dipilih dan subkategori dihapus (atau dinonaktifkan) hak paten nya dibawah ini harus clear :

"IDE ATA / ATAPI Controllers"

"Network adapter"

"Sound Driver dan Game Controller"

"Universal Serial Bus Controller"

"VGA Adapter"

Setelah kelompok perangkat keras di atas dihapus atau dinonaktifkan, sekarang saatnya untuk menghapus perangkat lunak yang berhubun den hardware / driver yang dihapus.

12.) Pilih menu "Start", kemudian klik "Control Psayal." Berikutnya, klik da "Add or Remove Programs." Uninstall software APAPUN yang berhubun den perangkat keras yang akan diti. (Misalnya, jika Anda memiliki NVIDIA kartu vga onboard pada motherboard lama Anda, dan Anda melihat software untuk: "NVIDIA Control Psayal", HAPUS Jika anda tidak yakin apa yang harus dihapus SKIP LANGKAH INI, dan hubungi produsen yang berpengalaman,. atau costumer service (sekarang lah tinggal bikin thread di kaskus udah dibantu)

13). Selanjutnya, saatnya untuk mematikan komputer anda dan mempersiapkan diri untuk menganti motherboard baru. Sekarang perangkat lunak lama dan driver tidak boleh bertentan den perangkat keras baru, dan driver yang akan diinstal setelah pengtian.

Sumber : https://kevinsource.blogspot.com/2013/04/ganti-motherboard-tanpa-install-ulang.html?m=1

24. cara menginstal windows 2010


Boot komputer anda dengan media instalasi windows 8 Developer. Mungkin butuh beberapa menit untuk memuat file-file, dan kemudian akan membawa Anda ke layar instalasi. Pilih pilihan yang sesuai dan klik Next. Sekarang klik pada “Install Now” untuk melanjutkan centang “Accept the terms and conditions” dan kemudian klik “Next”. Disini kita bisa memilih opsi apakah ingin meng-upgrade windows kita ke windows 8 Edisi Developer atau ingin melakukan instalasi bersih. Pilih Custom (Advanced) untuk melanjutkan instalasi bersih. Pada layar ini Anda dapat memilih drive yang ingin Anda instal, di sini kita memiliki kemampuan untuk mengelola disk, format, membuat partisi dll Jika Anda berencana untuk men-setup Dual boot maka Anda dapat memilih drive yang berbeda. Setelah Anda mengkonfigurasi drive klik “Next” Itu saja sekarang akan dimulai proses instalasi, tunggu sampai selesai yang mengambil waktu 10 menit sampai 1 jam tergantung pada konfigurasi Hardware. Setelah itu menyelesaikan instalasi Komputer Anda akan reboot dan akan mempersiapkan pengaturan2. Jadi silahkan menunggu beberapa saat. Sekarang akan membawa Anda untuk mempersonalisasi pengaturan desktop Anda. Silahkan pilih “Express settings” yang pada layarnya tertera keterangan tentang apa saja settingan akan dilakukan. Tidak seperti sistem operasi lain, pada Windows 8 Anda dapat login menggunakan account Windows Live. Jadi, Anda dapat memasukkan informasi account Windows Live Anda. Jika tidak ingin menggunakan metode login ini, silahkan klik pada “I don’t want to log in with a Windows Live ID”. Anda dapat memilih “Local account” untuk membuat Username dan Password untuk login ke Windows Anda. Setelah semua informasi telah dimasukkan, klik “Next”. Sekarang Windows akan mengkonfigurasi pengaturan Anda Setelah semuanya di setup, maka kita akan disuguhkan tampilan Desktop Windows 8.

27. Cara menginstal windows 8.1


Pertama, siapkan media instalasi Windows 8.1 (DVD) lalu boot dari DVD tersebut. Biasanya menu booting bisa diakses dengan menekan tombol ESC berulang kali sesaat sebelum proses booting berlangsung.
Kedua, Setelah booting dari DVD Windows 8.1, tampilan Windows Setup akan muncul. Pilih bahasa, waktu, dan keyboard lalu tekan Next. WinPoin membiarkan tetap default (English United States)
KETIGA, Setelah itu pilih Install Now
4. Masukkan key original Windows 8.1 di form yang disediakan dan klik Next
5. Pada window Software License Agreement, centang “I accept the license terms” lalu klik Next
6. Jika kamu akan melakukan upgrade dari versi Windows sebelumnya ke Windows 8, pilih opsi 1. Jika ingin melakukan fresh install Windows 8.1, pilih opsi 2.
7. Sekarang kamu tinggal memilih partisi yang akan diinstal Windows 8.1. Kamu bisa membuat partisi baru atau memformatnya sesuai kebutuhan. Jika sudah, klik Next
8. Proses instalasi akan berlangsung cukup lama, jadi kamu harus sabar menunggu hingga proses instalasi selesai.
9. Di akhir proses instalasi ada menu pengaturan personalisasi. Masukkan nama PC dan pilih warna sesuai keinginan, lalu klik Next.
10. Pada menu pengaturan, pilih Express Settings untuk menggunakan peraturan default
11. Untuk mendapatkan seluruh manfaat dari Windows 8.1, WinPoin sarankan kamu login menggunakan Microsoft Account. Jika belum ada, kamu bisa mendaftar akun Outlook.com terlebih dahulu.
