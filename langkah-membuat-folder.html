6. bagaimana langkah langkah membuat subfolder di dalam folder


ejatinya memang dari dulu hp s60v2 tidak mendukung membuat folder didalam folder. Bisanya hanya diluar atau dimenu utama, tidak seperti hp java yang lain.Nah disini ada tips untuk membuat folder didalam folder. Alat-alat yang diperlukan: 1. X-plore atau fileman atau seleq. 2. Pilih Aplikasi apa saja. Contoh : UltraMp3 Pelaksanaan: 1. Buat dulu folder baru dangan nama apa bebas terserah kamu. Misalkan, nama foldernya BROWSER. 2. Pindahkan folder BROWSER tersebut ke posisi paling atas disamping kiri (Posisi ke 1). 3. Berhubung aku kasih contoh aplikasinya UltraMp3 , maka pindahkan aplikasi UltraMp3 ke posisi tengah atas (Posisi ke 2). 4. Kemudian, buka FileMan atau sejenisnya dan masuk ke E:/ system/ apps/ UltraMp3. 5. Rename atau ganti nama UltraMp3 menjadi nama apa saja. Contoh : UltraMp31 6. Setelah itu cepat-cepat keluar atau tekan tombol menu. Dan langsung Sorot aplikasi UltraMp3 yang ada diposisi kedua tadi, lalu tekan pilihan. 7. Tunggu kurang lebih 10 detik sampai tanda penyorot yang sedang dipakai untuk menyorot UltraMp3 tadi pindah kepojok kiri atas dan menyorot ke folder BROWSER yang kita buat tadi. 8. Setelah itu pilih MOVE TO FOLDER (Pindah Ke Folder). Terserah kamu mau dimana saja. Kalau folder yang dibuat tadi ingin ditaruh di folder Ekstra, Maka pilih Ekstra. 9. Sekarang coba kamu buka folder Ekstra. Disana pasti ada folder BROWSER yang kita buat tadi. 12. langkah langkah membuat folder adalah klik


Dengan dokumen yang terbuka, klik File > Simpan Sebagai.

Di bawah Simpan Sebagai, pilih tempat untuk membuat folder baru Anda. Anda mungkin perlu mengklik Telusuri atau Komputer, dan menavigasikan ke lokasi folder baru.

Dalam kotak dialog Simpan Sebagai yang terbuka, klik Folder Baru.

Klik Folder Baru dalam kotak dialog Simpan Sebagai.
Ketikkan nama folder baru Anda, lalu tekan Enter.

Ganti nama folder baru.
Catatan: Anda tidak dapat menggunakan garis miring, titik dua, titik koma, garis pisah, atau titik pada nama folder.

Klik Simpan. Dokumen Anda disimpan ke folder baru.1. Buka File di Laptop Atau komputer2. Klik Mouse Sebelah kanan , lalu klik New , Dan Klik Folder3. Tulis Nama Folder4. Enter
semoga Bermanfaat :)

17. jelaskan Langkah-langkah dalam: 1. Membuat folder 2. Menghapus Folder 3. Menggandakan Folder 4. Menyalin Folder 5. Memberi nama Folder


Jawaban:

Membuat folder :

1. Klik kanan mouse

2. Arahkan kursor ke pilihan new

3. Ketika pilihan lain muncul, klik folder.

Menghapus Folder :

1. Klik kanan mouse pada folder

2. Klik delete

Atau :

1. Pilih folder dengan menekan kiri mouse

2. Klik delete

Menggandakan folder :

1. Klik kanan mouse pada folder

2. Klik copy

3. Klik paste

Menyalin folder :

1. Klik kanan mouse pada folder

2. Klik copy (ctrl + c)

Memberi nama folder :

1. Buat folder baru

2. Beri nama

Kalau mau mengubah nama foldernya, lakukan dengan cara :

1. Klik foldernya dua kali di nama folder itu

2. Tulis nama folder yang baru

3. Atau gunakan rename

Kelas : 7

- Kemudian pilih My Computer dan pilih partisi disk/tempat yang ingin kita buat folder baru (contoh kita pilih Local Disk D)
- Lalu, arahkan kursor ke bidan kosong, kemudian klik kanan, lalu klik/pilih New dan pilih Folder dan muncullah folder dengan nama New Folder
