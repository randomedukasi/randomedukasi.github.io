3. Pendidikan Non-Formal

Pendidikan non-formal adalah lembaga pendidikan yang dilaksanakan di luar keluarga, sepertilembaga kursus.

17. Jabarkan mengenai jenis jenis wilayah!


Jawaban:

Materi Geografi

Wilayah (region) didefinisikan sebagai suatu unit geografi yang dibatasi oleh kriteria tertentu dan bagian-bagiannya tergantung secara internal.

Wilayah dapat dibagi menjadi 4 jenis:

1)Wilayah Homogen

Wilayah yang dipandang dari aspek/kriteria mempunyai sifat-sifat atau ciri-ciri yang relatif sama.

2)Wilayah Nodal

Wilayah Nodal (Nodal Region) adalah wilayah yang secara fungsional mempunyai ketergantungan antara pusat (inti) dan daerah belakangnya (hinterland).

3)Wilayah Administratif

Wilayah administratif adalah wilayah yang batas-batasnya ditentukan berdasarkan kepentingan administrasi pemerintahan atau politik, seperti provinsi, kabupaten, kecamatan, desa/kelurahan, dan RT/RW.

4)Wilayah Perencanaan

Boudeville (dalam Gloson, 1978) mendefinisikan wilayah perencanaan (planning region atau progamming region) sebagai wilayah yang memperlihatkan kohorensi atau kesatuan keputusan-keputusan ekonomi.

22. Apabila kita ingin memulai merintis bisnis toko online,apa jenis hosting yang cocok kita sewa?


Apabila kita ingin memulai merintis bisnis toko online,apa jenis hosting yang cocok kita sewa?

Sebaiknya dalam berbisnis selalu dilakukan secara terukur, khususnya berkaitan dengan pengeluaran biaya atau pun investasi pada bisnis.

Jika memang baru memulai dan belum memiliki pelanggan atau masih sangat sedikit, sebaiknya bergabung pada market-place yang sesuai dengan produk atau jasa yang kita bisniskan serta target pasar yang jadi sasaran.

Jika telah mulai memiliki pelanggan tetap dan terus bertumbuh, membangun website sendiri sangat disarankan. Dalam kondisi ini, ada beberapa alternatif yang bisa dilakukan sbb;

1. cukup menyewa domain saja, hosting menggunakan layanan gratisan.

seperti wordpress atau bisa juga sejenis infinityfree.net yang menyediakan cpanel dan akses database mysql gratis selamanya (selagi web tersebut diakses setiap bulan). Ini kombinasi yang paling murah.

2. menyewa domain dan shared hosting

jika menggunakan opsi ini, maka wajib mengamati tingkat pertumbuhan akses ke domain dan server. Agar bisa menentukan kapan melakukan upgrade ke level hosting yang lebih cepat sesuai dengan pertumbuhan bisnis.

Bila bisnis semakin berkembang dan traffic ke server telah mendekati batasan shared hosting, barulah melakukan peralihan ke Virtual Private Server atau Cloud hosting yang butuh biaya dan investasi lebih tinggi.

Dengan pendekatan ini biaya dan investasi yang dikeluarkan sepadan dengan kebutuhan sistem bisnis dan kemampuan menutupi biaya operasionalnya.

28. Ada 3 jenis Server Hosting, yakniServer Hosting USA (Los Angeles)Server Hosting Indonesia (Server IIX)Server Hosting Singapura (SG)Tugas !!!Carilah Karakteristik ketiga jenis server hosting di atas !​


Server Hosting Indonesia berarti komputer web server yang digunakan untuk menyimpan data website / datacenter terletak di Indonesia (Jakarta).

Secara fisik, server yang berada di datacenter yang berlokasi di Indonesia atau biasa disebut Indonesia Internet Exchange (IIX).

Tidak terpengaruh oleh kondisi konektivitas internasional dari ISP Indonesia yang digunakan pengguna.

Lebih cepat diakses dari dalam negeri (Indonesia) karena routingnya lebih pendek.

Sangat cocok jika Anda menargetkan hanya pengguna yang ada di Indonesia.

Server Hosting Singapore berarti komputer web server yang digunakan untuk menyimpan data website / datacenter terletak di Singapore.

Hosting server Singapore artinya secara fisik, komputer atau server yang berada di datacenter yang berlokasi di Singapore. Kita ketahui bahwa akses internet Singapore lebih baik dibandingkan negara tetangga sehingga lebih cepat diakses di seluruh negara ASIA.

Bisa menjembatani antara server di IIX dengan server US sehingga relatif cepat ketika diakses, baik dari Indonesia, Asia maupun US.

Lebih cepat Singapore daripada Indonesia. Maksudnya ketika diakses dari kawasan Asia Pasifik lebih cepat server Singapore.

Terpengaruh oleh kondisi konektivitas jaringan internasional dari ISP Indonesia yang digunakan pelanggan.

Server Hosting USA artinya komputer web server yang digunakan untuk menyimpan data website / datacenter terletak di USA.

Cepat diakses dari seluruh dunia karena pusat server internet dunia ada di USA.

Cocok untuk website yang pengunjung utama berasal dari dunia internasional.

Terpengaruh oleh kondisi konektivitas jaringan internasional dari ISP Indonesia yang digunakan pelanggan.

Pengiriman dan penerimaan e-mail cepat atau real time karena routing ke mail server relatif lebih pendek, misal ke Yahoo, Hotmail, Gmail, dan pengguna server berbasis di USA lainnya.
