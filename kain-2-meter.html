17. Seorang penjahit membutuhkan 2 meter kain A,1 meter kain B dan 3 meter kain C yang dibeli seharga 106.000,untuk membuat gorden model pertama.sementara untuk membuat gaun dibutuhkan 2 meter kain B dan 2 meter C yang dibeli seharga 64.000,penjahit itu membeli kain tambahan untuk pesanan tambahan yaitu 3 meter kain A,2 meter kain B seharga 90.000,.Berapakah harga setiap meter kain A,B,dan C?


Harga kain A tiap meter = Rp20.000,00

Harga kain B tiap meter = Rp15.000,00

Harga kain tiap meter = Rp17.000,00

Pembahasan

Sistem persamaan linear tiga variabel adalah suatu persamaan Matematika yang terdiri atas 3 persamaan linier yang masing - masing persamaannya memiliki tiga variabel (biasanya x, y, dan z) dengan 1 penyelesaian.

Bentuk umum sistem persamaan linear 3 variabel adalah:

[tex]ax + by + cz = d[/tex]

dimana

a adalah koefisien dari variabel x.

b adalah koefisien dari variabel y.

c adalah koefisien dari variabel z.

d adalah konstanta.

=======================================

Analisa soal

Diketahui:

Seorang penjahit membutuhkan 2 meter kain A, 1 meter kain B dan 3 meter kain C yang dibeli seharga 106.000 untuk membuat gorden model pertama.

Untuk membuat gaun dibutuhkan 2 meter kain B dan 2 meter C yang dibeli seharga 64.000.

Penjahit itu membeli kain tambahan untuk pesanan tambahan yaitu 3 meter kain A, 2 meter kain B seharga 90.000.

Ditanya:

Berapakah harga setiap meter kain A,B,dan C?

Dijawab:

Persamaan linear:

kain A = x

Kain B = y

Kain C = z

[tex]2x + y + 3z = 106.000[/tex] .................(1)

[tex]2y + 2z = 64.000[/tex]  .................(2)

[tex]3x + 2y = 90.000[/tex]  .................(3)

Langkah 1. Sederhanakan persamaan 2 dan substitusikan nilai y ke dalam persamaan 1

2y + 2z = 64.000 : 2

y + z = 32.000

[tex]y = 32.000 - z[/tex] .................(4)

↓

2x + y + 3z = 106.000

2x + 32.000 - z + 3z = 106.000

2x + 32.000 + 2z = 106.000

2x + 2z = 106.000 - 32.000

[tex]2x + 2z = 74.000[/tex]  .................(5)

Langkah 2. Eliminasi persamaan 2 dan 3

3x + 2y = 90.000

2z + 2y = 64.000 -

[tex]3x - 2z = 26.000[/tex] .................(6)

Langkah 3. Eliminasi persamaan 5 dan 6

2x + 2z = 74.000

3x - 2z = 26.000 +

5x         = 100.000

x           = [tex]\frac{100.000}{5}[/tex]

            = 20.000

Langkah 4. Substitusikan nilai x ke dalam persamaan 3

[tex]3x + 2y = 90.000[/tex]

3(20.000) + 2y = 90.000

60.000 + 2y = 90.000

2y = 90.000 - 60.000

y = [tex]\frac{30.000}{2}[/tex]

y = 15.000

Langkah 5. Subtitusikan nilai y ke dalam persamaan 2

[tex]2y + 2z = 64.000[/tex]

2(15.000) + 2z = 64.000

30.000 + 2z = 64.000

2z = 64.000 - 30.000

2z = 34.000

z = [tex]\frac{34.000}{2}[/tex]

z = 17.000

Maka,

kain A = x = Rp20.000,00

Kain B = y = Rp15.000,00

Kain C = z = Rp17.000,00
