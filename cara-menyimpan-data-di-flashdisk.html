5. bagaimana suatu benda mati seperti flashdisk bisa menyimpan data? apa yang terjadi di dalam chip yang sangat kecil itu??


Sebelumnya, kk tau binary nda?
Data komputer itu sebenarnya cuma terdiri dari 0 sama 1.
Misalnya angka 2 itu sebenarnya diartikan komputer 0010.
1 itu klo ada elektron di situ.
0 itu klo sedikit/tidak ada elektron.

Sama halnya dengan chip flashdisk.
Chip flashdisk punya cell2 yg terdiri dari floating gate sm control gate.
Floating gate mengandung elektron yang banyak, pada flash disk kosong, nilai cellnya 1 semua.

Kalau kk mau masukin data angka 2 misalnya, komputer bakal ngasih aliran listrik ke flashdisk kk, sehingga mendorong elektron dari floating gate menuju ke antara floating sm control gate dan terperangkap di situ.
Karena elektronnya hilang, cell itu bernilai 0. Kalo dilakuin 3 kali, kan bisa mbentuk 0010, nah itu kan data buat angka 2 :)

Prinsip kerja USB (Universal Serial Bus)

Bus itu sendiri memiliki koneksi yang biasa disebut koneksi “host/slave“, yang artinya PC yang mengelola semua transfer dan periferal hanya dapat meresponnya. Sebuah pengontrol host USB pada PC–biasanya terintegrasi pada chipset motherboard generasi terbaru–mengatur semua pengiriman dan penerimaan data antara PC dan perangkat eksternal. Sebuah kabel berisi empat buah kawat menghubungkan periferal ke PC melalui port USB yang terdapat pada keduanya. Di dalam kabel tersebut, dua kawat akan menangani transmisi data, sebuah lagi menangani ground dan sebuah lagi memasok daya sebesar lima volt ke periferal.

Transfer data pada USB dikirim dalam bentuk paket, sementara port paralel dan serial mentransfernya dalam bentuk bit individual. Sebagai contoh, bila Anda ingin menyimpan sebuah file pada sebuah drive Zip USB, pertama-tama PC akan memotong-motong file tersebut menjadi potongan-potongan sebesar 64-byte–setiap potongan menyertakan informasi pengalamatan dan data itu sendiri–dan kemudian mengirimkan potongan tersebut ke port USB

21. 33. Pernyataan yang salah dari perbedaan antara harddisk dan flashdisk berikut, adalah…. A. Harga hardisk tentunya jauh lebih mahal dibandingkan dengan harga flashdisk B. Flashdisk adalah penyimpan data internal, Harddisk penyimpan eksternal C. Kelebihan dari Harddisk pembacaan data lebih cepat D. Flashdisk dirancang untuk penyimpanan data portable E. Saat ini kapasitas penyimpanan Harddisk masih lebih besar dari kapasitas penyimpanan Flashdisk


Pernyataan yang salah dari perbedaan antara harddisk dan flashdisk dari ke 5 pernyataan yang tertera pada soal adalah b. Flashdisk adalah penyimpan data internal, harddisk penyimpan data eksternal.

Pembahasan

Harddisk ( atau Hard drive )

Adalah alat pembaca yang dibuat pada tahun 1990-an yang digunakan untuk menyimpan data namun pada saat ini juga bisa membaca data pada sebuah piringan kecil ( atau yang disebut Cakram atau CD ). Harddisk berbentuk seperti driver mesin ( lihat lampiran ) dan memiliki ukuran yang bervariasi. Harddisk dapat menyimpan data hingga 4TB lebih.

Flashdisk ( atau Flash Drive )

Adalah alat pembaca dari tahun 2006 yang merupakan hasil modifikasi dari Hard drive dengan lebih banyak keunggulan. Flashdisk dapat dibaca menggunakan menusukkan alat ini ke usb slot yang tersedia. Flashdisk dapat menyimpan data hingga 512 GB namun Flashdisk hanya dapat menahan data hingga 15 tahun.

Pembahasan masing-masing pilihan gandaa. Harga harddisk tentunya jauh lebih mahal dibandingan harga flashdisk

Dikarenakkan perbedaan kualitas, fungsi dan besar penyimpanan barang maka Harddisk jauh lebih mahal ( 500gb untuk Rp230.000 ) daripada flashdisk ( 64gb untuk Rp 15.000) . ( Benar )

b. Flashdisk adalah penyimpanan data internal, Harddisk penyimpan eksternal

Karena kapasitasnya yang kecil serta fungsinya sebagai memindahkan data, Flashdisk tidak dapat digunakkan sebagai penyimpanan data internal. Meskipun Harddisk pada umumnya digunakkan sebagai penyimpanan data eksternal, Harddisk terkadang juga digunakkan untuk menyimpan data internal. ( Salah )

** Internal, alat untuk menyimpan data namun sulit untuk dipindahkan. Eksternal, alat untuk menyimpan data dan mudah untuk dipindahkan.

c. Kelebihan dari Harddisk pembacaan data lebih cepat

Salah satu yang membuat harddisk mahal adalah kecepatan membaca data. Harddisk mampu transfer data pada kecepatan 6 gbps atau lebih, sedangkan flashdisk mampu transfer data pada kecepatan kurang dari 272 mbps ( Benar )

**Gbps : Giga bit per second, mbps : Mega bit per second

d. Flashdisk dirancang untuk penyimpanan data portable

Definisi data portable itu sendiri ialah "Penyimpanan yang mudah dibawa", flashdisk merupakkan salah satu contoh alat tersebut karena dapat disimpan dalam kantong celana ataupun tempat lainnya. ( Benar )

e. Saat ini kapasitas penyimpanan Harddisk masih lebih besar dari kapasitas penyimpanan Flashdisk

Kapasitas penyimpanan Harddisk dapat mencapai 4TB, ( 4000 Gb ). Sedangkan penyimpanan Flashdisk mencapai 128GB ( benar )

27. Jelaskan cara penyimpanan dokumen pada perangkat keras penyimpanan data. contoh flashdisk


Penyimpanan dokumen berupa data bisa dilakukan pada perangkat keras (storage device) seperti flashdisk. Cara penyimpanan dokumen pada perangkat ini terbilang mudah yakni dengan memilih data yang hendak disimpan, kemudian melakukan klik kanan pada data tersebut dan bisa memilih opsi COPY ataukah SEND TO.

Apabila opsi yang dipilih adalah COPY maka langkah KLIK KANAN DATA > KLIK COPY > KLIK KANAN FOLDER FD > PASTE.

Jika opsi yang dipilih adalah SEND TO maka langkahnya adalah KLIK KANAN DATA > ARAHKAN KE SEND TO > KLIK REMOVABEL DISK (folder flashdisk).
