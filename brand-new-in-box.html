7. Jane packed 216 pens into 3 boxes The ratio number of pens in box A to the number of pens in box B to the number of pens in box C is 1:3:4 how many more pens were there in box B than in box A​


Jawab: 54 pens

Penjelasan dengan langkah-langkah:

A+B+C = 1+3+4 = 8

difference between B and A = 3 -1 = 2

[tex]= \frac{2}{8} \times 216\\\\= 54[/tex]

So box B have 54 more pens than box A

25. I … a brand new carA. Had B. Having C. Have D. Has ​


Jawaban:

B. have

Penjelasan:

Hi all

introduce my name is syasya, i am now in 7th grade. I will explain and give the difference from the words have, has, has and had

# Have digunakan bila subjeknya berupa I, you, they, dan we.

# Has digunakan jika subjeknya merupakan he, she, it.

# Had merupakan bentuk lampau (verb 2) dari have. Had bisa

  digunakan untuk semua subjek seperti I, you, they, we, he, she, dan

   it.

# Having merupakan bentuk kata kerja (verb) yang ditambah imbuhan

 -ing dari have. Having bisa digunakan untuk semua subjek seperti I,

  you, they, we, he, she, dan it.

I hope it's useful for you all
