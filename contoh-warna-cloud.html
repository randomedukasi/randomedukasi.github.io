3. Sebutkan contoh website cloud​


1. Google Drive

Google Drive adalah salah satu contoh cloud computing yang sering digunakan juga. Penggunanya tidak perlu install aplikasi drive. Hanya dengan memiliki akun Google Mail maka penggunanya langsung bisa akses Google Drive dan aplikasi lain milik Google seperti Google Meet, Google Classroom, dan lain-lain

2. Media Sosial

Media sosial juga merupakan contoh cloud computing. Kemampuannya tak hanya via aplikasi yang harus di install namun juga optimal di versi web. Media sosial yang sering diakses melalui versi web antara lain Youtube, Linkedin, Facebook, dan lain-lain.

3. Dropbox

Dropbox menjadi salah satu contoh cloud computing yang sering digunakan terutama oleh mahasiswa. Dropbox memungkinkan untuk menyimpan dan membagikan file secara mudah.

4. Bidang Kesehatan

Alat-alat kedokteran juga sudah sudah menerapkan sistem cloud diantaranya Telemedicine yaitu alat komunikasi jarak jauh antara tenaga kesehatan dengan pasien dan juga sebagai media perantara transfer data medis dari satu lokasi ke lokasi lainnya. Tidak hanya Telemedicine, cloud computing juga diterapkan di Electronic Health Records, dan analisis Teleradiology.

5. Bidang Pendidikan

Cloud computing yang diterapkan di bidang pendidikan antara lain Google Talk, Google Video for Education, dan Google Classroom. Semua aplikasi ini dapat lebih memudahkan guru dan siswa dalam proses belajar mengajar.

6. Bidang Pemerintahan

Bidang pemerintah juga sudah menerapkan cloud computing. Contoh cloud computing di bidang pemerintahan adalah layanan jasa alih daya pengelolaan TIK oleh Badan Pengkajian dan Penerapan Teknologi (BPPT) dan penerapan cloud computing dalam sistem kerja Badan Pengawas Tenaga Nuklir dan Kementerian Riset dan Tekn

21. 4.Apakah yang Anda ketahui dengan aplikasi berbasis cloud (cloud based application) ? 5. Apakah keuntungan dari penggunaan apkikasi berbasis cloud ?


Aplikasi berbasis cloud (cloud based application) adalah aplikasi yang disediakan oleh perusahaan penyedia layanan cloud. Maksud dari layanan cloud adalah aplikasi yang kita gunakan berada di internet dan tidak terinstall di komputer kita. Untuk mengakses aplikasi tersebut kita memerlukan akses internet.

Keuntungan dari penggunaan aplikasi berbasis cloud adalah keamanan data dan kita tidak dipusingkan untuk melakukan perawatan terhadap aplikasi tersebut karena sudah menjadi tanggung jawab penyedia layanan cloud tersebut.

Pembahasan

Setelah mengetahui apa itu aplikasi berbasis cloud dan keuntungan dari menggunakan aplikasi cloud, mari kita sebutkan jenis-jenis dari aplikasi berbasi cloud, beberapa jenis aplikasi cloud adalah sebagai berikut

Infrastructure as a Service contohnya adala Amazon EC2, Telkom Cloud.Platform as a Service contoh aplikasinya adalah Windows AzureSoftware as a Service aplikasi Office 365 merupakan contoh aplikasi cloud jenis Software as a Service

Penggunaan aplikasi cloud tentu sangat membantu seorang pengusaha, karena bisa lebih fokus untuk pengembangan usahanya dan tidak dipusingkan dengan mengurus komputer dan aplikasi-aplikasi pendukungnya. Selain contoh-contoh diatas, masih ada contoh aplikasi cloud yang sering kita gunakan yaitu email dan media sosial.

Pelajari lebih lanjut

Demikian pembahasan mengenai aplikasi berbasis cloud dan keuntungan dari penggunaan aplikasi berbasi cloud. Untuk menggunakan aplikasi berbasis cloud kita harus terhubung dengan internet terlebih dahulu. 26. 11. Komponen dari dua data atau lebih infrastruktur cloud disebut... a. Hybrid cloud b. Community cloud c. On-site-cloud d. Private cloud e. Public cloud​


Jawaban:

A. Hybrid cloud

Penjelasan:

Hybrid cloud adalah jenis infrastruktur cloud yang terdiri dari dua atau lebih komponen dari jenis infrastruktur cloud yang berbeda, seperti public cloud, private cloud, atau on-premises cloud. Dalam hybrid cloud, setiap jenis infrastruktur cloud tetap terpisah, tetapi dihubungkan melalui jaringan yang aman dan dapat berkomunikasi dengan lancar satu sama lain. Dengan menggunakan hybrid cloud, organisasi dapat memanfaatkan keuntungan dari setiap jenis infrastruktur cloud yang tersedia, misalnya keamanan dan kontrol yang lebih baik dari private cloud dan skalabilitas dan elastisitas dari public cloud. Hybrid cloud juga memberikan fleksibilitas bagi organisasi untuk memilih infrastruktur cloud yang paling cocok dengan kebutuhan bisnis mereka serta memungkinkan organisasi untuk memindahkan data dan aplikasi dari satu infrastruktur cloud ke infrastruktur cloud lainnya jika diperlukan.


27. private cloud public cloud hybrid cloud merupakan​


Jawaban:

Public cloud adalah cloud computing yang dikirimkan melalui internet dan dibagikan ke seluruh organisasi atau perusahaan. Private cloud adalah cloud computing yang didedikasikan hanya untuk organisasi atau perusahaan Anda. Hybrid cloud merupakan gabungan antara public cloud maupun private cloud.
