1. lidah api dapat menjulang sejauh ribuan sampai jutaan KM sebagai dampak dari gangguan yang terjadi pada kromosfer semburan Gas mempunyai lidah api besar berwarna merah berbentuk menyerupai Avita simpul spiral dan gunung B simpul spiral gunung dan cincin C pita gunung cincin dan gelang D bermata mahkota simpul dan pita e mahkota simpul cincin dan pita ​


Lidah api matahari bisa menjulang sangat jauh hingga ribuan sampai jutaan kilometer akibat dari gangguan yang muncul pada lapisan kromosfer. Semburan gas lidah api ini sangat besar, berwarna merah dengan bentuk yang serupa dengan pita, simpul, spiral dan gunung. Jawaban yang benar adalah A.

Pembahasan

Nama lain dari lidah api adalah protuberans atau prominensa. Lidah api ini adalah gas yang sangat panas yang disemburkan dengan kekuatan yang sangat besar dari arah permukaan matahari tepatnya pada lapisan kromosfer. Jika diperhatikan, semburan gas lidah api dari matahari ini warnanya selalu kemerahan namun bentuknya variatif atau beraneka ragam. Bentuk-bentuk yang paling umum dijumpai muncul antara lain mirip seperti pita, serupa dengan loop atau simpul, mirip gunung, mirip dengan spiral dan mirip pula dengan tabir.

Lidah api dari matahari ini mampu menyembur jauh dan memanjang keluar dari permukaan lapisan matahari. Jauhnya bisa mencapai ribuan hingga jutaan mil. Lidah api matahari ini, dari bumi bisa kita amati dengan waktu-waktu tertentu semisal saat gerhana matahari tital sedang berlangsung.
