6. persamaan garis lurus k yang tegak lurus dengan garis m​


Gradien garis m = y/x

= 5/3 *garis miring ke kanan positif

tegak lurus garis k, jadi gradien k

= gradien m × gradien k = -1

= 5/3 × gradien k = -1

= gradien k = -1 : 5/3

= gradien k = -1 × 3/5

Gradien k = -3/5

Persamaan garis k melewati titik (0,-4)

= y - y1 = m (x - x1)

= y -(-4) = -3/5 (x - 0)

= 5(y + 4) = -3 (x - 0)

= 5y + 20 = -3x + 0

= 5y + 3x + 20 = 0

atau 3x + 5y + 20 = 0 (D)

untuk menentukan gradien pada persamaan garis lurus :

ax+by+c=0

by = -ax – c

y = -a/bx – c/b

m(gradient) = -a/b

jadi :

2x-4y=-10

-4y=-2x-10

y=-2/-4x - 10/-4

m= -2/-4
