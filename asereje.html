16. panjang jari jari roda sepeda 49cm seretelah berputar 100 kali panjang lutasnya adalah


Pembahasan.

1.Rumus hubungan panjang lintasan , keliling dan banyaknya putaran dirumuskan sebagai berikut :

J = K × n

Keterangan

J = Jarak
K = Keliling lingkaran
n = banyaknya putaran.

Mari kita terapkan.

Diket

r = 49
n = 100

Dita
J = ?

J = K × n
J = ( 2 π r ) × 100
J = ( 2 × ²²/7 × 49 ) × 100
J = 308 × 100
J = 30800 cm / 308 m
