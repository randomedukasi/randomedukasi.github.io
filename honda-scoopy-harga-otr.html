12. Jumlah usia honda dan bambang adalah 93. Jika usia honda 3 tahun yang lalu sama dengan usia bambang 8 tahun dari sekarang, berapa usia honda sekarang? sepi...


Usia Honda sekarang adalah 52 tahun

PENJELASAN

Misal:

Usia Honda = xUsia Bambang = y

Diketahui:

Jumlah usia x dan y adalah 93Usia x - 3 = y + 8

Ditanyakan:

Usia x

Jawab:

Jumlah x dan y adalah 93

=> x + y = 93  (I)

x - 3 = y + 8 (II)

=> x = y + 8 + 3

=> x = y + 11 (III)

Substitusikan III ke I

x + y = 93

y + 11 + y = 93

2y + 11 = 93

2y = 93 - 11

2y = 82

y = 82/2

y = 41

Nilai y (usia Bambang) adalah 41

Substitusikan nilai y ke III

x = y + 11

x = 41 + 11

x = 52

Nilai x (usia Honda) adalah 52

PEMBUKTIAN

x + y = 93

52 + 41 = 93

93 = 93  (Benar)

x - 3 = y + 8

52 - 3 = 41 + 8

49 = 49  (Benar)

KESIMPULAN

Usia Honda sekarang adalah 52 tahun

Mata Pelajaran : Matematika

Kelas : VIII

Materi : Sistem Persamaan Linier Dua Variabel

Kode Soal : 2

Kode Kategorisasi : 8.2.5

Penjelasan dengan langkah-langkah:

misal

mobil toyota fortuner = x

motor honda cbr = y

kita sederhanakan dulu 9.960.000.000 dan 8.450.000.000 agar mudah menghitungnya

dari soal dibuat persamaan

12x + 8y = 9960

=> 3x + 2y = 2490............(1)

10x + 10y = 8450

=> x + y = 845

=> x = 845 - y....................(2)

substitusikan x ke pers 1

3(845-y) + 2y = 2490

2535 - 3y + 2y = 2490

y = 2535-2490

y = 45

substitusikan y ke pers 2

x = 845 - 45

x = 800

jadi harga 1 mobil Rp 800.000.000

dan motor Rp 45.000.000

5. 30 mobil toyota fortuner dan 30 motor honda cbr dengan potongan harga 20%?
