2. Cara mengganti font style dan font size pada office word


Jawaban:

Tempat untuk font size dan font style (mengganti font) ada di kiri atas layar. Ada kolom yang bertuliskan calibri (default, kalau pernah ganti bisa berbeda seperti arial, times new roman, dsb.) itu untuk font style. Untuk font size ada di sebelah kanan kolom dropdown font style, yang bertuliskan angka.

Klik dropdownya, kemudian pilih sesuai yang diinginkan, kemudian kata yang diselect akan berubah.

6. jelaskan cara membuka kotak dialog font dari menu pada microsoft word!​


Jawaban:

Untuk membuka kotak dialog font dapat dengan cara.

1) Klik icon atau tombol kecil yang ada pada grup Font. Grup Font berada pada Tab Menu Home.

2) Cara kedua dengan menggunakan tombol kombinasi pada keyboard. Ada beberapa kombinasi tombol yang dapat digunakan untuk membuka kotak dialog Font.

a. CTRL + D

b. CTRL + SHIFT + F

c. CTRL + SHIFT + P

14. bagaimana cara memilih jenis font pada microsoft word


Cara memilih jenis font pada Microsoft Word adalah dengan mencari menu Font pada ribbon Home atau dengan menggunakan shortcut Ctrl + Shift + F maka akan muncul pilihan untuk melakukan perubahan font yang tersedia.

Pembahasan

Font adalah salah satu unsur penting dalam pembuatan dokumen menggunakan Microsoft Word di semua tahun keluarannya. Oleh karena itu, menu ini disediakan karena beberapa perusahaan / instansi mempunyai ketentuan sendiri perihal font resmi yang digunakan pada dokumen mereka.

Cara memilih jenis font dan melakukan edit pada text :

1. Pilih text yang akan diubah dengan cara diblok menggunakan mouse atau keyboard ( shift + arrow left/right)

2. Buka ribbon Home, menu Font. Akan tersedia :

Jenis Font, Contoh : Arial, Times New Roman, Verdana, dan lain - lainStyle Font, berupa regular, italic, bold, dan underlineFont size, merupakan ukuran font yang akan digunakan dalam range 8 hingga 72Font colour, merupakan pilihan warna yang diberikan pada fontEffect, berupa tambahan kelengkapan unsur text seperti perpangkatan, pencoretan text, pengubahan menjadi huruf kapital semua dan sebaliknya.Highlighter, merupakan fitur untuk menghighlight bagian yang diperlukan. Mirip dengan fungsi stabilo untuk kertas.

3. Setelah menentukan pengubahan gaya text yang diinginkan, tampilan text akan berubah

24. bagaimana cara mengubah font pada Microsoft word 2010​


Cara Mengganti Jenis Huruf/Font Default Di Microsoft Word

1. Buka menu ribbon home.

2. Pada grup styles, cari style Normal lalu klik kanan pada style tersebut.

3. Pada popup menu yang muncul, klik item modify.

4. Selanjutnya pada dialog Modify Style ganti jenis huruf (font) serta ukurannya.

5. Kemudian pada bagian bawah dialog, pilih opsi New documents based on this template.

26. Cara mengubah font dan zize huruf pada microsot word.


Jawaban:

Misalkan pada Ms. Word 2013, caranya:

- sorot/ blok teks yg ingin diubah font dan size hurufnya

- klik Font yg ada di ribbon Home dan pilih salah satu

- klik Size yg ada di sebelah Font sesuai ukuran yg diinginkan

Penjelasan tambahan:

Amatilah lampiran:

- menyorot/ memblok teks ditunjukkan dg tanda panah hitam

- mengklik Font ditunjukkan dg tanda panah merah

- mengklik Size ditunjukkan dg tanda panah biru

Perlu dimengerti, bahwa Size diisikan angka:

- Semakin tinggi angkanya, ukuran huruf semakin besar

- Semakin rendah angkanya, ukuran huruf semakin kecil

(pada contoh gambar, ukuran Size berisi angka 11)


27. jelaskan fitur pengaturan font pada microsoft word ?​


Jawaban:

Buka File > Opsi > Email > Alat tulis dan Font...

Memilih font yang ingin Anda ubah

Pesan email baru - Atur font default untuk pesan asli yang Anda buat.

Membalas atau meneruskan pesan - Anda dapat memiliki font berbeda pada pesan yang diteruskan atau dibalas. Pengaturan ini umumnya untuk dua kotak centang yang memungkinkan Anda menandai komentar dengan nama Anda (atau teks lain) atau dengan warna yang berbeda dari teks.

Membuat dan membaca pesan teks biasa - Pesan teks biasa akan diterima oleh orang lain dengan font standar, tetapi jika ingin menggunakan font yang lebih bagus ketika sedang menulis pesan, Anda dapat menggunakannya. Cukup pahami bahwa font tidak akan dikirim dengan pesan.

Pilih opsi yang ingin Anda terapkan ke font default, seperti gaya font dan ukuran font.

Klik OK sepanjang proses keluar untuk menyimpan perubahan Anda.
