D. Luas persegi panjang (bawah/ dasar) = panjang sisi miring trapesium x panjang rusuk persegi panjang (kiri/ kanan)
Panjang sisi miring trapesium = √(20^2 + 1^2) = √401 = 20,0249843945 = 20,025
Jd luas permukaan dasar kolam =20,025*10=200,25 m2

Sehingga luas permukaan kolam renang yg akan dicat adalah
=60+20+10+200,25
=290,25 m2

Jika diketahui harga 1 kaleng cat lebih murah Rp35.000 dari harga 1 karung semen dan total harga tidak lebih dari Rp455.000, tentukanlah: a) Model matematika dari ilustrasi tersebut b) Harga 1 karung semen dan 1 kaleng cat yang sebenarnya​


Penjelasan dengan langkah-langkah:

misal:

semen = x

cat = y

harga cat = harga semen - Rp 35.000

total harga = Rp 455.000

a) model matematika

y = x - 35.000

5x + 2y = 455.000

b) harga semen dan cat yang sebenarnya

5x + 2y = 455.000

5x + 2(x - 35.000) = 455.000

5x + 2x - 70.000 = 455.000

7x = 455.000 + 70.000

7x = 525.000

x = 75.000

y = x - 35.000

y = 75.000 - 35.000

y = 40.000

Jadi, harga 1 karung semen adalah Rp 75.000 dan harga 1 kaleng cat adalah Rp 40.000

26. Pak syarief membeli empat kaleng cat untuk bagian luar dan dua kaleng cat untuk bagian dalam dengan harga rp.880.000,00. sepekan kemudian ia membeli di toko yang sama masing – masing tiga kaleng cat untuk bagian luar dan satu kaleng cat untuk bagian dalam dengan harga rp.620.000,00. jumlah harga satu kaleng cat bagian luar dan satu kaleng cat bagian dalam adalah …


Harga satu kaleng cat bagian luar dan harga satu kaleng cat bagian dalam adalah Rp180.000,00 dan Rp80.000,00. Hasil ini diperoleh dengan menggunakan metode eliminasi dan substitusi.

Penjelasan dengan langkah-langkah

Diketahui

Harga 4 kaleng cat untuk bagian luar dan 2 kaleng untuk bagian dalam = Rp880.000,00.Harga 3 kaleng cat untuk bagian luar dan 1 kaleng cat untuk bagian dalam = Rp620.000,00.

Ditanya

Berapa harga 1 kaleng cat bagian luar?Berapa harga 1 kaleng car bagian dalam?

Jawab

Misal

L = harga 1 kaleng cat untuk bagian luarD = harga 1 kaleng cat untuk bagian dalam

Kita dapatkan persamaan

4L + 2D = 880.000 ... (pers. 1)

3L + D = 620.000 ... (pers. 2)

Kalikan persamaan 2 dengan bilangan 2 sehingga didapatkan 2D sama dengan persamaan 1.

3L + D = 620.000 (2×) → 6L + 2D = 1.240.000 ... (pers. 3)

Eliminasi persamaan 1 dan 3!

4L + 2D = 880.000

6L + 2D = 1.240.000 _

2L = 360.000

L = Rp180.000

Substitusikan nilai L ke dalam persamaan 1!

4(180.000) + 2D = 880.000

720.000 + 2D = 880.000

2D = 880.000 - 720.000

2D = 160.000

D = Rp80.000

Kesimpulan

Jadi, harga satu kaleng cat bagian luar dan harga satu kaleng cat bagian dalam adalah Rp180.000,00 dan Rp80.000,00.
