9. Diketahui NetID 192.168.15.0/28 Carilah subnermask, jumlah subnet, blok subnet, jumlah host, IP subnet, host pertama, hosterakhir, brodcast setiap subnet.


NetID 192.168.15.0/28 merupakan notasi dari sebuah jaringan IP yang terdiri dari alamat IP dan subnet mask. Berikut ini adalah informasi yang dapat Anda cari mengenai NetID tersebut:

Subnet mask: Subnet mask merupakan notasi yang menunjukkan bagian mana dari alamat IP yang digunakan untuk menentukan subnet. Dalam NetID 192.168.15.0/28, subnet mask adalah /28, yang artinya hanya 4 bit pertama dari alamat IP digunakan untuk menentukan subnet. Subnet mask dalam notasi decimal adalah 255.255.255.240.

Jumlah subnet: Jumlah subnet yang dapat dibuat dari NetID 192.168.15.0/28 adalah 16, karena hanya terdapat 4 bit yang digunakan untuk menentukan subnet.

Blok subnet: Blok subnet adalah jumlah alokasi IP yang diberikan kepada setiap subnet. Dalam NetID 192.168.15.0/28, setiap subnet memiliki blok subnet sebesar 16 IP.

Jumlah host: Jumlah host adalah jumlah IP yang tersedia untuk digunakan oleh perangkat di dalam sebuah subnet. Dalam NetID 192.168.15.0/28, setiap subnet memiliki 14 IP yang dapat digunakan sebagai host.

IP subnet: IP subnet adalah alamat IP yang digunakan untuk menentukan subnet. Dalam NetID 192.168.15.0/28, IP subnet untuk setiap subnet adalah 192.168.15.0, 192.168.15.16, 192.168.15.32, dan seterusnya, dengan interval 16 IP.

Host pertama: Host pertama adalah alamat IP yang pertama kali tersedia untuk digunakan sebagai host di dalam sebuah subnet. Dalam NetID 192.168.15.0/28, host pertama untuk setiap subnet adalah 192.168.15.1, 192.168.15.17, 192.168.15.33, dan seterusnya.

Host terakhir: Host terakhir adalah alamat IP yang terakhir kali tersedia untuk digunakan sebagai host di dalam sebuah subnet. Dalam NetID 192.168.15.0/28, host terakhir untuk setiap subnet adalah 192.168.15.14, 192.168.15.30, 192.168.15.46, dan seterusnya.

broadcast untuk setiap subnet dalam NetID 192.168.15.0/28 adalah alamat IP terakhir yang tersedia di dalam subnet tersebut.

21. Jumlah subnet,jumlah host persubnet dan blok subnet dari 191.198.1.1/26


Jawaban:Jika menggunakan subnet mask /26 pada alamat IP 191.198.1.1, maka jumlah subnet yang dihasilkan adalah 2 (dua). Jumlah host per subnet adalah 62, sedangkan blok subnet adalah 64.

Penjelasan:

Untuk menghitung jumlah subnet, host per subnet, dan blok subnet, pertama-tama Anda perlu mengetahui jumlah bit yang digunakan untuk menyatakan subnet mask. Dalam kasus /26, terdapat 26 bit yang digunakan untuk menyatakan subnet mask, sehingga sisa bit yang tersisa untuk menyatakan host adalah 32 - 26 = 6 bit.

Dengan menggunakan 6 bit untuk menyatakan host, maka jumlah host per subnet adalah 2^6 = 64. Karena setiap subnet harus menyisakan satu alamat untuk menyatakan network address dan satu alamat lainnya untuk menyatakan broadcast address, maka jumlah host yang sebenarnya tersedia per subnet adalah 64 - 2 = 62.

Blok subnet adalah jumlah alamat yang terdiri dari network address, host address, dan broadcast address. Dengan menggunakan 6 bit untuk menyatakan host, maka blok subnet adalah 2^6 = 64.

Dengan demikian, jumlah subnet yang dihasilkan dari alamat IP 191.198.1.1 dengan subnet mask /26 adalah 2, jumlah host per subnet adalah 62, dan blok subnet adalah 64.

23. Pada IP 196.189.200.1/26 tentukanjumlah subnetjumlah host persubnetblok subnet​


Jawaban:

IP 196.189.200.1/26 merupakan alamat IP dengan subnet mask 255.255.255.192 atau dalam bentuk CIDR /26. Subnet mask tersebut menunjukkan bahwa terdapat 26 bit yang direservasi untuk network ID dan sisa bit (6 bit) digunakan untuk host ID.

Jumlah Subnet

Jumlah subnet dapat dihitung dengan menggunakan formula 2^n dimana n adalah jumlah bit yang direservasi untuk subnet. Pada kasus ini, terdapat 6 bit yang direservasi untuk subnet, sehingga jumlah subnet yang dapat dibentuk adalah 2^6 = 64 subnet.

Jumlah Host Persubnet

Jumlah host persubnet dapat dihitung dengan menggunakan formula 2^n-2 dimana n adalah jumlah bit yang direservasi untuk host. Karena pada kasus ini terdapat 6 bit yang direservasi untuk host ID, maka jumlah host persubnet yang dapat dihitung adalah 2^6-2 = 62 host.

Blok Subnet

Untuk menentukan blok subnet, kita perlu menghitung nilai dari setiap oktet pada alamat IP dan subnet mask. Berikut adalah nilai dari setiap oktet pada alamat IP dan subnet mask:

Alamat IP: 196.189.200.1 = 11000100.10111101.11001000.00000001

Subnet mask: 255.255.255.192 = 11111111.11111111.11111111.11000000

Untuk menentukan blok subnet, kita cukup melakukan operasi bitwise AND antara alamat IP dan subnet mask. Berikut adalah hasil operasi bitwise AND tersebut:

11000100.10111101.11001000.00000001 (IP Address)

11111111.11111111.11111111.11000000 (Subnet Mask)

Jumlah subnet = 256 subnetHost per subnet = 65.534 host per subnetBlok subnet = 1Tabel subnet terdapat pada bagian pembahasan.

 

Pembahasan

Diketahui IP: 10.10.0.0/16

Oktet pertama = 10, maka termasuk pada class A.CIDR subnet mask = /16Subnet mask (biner)
= 11111111.11111111.00000000.00000000Subnet mask = 255.255.0.0x: banyak bit 1 pada oktet ke-2 sampai oktet ke-4 dari subnet mask
⇒ x = 8y: banyak bit 0 pada oktet ke-2 sampai oktet ke-4 dari subnet mask
⇒ y = 16

JUMLAH SUBNET (S)
S = [tex]2^x = 2^8[/tex] = 256 subnet.

JUMLAH HOST PER SUBNET (H)
H = [tex]2^y - 2 = 2^{16} - 2[/tex]
⇒ H = 65.536 – 2
⇒ H = 65.534 host per subnet

BLOK SUBNET (B)
B = 256 – nilai oktet kedua
⇒ B = 256 – 255
⇒ B = 1

Jadi, pada oktet kedua network address dari subnet, angkanya adalah 0, 1, 2, 3, ..., 255.Untuk setiap blok subnet, range dari oktet ketiga adalah dari 0 sampai 255.Untuk setiap blok subnet, semua kemungkinan nilai oktet ketiga dan keempat dari IP host dapat digunakan, kecuali yang bernilai *.*.0.0 (untuk network address), dan *.*.255.255 (untuk broadcast address).

30. jumlah subnet dan host per subnet dari ip 192.168.1.1/19 berapa?


Jumlah subnet dan host per subnet dari IP 192.168.1.1/19 adalah sebagai berikut:

- Notasi CIDR /19 menandakan bahwa 19 bit pertama pada alamat IP tersebut digunakan untuk network ID, sehingga sisa 13 bit digunakan untuk host ID.
- Jumlah subnet dapat dihitung dengan rumus 2^k, di mana k adalah jumlah bit yang digunakan untuk network ID. Dalam hal ini, k = 19, sehingga jumlah subnet adalah 2^19 = 524,288 subnet.
- Jumlah host per subnet dapat dihitung dengan rumus 2^h - 2, di mana h adalah jumlah bit yang digunakan untuk host ID. Dalam hal ini, h = 13, sehingga jumlah host per subnet adalah 2^13 - 2 = 8,190 host per subnet.

Jadi, IP 192.168.1.1/19 memiliki 524,288 subnet dan 8,190 host per subnet.
