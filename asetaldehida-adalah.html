Pembahasan:

Reaksi Adisi HCN (nukleofilik)

Penambahan Hidrogen Sianida [tex](HCN)[/tex] pada aldehida dan keton menghasilkan senyawa Sianohidrin. Sianohidrin merupakan senyawa yang terdapat sebuah gugus siano [tex](-CN)[/tex] dan gugus hidroksil [tex](-OH)[/tex] yang terikat dalam atom karbon yang sama.

⟡ ⟡ ⟡

Berikut persamaan reaksi antara Asetaldehida dengan HCN.

[tex]{CH}_3CH=O\ +\ HCN\ \xrightarrow{OH^-}\ {CH}_3CH(OH)CN\\asetaldehida\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ asetaldehida\ sianohidrin[/tex]

[tex]\bold{Mekanisme:}[/tex]

Ikatan [tex]C=O[/tex]  bersifat polar karena perbedaan keelektronegatifan antara atom karbon dan oksigen. Oksigen dalam [tex]C=O[/tex] mempunyai muatan parsial negatif [tex](\delta^-)[/tex] sementara karbon memperoleh muatan parsial positif [tex](\delta^+)[/tex]. Muatan parsial positif pada karbon menarik [tex]CN^-[/tex] dari [tex]HCN[/tex]. Oleh karena itu, ikatan rangkap duanya dengan oksigen terputus dan terbentuklah ikatan [tex]C-CN[/tex] yang baru. Alhasil, oksigen menjadi tidak stabil karena kelebihan satu elektron, sehingga menarik [tex]H^+[/tex] dari [tex]{H}^+{CN}^-[/tex] agar stabil.

⟡ ⟡ ⟡

Berikut persamaan reaksi antara Aseton dengan HCN.

[tex]CH_3COCH_3\ +\ HCN\ \xrightarrow{OH^-}\ {(CH_3)}_2C(OH)CN\\2-propanon\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ aseton\ sianohidrin[/tex]

Reaksinya sama untuk keton, namun atom karbon yang berikatan dengan gugus hidroksinitril juga berikatan dengan dua gugus alkil.

Reaksi Adisi Bisulfit (nukleofilik)

Aldehida yang bereaksi dengan Natrium Bisulfit (Natrium Hidrogen Sulfit) akan menghasilkan adduk berupa senyawa adisi Natrium Bisulfit. Untuk keton, kecuali metil keton, tidak dapat bereaksi dengan Natrium Bisulfit karena hambatan sterik (crowded).

Berikut persamaan reaksi antara Asetaldehida dengan NaHSO₃.

[tex]{CH}_3CH=O\ +\ {NaHSO}_3\ \xrightarrow{}\ {CH}_3CH(OH){SO}_3-Na^+\\asetaldehida\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ asetaldehida\ natrium\ bisulfit[/tex]

Reaksi Adisi Hidroksilamin (nukleofilik)

Aldehida dan keton bereaksi dengan hidroksilamin [tex]({NH}_2OH)[/tex] untuk membentuk oksim.

[tex]{CH}_3CH=O\ +\ {NH}_2OH\ \xrightarrow{}\ {CH}_3CH=NOH\ +\ {H}_2O\\asetaldehida\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ asetaldoksim[/tex]

5. Bagaimana Membedakan Antara Asetaldehida dengan Benzaldehida?​


Asetaldehida merupakan senyawa organik yang masih dalam kelompok aldehida, dengan rumus kimia CH3CHO atau bisa juga ditulis dengan rumus kimia MeCHO. Dalam hal ini, Asetaldehida merupakan cairan yang mudah terbakar dengan bau buah-buahan. Asetaldehida terdapat dalam buah-buahan, kopi, dan juga roti yang masih segar. Senyawa Asetaldehida dihasilkan tumbuhan dalam metabolisme normalnya. Sementara itu,

Benzaldehida merupakan senyawa yang tersusun atas cincin benzena dengan sebuah substituenaldehida. Benzaldehida merupakan senyawa aldehida aromatik pada suhu kamar. Benzaldehida merupakan cairan transparan dengan bau seperti kacang almond, dan dapat juga diekstrasi dari berbagai bahan alami seperti buah ceri dan buah persik.

Jawaban beserta penjelasan:

Cara membedakan antara Asetaldehida dan Benzaldehida melalui reaksi kimia antara lain sebagai berikut.

1.) Tes Haloform

Merupakan reaksi penambahan basa Natrium Hidroksida dan halogen X₂, dimana X dapat berupa unsur Cl, I atau Br, namun lebih umum menggunakan reagen yodium (I₂) dalam hal ini sama halnya dengan tes Iodoform. Asetaldehida memberikan hasil positif dengan ditandai terbentuknya endapan kuning pucat. Sedangkan benzaldehida tidak terbentuk.

2.) Reaksi dengan reagen Fehling

Asetaldehida yang bereaksi dengan larutan Fehling akan memberikan hasil positif berupa endapan berwarna merah yang merupakan senyawa Tembaga (I) oksida (Cu₂O), sementara untuk Benzaldehida tidak ada endapan merah yang akan terbentuk.

7. Bagaimana Cara Sintesis Asetaldehida dari Formaldehida? Jelaskan!​


Jawaban beserta penjelasan:

Ada beberapa cara untuk mensintesis Asetaldehida dari Formaldehida, antara lain:

[tex]\boxed{\bold{Sintesis\ cara\ (1)}}[/tex]

[tex]\ \ \ HCHO\ \ \xrightarrow[\text{hidrolisis}]{\text{CH3MgBr}}\ CH_3CH_2OH\ \xrightarrow{\text{Cu/573K}}\ CH_3CHO\\_{formaldehida}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{etanol} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{asetaldehida}[/tex]

Menggunakan reagen Grignard CH₃MgBr dan dihidrolisis menghasilkan etanol (CH₃CH₂OH). Selanjutnya, menggunakan katalis logam Tembaga (Cu) serta dipanaskan sampai suhu menjadi 573 K atau 300⁰C dan pada akhirnya terbentuklah Asetaldehida.

◇ ◇ ◇

[tex]\boxed{\bold{Sintesis\ cara\ (2)}}[/tex]

[tex]\ \ \ HCHO\ \ \xrightarrow[\text{Eter}]{\text{CH3MgI}}\ CH_3CH_2OMgI\ \xrightarrow{\text{H3O+}}\ CH_3CH_2OH \xrightarrow[\text{300$^{\circ}$C}]{\text{Cu}}\ CH_3CHO\\_{formaldehida}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{etilalkohol} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{asetaldehida}[/tex]

Menggunakan reagen Grignard CH₃MgI sehingga menghasilkan Magnesium iodia Etoksida (CH₃CH₂MgI).  Kemudian, diprotonasi ion hidronium menjadi Etil alkohol (C₂H₅OH). Setelah itu, mengkatalis dengan logam Tembaga (Cu) lalu dipanaskan hingga suhu 300⁰C dan akhirnya terbentuklah Asetaldehida.

◇ ◇ ◇

[tex]\boxed{\bold{Sintesis\ cara\ (3)}}[/tex]

[tex]\ \ \ HCHO\ \ \xrightarrow[\text{H2SO4}]{\text{K2Cr2O7}}\ HCOOH\ \xrightarrow{\text{Ca(OH)2}}(HCOO)_2Ca \xrightarrow[\text{panas}]{\text{(CH3COO)2Ca}}\ CH_3CHO\\_{formaldehida}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{asam\ format} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{kalsium\ format} \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ _{asetaldehida}[/tex]

Menggunakan oksidator Kalium dikromat (K₂Cr₂O₇) dengan bantuan asam sulfat (H₂SO₄) sehingga menghasilkan asam format (HCOOH). Lalu, dititrasi oleh basa Kalsium hidroksida (Ca(OH)₂) untuk menghasilkan larutan garam Kalsium format ((HCOO)₂Ca). Setelah itu, dicampur dengan Kalsium asetat kemudian dipanaskan dan pada akhirnya terbentuklah Asetaldehida.

10. Bagaimana Cara Mensintesis Aseton dari Asetaldehida?​


Jawaban beserta penjelasan:

Adapun cara untuk mensintesis aseton dari asetaldehida, yaitu terdiri dari dua langkah yang dijabarkan seperti berikut.

[tex]\boxed{\bold{Langkah\ 1:}}[/tex] Mengubah asetaldehida menjadi isopropil alkohol.

[tex]CH_3CHO\ +\ CH_3MgBr\xrightarrow{Eter}\ C_3H_7OMgBr\ \xrightarrow{H2O}\ C_3H_7OH\ +\ Mg(OH)Br\\asetildehida\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ isopropil\ alkohol[/tex]

Asetaldehida diberi reagen Grignard Metil Magnesium Bromida (CH₃MgBr) selagi dilarutkan dalam larutan Dietil Eter (C₄H₁₀O), lalu menghasilkan senyawa kompleks Mg berupa Propanol Magnesium Bromida atau C₃H₇OMgBr. Senyawa tersebut dihidrolisis hingga membentuk alkohol sekunder, yakni Isopropil alkohol (C₃H₇OH).

◈ ◈ ◈

[tex]\boxed{\bold{Langkah\ 2:}}[/tex] Reaksi oksidasi dari isopropil alkohol.

[tex]\ \ C_3H_7OH\ \ \xrightarrow[H+]{K2Cr2O7}\ \ (CH3)_2CO\\asetaldehida\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ aseton[/tex]

Isopropil alkohol kemudian dioksidasi dengan Kalium Dikromat (K₂Cr₂O₇) yang telah diasamkan untuk membentuk senyawa keton, yakni Aseton (C₃H₆O).

13. Bagaimana Cara Mensintesis Asam Laktat dari Asetaldehida?​


Jawaban beserta penjelasan:

Adapun cara untuk mensintesis asam laktat dari asetaldehida, yakni:

[tex]\boxed{\bold{Sintesis\ Asam\ Laktat}}[/tex]

[tex]\ \ CH_3CHO\ \xrightarrow{HCN}\ CH_3CH_2CNO\ \xrightarrow{H3O+}\ CH_3CH(OH)CO_2H\\asetaldehida\ \ \ \ \ \ \ \ \ \ \ \ \ laktonitril\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ asam\ laktat[/tex]

Asetaldehida (aldehida) mengalami reaksi adisi (penambahan) oleh senyawa Hidrogen Sianida (HCN) menghasilkan Laktonitril atau 2-hidroksipropananitril; sebuah senyawa hidroksinitril dengan gugus fungsi Sianonitril, dimana gugus siano (CN) dan hidroksi terikat pada atom karbon yang sama. Kemudian, Laktonitril dihidrasi ion Hidronium (H₃O⁺) untuk menjadikan senyawa tersebut cenderung asam, yang mana hasil dari reaksi itu membentuk Asam Laktat.

14. Bagaimana Cara Mensintesis Asetaldehida dari Asam Asetat?​


Jawaban beserta penjelasan:

Adapun beberapa cara untuk mensintesis asetaldehida dari asam asetat, antara lain:

[tex]\boxed{\bold{Sintesis\ Cara\ 1:}}[/tex]

[tex]CH_3COOH\xrightarrow[\text{LiAlH4/Eter}]{\text{4H+}}CH_3CH_2OH\xrightarrow[\text{CH2Cl2}]{\text{PCC}}CH_3CHO\\asam\ asetat\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ etil\ alkohol\ \ \ \ \ \ \ \ \ \ \ \ asetaldehida[/tex]

Asam asetat direduksi keasamannya dengan empat proton Hidrogen dan diubah menjadi alkohol primer dengan senyawa Litium Aluminium Hidrida (LiAlH₄) dalam larutan eter. Etil alkohol kemudian dioksidasi oleh Piridinium Klorokromat (C₅H₅NHClCrO₃) atau PCC dalam larutan Diklorometana (CH₂Cl₂) dan pada akhirnya membentuk Asetaldehida.

◈ ◈ ◈

[tex]\boxed{\bold{Sintesis\ Cara\ 2:}}[/tex]

[tex]CH_3COOH\xrightarrow{\text{PCl5}}\ CH_3COCl\ \xrightarrow[\text{BaSO4}]{\text{H2/Pd}}CH_3CHO\\asam\ asetat\ \ \ \ \ \ \ asetil\ klorida\ \ \ \ \ \ \ \ \ asetaldehida[/tex]

Asam asetat diklorinasi secara oksidatif oleh senyawa Fosfor Pentaklorida (PCl₅) menjadi Asetil Klorida. Lalu, dihidrogenasi dan diubah menjadi Asetaldehida dengan katalis logam Paladium (Pd) dalam larutan garam Barium Sulfat (BaSO₄).

18. Akseptor elektron terakhir pada proses fermentasi alkohol adalah.. A. oksigen B. asetaldehida C. piruvat D. fosfoenolpiruvatE. metanol


Akseptor elektron terakhir pada proses fermentasi alkohol adalah B. asetaldehida. Asetaldehida adalah senyawa yang dihasilkan dari perubahan asam piruvat dengan melepaskan karbondioksida. Adapun hasil akhir dari fermentasi alkohol adalah 2 etanol, 2 CO2, dan 2 ATP. Fermentasi alkohol adalah salah satu peristiwa respirasi an-aerob.  

Pembahasan

Peristiwa fermentasi disebut juga respirasi an-aerob yaitu proses pembebasan energi dalam bahan bakar organik seperti karbohidrat melalui rangkaian proses tanpa menggunakan oksigen. Fermentasi dapat terjadi pada jamur bersel satu, bakteri yang hidup di rawa atau lumpur miskin oksigen, sel hewan dan sel manusia.  

Karena proses ini tidak menggunakan oksigen, sehigga digunakan senyawa seperti asam piruvat dan asetaldehida sebagai akseptor (penerima) elektron terakhir dan mengikat H⁺. Fermentasi  terjadi melalui dua tahapan yaitu glikolisis dan transfer elektron. Proses glikolisis yang terjadi sama dengan respirasi aerob yaitu pengubahan glukosa menjadi asam piruvat, NADH dan ATP. Adapun trasnpor elektron terjadi dari NADH ke asam piruvat atau turunannya dan menghasilkan NAD⁺, lalu NAD⁺ masuk kembali ke glikolisis.  

Berdasar produk yang dihasilkan, fermentasi dibedakan menjadi dua yaitu:

1. Fermentasi alkohol

Fermentasi alkohol disebut juga peragian alkohol atau alkoholisasi. Fermentasi alkohol merupakan jenis fermentasi yang banyak digunakan manusia selama ribuan tahun dalam pengolahan bahan makanan. Khamir banyak digunakan dalam pembuatan roti dan minuman beralkohol. Jamur yang digunakan dalam proses fermentasi seperti Saccharomyces sp.  Jamur tersebut digunakan dalam proses pembuatan tapai. Dalam prosesnya, wadah harus ditutup rapat agar tidak ada oksigen sehingga jamur dapat melakukan repirasi an-aerob dengan sempurna. Jika wadah terbuka, maka proses akan terganggu dan memacu tumbuhya mikroorganisme lain yang berakibat rasa tapai asam dan tidak manis. Persamaan reaksi fermentasi alkohol:

Glukosa → 2asam piruvat → 2 asetaldehida → 2 etanol + 2CO₂ + 2ATP

Mekanisme rekasi pada persamaan diatas dapat dijelaskan sebagai berikut:

Bahan baku berupa glukosa (6C) diubah menjadi asam piruvat melalui rekasi glikolisis. Asam piruvat (3C) melepas CO2 menjadi asetaldehida (2C). Asetaldehida direduksi oleh NADH menghasilkan etanol dan melepaskan NAD⁺. NAD⁺ masuk kembali ke glikolisis, dan asetaldehida berperan sebagai akseptor elektron.  Pada fermentasi alkohol, setiap 1 molekul glukosa akan menghasilkan 2 etanol, 2 CO2, dan 2 ATP.    

2. Fermentasi asam laktat  

Peristiwa ini terjadi pada sel otot hewan dan manusia jika dalam kondisi kekurangan oksigen.  Terjadi saat katabolisme gula untuk membentuk ATP/energi melebihi pasokan oksigen dalam darah.  Eritrosit bersifat anaerob dan menghasilkan asam laktat karena tidak memiliki mitokondria.  Contoh mikroorganisme dalam fermentasi asam laktat adalah Streptococcus sp (yang juga digunakan dalam pembuatan keju dan yoghurt). Penimbunan asam laktat di otot manusia, akan menyebabkan berkurangnya elastisitas otot sehingga menimbulkan rasa lelah dan nyeri.  Rasa nyeri tersebut akan sedikit demi sedikit menghilang saat asam laktat diangkut oleh darah menuju hati untuk diubah menjadi asam piruvat kembali.  Persamaan reaksi fermentasi asam laktat yaitu:

Glukosa → 2 asam piruvat → 2 fosfoenol piruvat → 2asam laktat + 2 ATP

Mekanisme proses dari persamaan reaksi diatas dijelaskan sebagai berikut:

Bahan baku berupa glukosa (6C) diubah menjadi asam piruvat melalui reaksi glikolisis. Asam piruvat direduksi NADH sehingga membentuk asam laktat (3C) dan melepas NAD⁺. NAD⁺ masuk kembali ke glikolisis dan asam piruvat berperan sebagai akseptor elektron.

26. Tuliskan persamaan reaksi yang terjadi pasa Asetaldehida + PCl5


Tuliskan persamaan reaksi yang terjadi pasa Asetaldehida dengan PCl₅!

Persamaan reaksi yang terjadi antara asetaldehida dengan PCl₅ adalah :

CH₃CHO + PCl₅ → CH₃CHCl₂ + POCl₃

Pembahasan

Aldehid atau alkanal merupakan senyawa turuna alkana dengan gugus fungsi –COH. Aldehid memiliki rumus umum CnH2nO. Penamaan senyawa aldehid sama seperti alkana dengan mengganti akhiran a menjadi al.  

Contoh senyawa aldehid yaitu formaldehida atau metanal (H-COH) berwujud gas yang tidak berwarna dan berbau menyengat pada suhu kamar digunakan untuk pengawet mayat dan preparat, penghilang bau pada sumbu lilin dan lampu, serta sebagai reagensia.

Berikut adalah sifat fisika dan sifat kimia pada senyawa aldehid  

Memiliki titik didih lebih tinggi dibandingkan senyawa alkana dengan jumlah atom C yang samaMetanal berwujud gas pada suhu kamar, sedangkan aldehid dengan jumlah atom C lebih tinggi berwujud cair dan padatMetanal dan asetaldehida (CH₃CHO) dapat larut dalam air karena bersifat sedikit polarAldehid dapat dibuat dari oksidasi alkohol primer menggunakan oksidator kuat dalam suasana asamReaksi adisi dengan H₂ menghasilkan alkohol primer. Reaksi adisi merupakan reaksi masuknya atom atau gugus atom pada atom C berikatan rangkap.Reaksi dengan PCl₅ atau PX₅ berlebih menghasilkan geminal halida (R – CH – X₂). Geminal halida merupakan gugus alkil yang berikatan dengan atom CH yang terikat pada atom halogen (unsur pada golongan VIIA)Dapat dioksidasi dengan oksidator lemah yaitu pereaksi Tollens dan Fehling menghasilkan asam karboksilat. Jika aldehid direaksikan dengan pereaksi Tollens menghasilkan cermin perak berupa endapan Ag, dan jika direaksikan dengan pereaksi Fehling menghasilkan endapan merah bata (endapan Cu₂O)

28. Bagaimana Cara Mensintesis Asetaldehida dari Etanol?​


Jawaban beserta penjelasan:

Adapun cara untuk mensintesis asetaldehida dari etanol, yaitu:

[tex]\boxed{\bold{Sintesis\ Cara\ 1:}}[/tex]

[tex]CH_3CH_2OH\ \xrightarrow[Co]{H2}\ CH_3CHO\\etil\ alkohol\ \ \ \ \ \ \ \ \ \ \ \ \ etanal[/tex]

Etil alkohol (etanol) di-dehidrogenasi, mengeliminasi proton Hidrogen pada alkohol primer tersebut dengan menggunakan katalis logam Kobalt (Co). Sehingga, pada akhir reaksi terbentuklah Asetaldehida (etanal).

◈ ◈ ◈

[tex]\boxed{\bold{Sintesis\ Cara\ 2:}}[/tex]

[tex]CH_3CH_2OH+[O]\ \xrightarrow[H2SO4\ encer]{K2Cr2O7}\ CH_3CHO+H_2O\\etil\ alkohol\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ etanal[/tex]

Etil alkohol (etanol) dioksidasi oleh oksigen dalam senyawa pengoksidasi kuat Kalium Dikromat (Kr₂Cr₂O₇) yang diasamkan (acidified) reagen asam sulfat (H₂SO₄) encer. Setelah itu, pada akhir reaksi terbentuklah Asetaldehida (etanal).
