7. Perbedaan introvert dan outgoing introvert, can you?​


Introvert adalah sikap atau karakter seseorang yang memiliki orientasi subyektif secara mental dalam menjalani kehidupannya.

outgoing introvert ini adalah abu-abu di antara putihnya ekstrovert dan hitamnya introvert.

1. Teman baik outgoing introvert adalah media sosial.

Media sosial memiliki dua fungsi, yaitu tempat bersosialisasi dan juga tempat melarikan diri, seorang outgoing introvertakan merasa sangat nyaman berteman dengan media sosial karena media sosial bisa digunakannya untuk kabur dari banyak orang, tapi juga bisa membawanya menemukan orang lain jika dia mau.

2. Outgoing introvertcenderung tidak bisa menetap untuk kebanyakan hal.

Misalnya saja pekerjaan, pacaran, atau tempat favorit. Bukan berubah-ubah, tapi tidak bisa menetap. Jika di suatu pekerjaan dia sudah bertahan selama 3 tahun, maka sudah pasti dia harus mencari tempat baru. Soal urusan pasangan, outgoing introvert akan setia jika mendapat pasangan yang cocok.

3. Orang dengan kepribadian ini menganggap basa-basi itu sangat membuang waktu.

Outgoing introvert cenderung lebih tegas dari introvert. Jika introvert masih memikirkan bagaimana perasaan orang lain, outgoing introvert akan lebih dulu memikirkan dirinya. Basa-basi dengan orang lain membuatnya jengah dan membuang waktu. Lebih baik dia tidak berbasa-basi dan langsung ke intinya. Begitulah seorang outgoing introvert.

4. Outgoing introvert tidak suka sunyi, mereka akan menyalakan radio tanpa mendengarkannya hanya karena agar ada suara yang menemaninya.

18. bahasa jepang dari introvert dan ekstrovet adalah​


Introvert = 内向的 (Naikouteki)

Ekstrovert = 外向的 (Gaikouteki)

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

内向的 (Naikouteki)

Merupakan salah satu kata yang dapat diartikan "Introvert" dalam Bahasa Jepang. Merujuk pada orang yang kurang suka bersosialisasi, lebih suka dalam ruangan, dan lebih nyaman sendiri atau bersama orang-orang yang dekat dengannya saja. Kata ini bisa digunakan sebagai kata benda, ataupun sebagai kata sifat (Na).

外向的 (Gaikouteki)

Merupakan salah satu kata yang dapat diartikan "Ekstrovert" dalam Bahasa Jepang. Berkebalikan dari Naikouteki, kata ini merujuk pada orang yang suka bersosialisasi, lebih suka di luar ruangan, tidak nyaman bila hanya sendirian, dan bisa bergaul dengan siapa saja. Kata ini bisa digunakan sebagai kata benda, ataupun sebagai kata sifat (Na).

Contoh Kalimat :

1. 彼は内向的だ

= Kare wa naikouteki da

= Dia (laki) adalah introvert

2. 彼は外向的な人だ

= Kare wa gaikouteki na hito da

= Dia (laki) adalah orang yang ekstrovert

- - - - - -

Selain kata di atas, masih ada kata-kata lainnya yang memiliki arti serupa yang bisa digunakan. Salah satu contohnya seperti berikut.

陰キャ (In-kya) & 陽キャ (You-kya)

Kata 'In' di sini dapat diartikan sebagai "Yin", yang merujuk pada orang yang berkepribadian gelap dan cenderung negatif. Lalu kebalikannya yaitu 'You' yang berarti "Yang", yang merujuk pada orang yang berkepribadian cerah dan cenderung positif. Sedangkan kata 'Kya' di sini merupakan singkatan dari 'Kyara' yang berarti "Karakter". Jadi dapat disimpulkan bahwa 'In-kya' adalah "Introvert", dan 'You-kya' adalah "Ekstrovert". Kedua kata ini termasuk slang atau bahasa gaul.
