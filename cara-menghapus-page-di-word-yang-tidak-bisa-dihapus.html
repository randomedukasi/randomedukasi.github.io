1. Bagaimana cara menghapus page di Microsoft Word?


Jawaban:

Misalkan ada 3 halaman (Page) di Ms. Word dan ingin menghapus halaman ketiga, caranya:

- posisikan kursor di halaman 2

- tekan tombol Delete di keyboard secara terus menerus

- bila halaman 3 sudah terhapus, hentikan menekan tombol Delete

Penjelasan tambahan:

Amatilah contohnya lampiran:

- awalnya ada 3 halaman, bila posisi kursor sudah ada di halaman 2, maka keterangan di status bar adalah PAGE 2 OF 3 (ditunjukkan tanda panah merah)

- tekan tombol Delete di keyboard terus menerus hingga tampilan selanjutnya ditunjukkan tanda panah biru

- bila halaman 3 sudah terhapus, maka keterangan di status bar menjadi PAGE 2 OF 2 (ditunjukkan tanda panah hijau)

Cukup blok di halaman yang berlebih, lalu klik tombol "del" atau "delete", maka halaman tersebut secara otomoatis akan terhapus.

16. Jelaskan apa saja yang terdapat pada menu page word!






B. Icon Perintah di Group Page Setup

￼

Menu-menu yang terdapat di group Page Setup disediakan untuk melakukan pengaturan halaman, yakni dalam bidang ukuran kertas, margin halaman, pengaturan kolom teks, dll. Halaman dokumen dapat diatur sesuai dengan keinginan user, terutama untuk melakukan sinkronisasi dengan jenis dan ukuran kertas yang akan digunakan nantinya. Adapun menu perintah yang terdapat di group Page Setup yaitu : (1) Margins, (2) Orientations, (3) Size, (4) Columns, (5) Breaks, (6 ) Line Numbers, dan (7) Hypenation.

Margins: digunakan untuk mengatur batas teks di halaman dokumen.

Orientation: digunakan untuk mengatur posisi kertas.

Size: digunakan untuk mengatur ukuran halaman.

Columns: digunakan untuk mengatur jumlah kolom teks.

Breaks: digunakan untuk mengatur kontinuitas halaman maupun kolom teks.

Line Numbers: digunakan untuk mengatur kontinuitas nomor baris teks.

Hypenation: digunakan untuk memisahkan suku kata dengan tanda hubung secara otomatis.
