Jawaban:

Cardinal Numbers

A Cardinal Number says how many of something, such as one, two, three, four, five, etc.

Example: here are five coins:

five coins

It does not have fractions or decimals, it is only used for counting.

How to remember: "Cardinal is Counting"

Ordinal Numbers

An Ordinal Number tells us the position of something in a list.

1st, 2nd, 3rd, 4th, 5th and so on (seeCardinal and Ordinal Numbers Chart for more.)

Example: In this picture the girl is 2nd:

dog girl pups line

And the two pups are 3rd and 4th.

How to remember: "Ordinal says what Order things are in".

Nominal Numbers

A Nominal Number is a number used only as a name, or to identify something (not as an actual value or position)

soccer number 8

Examples:

the number on the back of a player: "8"

a zip code: "91210"

a model number: "380"

How to remember: "Nominal is a Name".

Example With Everything

cars racing

In this photo there are 6 cars. Car Number "99" (with the yellow roof) is in 1st position:

6 is a Cardinal Number (it tells how many)

1st is an Ordinal Number (it tells position)

"99" is a Nominal Number (it is basically just a name for the car)

17. ukuran kertas label verdors A-One dengan product number a-one 28394 ( )​


Jawaban:

Tahap I: Membuat Data Barcode di Excel

Langkah-Langkah:

Siapkan data barcode di Excel. Isi kolom B adalah kode yang akan dijadikan barcode.

Catatan: tutorial ini menggunakan contoh tipe barcode EAN-13.

Data barcode di Excel

Setelah selesai, simpan file Excel dan tutup dokumen Excel.

Tahap II: Membuat Label Barcode di Word

Buka dokumen Microsoft Word. Pada Mailings tab, klik Start Mail Merge (di Start Mail Merge grup), dan pilih Labels.

Pilih mail merge Labels

Pilih Label vendors dan Product number sesuai dengan ukuran label barcode yang digunakan. Tutorial ini menggunakan Label vendors: A-ONE dengan Product number: A-ONE 28394 untuk ukuran kertas A4 dengan isi 24 label per halaman. Klik OK bila sudah selesai. Label pada dokumen akan ditempatkan di tabel.

Pilih ukuran label

Selanjutnya ambil data Excel yang dibuat pada Tahap I di atas.

Pada Mailings tab, grup Start Mail Merge, klik Select Recipients.

Pilih Use Existing List. Cari lokasi dokumen dan kemudian klik Open untuk menampilkan kotak dialog Select Table.

Pada kotak dialog Select Table, klik lembar kerja yang berisi sumber data dan pastikan kotak First row of data contains column headers dicentang. Kemudian klik OK. Tampilan dokumen akan seperti gambar berikut. Semua label akan memiliki tulisan Next Record (kecuali pada label yang pertama).

Tampilan awal mail merge label barcode

Selanjutnya kita akan menempatkan data pada label. Klik pada label pertama.

Pada Mailings tab, grup Write & Insert Fields, klik Insert Merge Field dan pilih Label.

Ulangi hal yang sama untuk Barcode pada baris kedua.

Tampilan label pertama akan seperti gambar berikut.

Tampilan posisi merge field

Sekarang tekan tombol ALT + F9 untuk merubah tampilan ke kode field seperti gambar berikut.

Tampilan kode field

Langkah ini merupakan langkah yang paling penting. Di sini kita akan memodifikasi kode field mail merge agar bisa menampilkan gambar barcode. Ada 2 kode barcode yang bisa digunakan, yaitu: MERGEBARCODE dan DISPLAYBARCODE. Perbedaannya akan dijelaskan di akhir tutorial ini.

Bila ingin menggunakan MERGEBARCODE:

Highlight tulisan FIELD dan ubah menjadi BARCODE.

Ubah field ke barcode

Tambahkan tipe barcode EAN13.

Tambahkan field switch t untuk menampilkan kode di bawah gambar barcode.

Tip: anda bisa melihat field switch yang lain di situs Microsoft.

Ketik tipe barcode EAN13

Bila ingin menggunakan DISPLAYBARCODE:

Beri tanda petik dua (") pada MERGEFIELD Barcode.

Beri tanda petik dua

Kemudian pilih MERGEFIELD Barcode termasuk tanda petik dua (") seperti contoh gambar di atas dan tekan tombol CTRL + F9 untuk membuat kurung kurawal.

Pada posisi setelah kurung kurawal pembuka, ketik DISPLAYBARCODE.

Di bagian sebelum kurung kurawal penutup, ketik tipe barcode EAN13 dan field switch t untuk menampilkan kode di bawah gambar barcode seperti contoh berikut.

Ketik DISPLAYBARCODE

Setelah selesai pada label pertama, lanjutkan dengan menyisipkan data pada label yang lain. Caranya: pada Mailings tab, grup Write & Insert Fields, klik Update Labels. Word akan mengisi data pada semua label, sesuai dengan format dan layout yang kita buat pada label pertama.

Tampilan merge field pada semua label

Tekan tombol ALT + F9 untuk menyembunyikan kode field. Lihat hasilnya dengan mengklik tombol Preview Results. Tampilan barcode adalah seperti ini.

Tampilan gambar barcode

Untuk menyimpan hasil mail merge, klik tombol Finish & Merge, pilih Edit Individual Documents, klik All dan OK.

Kesimpulan

Jadi kita bisa membuat label barcode cukup dengan menggunakan mail merge Word (2013 dan 2016), tanpa harus menggunakan alat khusus.

Ada 2 kode yang bisa digunakan untuk menampilkan gambar barcode pada mail merge, yaitu: MERGEBARCODE dan DISPLAYBARCODE.

Anda bisa melihat aturan penulisannya untuk berbagai tipe barcode dan format tampilannya di situs Microsoft:

MERGEBARCODE

DISPLAYBARCODE

Kedua kode ini sama-sama bisa menampilkan gambar barcode. Perbedaannya:

Untuk melihat gambar barcode yang dibuat dengan MERGEBARCODE, kita harus menekan dulu tombol Preview Results. Sedangkan DISPLAYBARCODE akan langsung menampilkan gambar barcode.

Salah satu komentar di sumber ini menyatakan bahwa gambar barcode dari kedua kode ini bisa discan. Masalah hanya muncul pada barcode tipe QR. Dimana MERGEBARCODE akan menghasilkan garis-garis putih yang kadang membuatnya tidak bisa discan. Namun, masalah ini bisa diatasi dengan menggunakan DISPLAYBARCODE.
