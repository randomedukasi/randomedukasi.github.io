1. Berapa rupiah kita menerima bunga apabila kita menabung Rp. 100.000.000,- dengan suku bunga 4 1/2% selama 3 bulan?


Tabungan = Rp100.000.000,-
Suku bunga = 4 ½ % per tahun
Lama menabung = 3 bulan

Bunga = Nilai tabungan × suku bunga × lama
Bunga = Rp100.000.000 ×4 ½ % × 3/12
Bunga = Rp1.125.000,-

2. Berapa rupiah kita menerima bunga apabila kita menabung Rp. 100.000.000,- dengan suku bunga 4 1/2% selama 3 bulan?


Jika suku bungan bulanan:
Bunga = Tabungan pokok x tarif x waktu
             = 100.000.000 x 4,5% x 3
             = 13.500.000

Jika suku bungan tahunan:
Bunga = Tabungan pokok x tarif x waktu
             = 100.000.000 x 4,5% x 3/12
             = 1.125.000

Angsuran setiap bulan dapat dihitung menggunakan rumus angsuran = (pinjaman * suku bunga) / (1 - (1 + suku bunga) ^ -jangka waktu), di mana jangka waktu dalam bulan.

Bunga = 12% / tahun

              = 12% / 12

              = 1% / bulan

  bunga 10 bulan = 10 x 1% x saldo

                             = 10% x Rp 1.000.000

                             = Rp 100.000

   = saldo + (4 x bunga)

   = Rp 1.000.000 + ( 4 x 12%)

   = Rp 1.000.000 + ( 4 x [tex]\frac{12}{100}[/tex] x Rp 1.000.000)

   = Rp 1.000.000 + (4 x Rp 120.000)

   = Rp 1.480.000
