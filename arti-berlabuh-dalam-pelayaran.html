9. apa arti berlayar,kemudi.,gerbong,pelabuhan,dan stasiun cepat ya thank you


Untuk menentukan arti dari suatu kata, kita harus mengetahui jenis kata tersebut. 

1. Berlayar
Jenis kata : Kata kerja
arti berlayar : Menggunakan (memakai) layar, mengarungi lautan, bepergian dengan kapal (perahu)

Kata berlayar berdasar dari kata layar dan memperoleh imbuhan ber- yang bermakna memakai dan melakukan.
Contoh kalimat berlayar dengan makna memakai:
- Kapal itu berlayar dengan memanfaatkan tenaga angin yang mendorong kapal itu bergerak.
Contoh kalimat berlayar dengan makna melakukan:
- Nakhoda itu telah berlayar mengelilingi Samudra Pasifik.

2. Kemudi
Jenis kata : kata benda
arti kemudi : perkakas pada kendaraan (mobil, pesawat terbang, kapal, dan sebagainya) yang gunanya untuk mengatur arah perjalanan
Contoh kalimat dengan kata kemudi:
- Ayah memutar kemudi mobil dengan cepat menghindari sepeda yang ada di depannya.
- Di dalam ruang lokomtof, terdapat ruang kemudi.
3. Gerbong
Jenis kata : kata benda
Arti gerbong : wagon kereta api (untuk orang atau barang)
Contoh kalimat dengan kata gerbong:
- Kereta api terpanjang saat ini dapat membawa 30 gerbong.
- Kereta api terdiri dari satu lokomotif dan beberapa gerbong.

4. Pelabuhan
Jenis kata : kata benda
Arti pelabuhan :  tempat berlabuh
Kata pelabuhan berasal dari kata dasar labuh yang memperoleh imbuhan pe- dan an-, yang berarti tempat
Contoh kalimat dengan kata pelabuhan:
- Kapal penangkap ikan merapat di pelabuhan setengah jam yang lalu.

5. Stasiun
Jenis kata : kata benda
Arti pelabuhan : tempat menunggu bagi calon penumpang kereta api dan sebagainya; tempat perhentian kereta api dan sebagainya
Contoh kalimat dengan kata stasiun:
- Kami menjemput paman di stasiun.
- Kami menunggu kereta yang membawa kami mudik di stasiun.

25. Sebuah kapal berlayar dari pelabuhan A ke pelabuhan B sejauh 180 km ke arah barat, kemudian berlayar dari pelabuhan B ke pelabuhan C sejauh 240 km ke arah selatan. Jika kapal akan berlayar dari pelabuhan C kembali ke pelabuhan A, jarak terdekat antara pelabuhan C dan A adalah .... km.


Jawaban:

300 km

Penjelasan:

C² = A² + B²

C² = 180² + 240²

C² = 32.400 + 57.600

C² = 90.000

C = √90.000

C = 300 km

>>Perpindahan & Jarak<<
