4. Tolong Sebutkan bentuk coding web untukk membuat link ke facebook dan yahooo


Jawaban:

Untuk menyisipkan tautan link dalam website kita menuju Facebook atau Yahoo, kita bisa gunakan tag <a> dalam HTML.

Berikut saya contohkan pengimplementasian kodenya dalam file HTML.

note : karena rules dalam brainly tidak boleh mencantumkan suatu website, maka saya samarkan saja ya kak.

<html>

<body>

<a href="facebook dot com">Klik untuk menuju ke FB</a>

<a href="yahoo dot com">Klik untuk menuju ke Yahoo</a>

</body>

</html>

Jadi, untuk membuat link dalam web, kita perlu menambahkan tag <a> pada dokumen HTML dalam web yang kita buat.
