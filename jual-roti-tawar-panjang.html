4. Seorang pengusaha roti setiap hari mengeluarkan biaya sebesar Rp 1.200.000 untuk memproduksi roti tawar. Ia mengembangkan bisnis lain, yakni dengan memproduksi roti pisang. Seorang pengusaha roti setiap hari mengeluarkan biaya sebesar Rp 1.200.000 untuk memproduksi roti tawar. Ia mengembangkan bisnis lain, yakni dengan memproduksi roti pisang. Perbandingan biaya produksi roti tawar dan roti pisang adalah 2 : 3. Jika ia mendapatkan keuntungan 15% dari total penjualan roti tawar dan roti pisang, maka hasil penjualannya adalah Rp 2.550.000.

Penyelesaian soal :

Diketahui : biaya produksi roti tawar = Rp 1.200.000

                  Perbandingan biaya produksi roti tawar : roti pisang = 2 : 3

                   keuntungan penjualan total = 15%

ditanya : hasil penjualan roti tawar dan roti pisang ?

jawab :

hitung biaya produksi roti pisang menggunakan rumus perbandingan senilai berikut ini :

misalkan: biaya produksi roti pisang = x

[tex]\frac{1.200.000}{x}[/tex] = [tex]\frac{2}{3}[/tex]

2x = 1.200.000 × 3

2x = 3.600.000

x = 3.600.000 / 2

x = 1.800.000

hitung biaya produksi total roti tawar dan roti pisang yaitu :

biaya produksi total = 1.200.000 + 1.800.000 = 3.000.000

kemudian hitung harga keuntungan yang diperoleh :

[tex]\frac{15}{100}[/tex] × 3.000.000 = 450.000

hitung hasil penjualan yang diperoleh :

hasil penjualan = biaya produksi total - keuntungan

                          = 3.000.000 - 450.000

                          = 2.550.000

∴ kesimpulan hasil penjualan roti tawar dan roti pisang adalah Rp 2.550.000.

Pembahasan :

KEUNTUNGAN

Besarnya keuntungan biasanya diukur relatif terhadap harga beli, sehingga keuntungan seringkali dinyatakan dalam persentase.

Untung atau rugi adalah selisih (perbedaan) antara harga jual dengan harga beli. Jika harga jual lebih besar dari harga beli maka selisih tersebut disebut untung, Jika harga jual lebih kecil dari harga beli maka selisih tersebut disebut rugi.

Untung = harga jual - harga beli

atau

keuntungan = % untung × harga beli

Sebelum menyelesaikan soal di atas, mari kita ketahui dulu arti dari model matematika. Arti dari model matematika adalah cara menterjemahkan suatu problem matematika menjadi bahasa matematika dengan menggunakan persamaan, pertidaksamaan, maupun fungsi. 

Untuk mengerjakan soal di atas dapat dilakukan dengan cara sebagai berikut: 

Asumsi untuk membentuk persamaan adalah sebagai berikut:

Persamaan untuk jumlah roti: 
x = roti manis
y = roti tawar
x+y < 600
dimana untuk x dan y adalah bilangan cacah dengan nilai positif karena merupakan banyaknya roti yang tidak dapat dalam bentuk negatif.
Untuk itu : x > 0 , y > 0 

Persamaan untuk modal: 
5.000x + 4.500y < 600.000

Keuntungan: 
roti manis : 500x 
roti tawar : 600y

Dari penjabaran di atas, maka fungsi untuk memperoleh keuntungan sebesar-besarnya adalah:
f(x,y) = 500x + 600y

Dari penjualan roti-roti ini, ia memperoleh keuntungan rp.500,00 dari sebungkus roti manis dan rp.600,00 dari sebungkus roti tawar. Jika modal yg di miliki Umar Bakri rp.600,000,00, buatlah model matematika dengan tujuan untuk memperoleh keuntungan sebesar-besarnya?


Program linear adalah metode penentuan nilai optimum dari suatu persoalan linear. Nilai optimum (maksimum atau minimum) diperoleh dari nilai dalam suatu himpunan penyelesaiaan persoalan linear. Di dalam persoalan linear terdapat fungsi linear yang disebut sebagai fungsi objektif. Persyaratan, batasan, dan kendala dalam persoalan linear merupakan sistem pertidaksamaan linear.

Pembahasan

Misal, x menyatakan roti manis, dan y menyatakan roti tawar.

Permasalahan tersebut ditulis ke dalam model matematikanya yaitu

x + y ≤ 600

x ≥ 0

y ≥ 0

F(x,y) = 500x + 600y

Pertidaksamaan tersebut digambar grafiknya dan didapatkan titik-titik pojoknya, yaitu A(0,600), O(0,0), dan B(600,0).

Masing-masing titik pojok disubstitusikan ke dalam F(x,y) = 500x +600y

A(0,600) ⇒ 500(0) +600(600) = 3.600.000

O(0,0) ⇒ 500(0) +600(0) = 0

B(600,0) ⇒ 500(600) +600(0) = 3.000.000

Jadi, keuntungan maksimumnya yaitu Rp 3.600.000 dengan menjual roti tawar sebanyak 600 bungkus.

Pak Roni seorang pedagang roti yang menjual roti manis dan roti tawar dengan menggunakan gerobak. Gerobak tersebut hanya dapat memuat 120 bungkus roti. Harga pembelian roti manis sebesar Rp6.000,00 per bungkus dan roti tawar sebesar Rp4.000,00 per bungkus dengan modal yang dimiliki pak Roni sebesar Rp600.000,00. Jawab :
Misalkan banyak roti manis = x, banyak roti tawar = y, dan P = pendapatan. Dari data di atas kita dapat menyusun model matematika sebagai berikut.
Gerobak tersebut hanya dapat memuat 120 bungkus roti.
x + y ≤ 120

Harga pembelian roti manis sebesar Rp6.000,00 per bungkus dan roti tawar sebesar Rp4.000,00 per bungkus dengan modal yang dimiliki pak Roni sebesar Rp600.000,00
6.000x + 4.000y ≤ 600.000
⇔6x + 4y ≤ 600
⇔3x + 2y ≤ 300

Karena x merupakan banyaknya roti manis dan y banyaknya roti tawar, maka x dan y adalah bilangan bulat yang tidak negatif (bilangan cacah).
x ≥ 0, y ≥ 0, dengan x, y ∈ C.
fungsi biaya sebuah roti manis dan sebuah roti tawar dapat ditulis sebagai berikut.
memaksimalkan fungsi tujuan P(x,y) = 3.000x + 1.000y.

Gambar terlampir.
Koordinat titik potong dapat ditentukan sebagai berikut. Pertidaksamaan linear kita ubah menjadi persamaan linear, sehingga diperoleh sistem persamaan linear dua variabel :
x + y = 120
3x + 2y = 300
Kita selesaikan dengan metode eliminasi dan substitusi.
x + y = 120      |.2| 2x + 2y = 240
3x + 2y = 300 |.1|  3x + 2y = 300
                             ____________-
                              - x = -60 ⇔ x = 60
KIta substitusikan x = 60 ke persamaan 
x + y = 120
⇔ 60 + y = 120
⇔ y = 120 - 60
⇔ y = 60

Koordinat titik potongnya adalah (60, 60).

Keuntungan maksimum f(x, y) = 3.000x + 1.000y
f(0, 120) = 0 + 120 x 1.000 = 120.000
f(120, 0) = 3.000 x 120 + 0 = 360.000
f(0, 150) = 0 + 150 x 1.000 = 150.000
f(100, 0) = 3.000 x 100 + 0 = 300.000
f(60, 60) = 60 x 3.000 + 60 x 1.000 = 180.000 + 60.000 = 240.000

Jadi, keuntungan maksimumnya adalah Rp360.000,00.
