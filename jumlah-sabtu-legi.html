8. perbandingan kelereng egi dan legi 3:2.jika selisih kelereng mereka 8,jumlah kelereng egi dan legi adalh...


Perbandingan kelereng egi dan legi 3 : 2.jika selisih kelereng mereka 8, jumlah kelereng egi dan legi adalah ...

Misal perbandingan A dan B adalah m : n

m berarti rasio untuk A, dan n adalah rasio untuk B

Jumlah perbandingan (jumlah rasio) = m + n

Selisih perbandingan (selisih rasio) = m - n dengan m > n

Jumlah A dan B adalah K

Selisih A dan B adalah L

Maka, berlaku rumus,

A = [tex]\frac{m}{m+n}[/tex] x K

B = [tex]\frac{n}{m+n}[/tex] x K

A = [tex]\frac{n}{m-n}[/tex] x L

B = [tex]\frac{n}{m-n}[/tex] x L

A = [tex]\frac{m}{n}[/tex] x B

B = [tex]\frac{n}{m}[/tex] x A

A + B = [tex]\frac{m+n}{m-n}[/tex] x L

Pembahasan

Diketahui:

Perbandingan Egi dan Legi = 3 : 2

m = 3

n = 2

Selisih kelereng mereka = 8

L = 8

Ditanya:

Jumlah kelereng Egi dan Legi = ... ? (K)

Jawab:

K = [tex]\frac{m+n}{m-n}[/tex] x L

  = [tex]\frac{3+2}{3-2}[/tex] x 8

  = [tex]\frac{5}{1}[/tex] x 8

  = 5 x 8

  = 40


Jadi Jumlah kereleng Egi dan Legi adalah 40 kelereng.
