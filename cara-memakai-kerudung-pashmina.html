17. Novi mempunyai 2 buah blazer dan 5 buah kerudung yang berlainan warna. Berapa cara Novi dapat memakai blazer dan kerudung?


2 buah blazer dan 5 buah kerudung.

Cara memakai blazer : 2C1 = 2
Cara memakai kerudung : 5C1 = 5

Cara memakai blazer dan kerudung :
2C1 x 5C1 = 2 x 5 = 10 cara

Jadi, ada 10 cara untuk memakai blazer dan kerudung

20. Dari 40 anak dalam suatu kelas tercatat 20 anak memakai kerudung,12 anak memakai kacamata,dan 17 anak tidak memakai keduanya. banyak anak yang memakai kerudung dan kacamata adalah....anak


Materi : Himpunan

S = Himpunan Semesta ( 40 )

a = Himpunan Kerudung ( 20 )

b = Himpunan Kacamata ( 12 )

n = Himpunan Keduanya ( ... )

ab^c = Tidak Keduanya ( 17 )

\_______________________/

( a - n ) + ( b - n ) + n = S - ab^c

( 20 - n ) + ( 12 - n ) + n = 40 - 17

32 - n = 23

n = 32 - 23

{ n = 9 siswa }
