10. paman ke Jogja naik kereta api pukul 08.15 tiba di Jogja 11.15 jika jarak Jakarta 180 km berapa kecepatan kereta​


Jawaban:

Diketahui :

t = 11.15-08.15 = 03.00 = 3 jam

s = 180 km

Ditanya : v =?

Jawab :

v = s/t

v = 180/ 3

v = 60 km/jam

Jawaban beserta Cara:

Diketahui : Naik kereta pukul 08.15

Sampai Jogja pukul 11.15

Jarak Jakarta-Jogja 180 km

Ditanya : Kecepatan Kereta

Dijawab : • 11.15 - 08.15 = 3 jam

• t = waktu ( jam )

S = Jarak ( km )

V = Kecepatan ( km/jam )

[tex]v = \frac{s}{t} = \frac{180 \: km}{3 \: jam} = 60 \: km \: per \: jam[/tex]

Jadi, kecepatan kereta adalah 60 km/jam

14. Sebuah kereta bergerak dari soo ke jogja yang berjarak 60 km dalam waktu 45 menit. hitunglah kelajuan kereta tersebut ​


Jawaban:

[tex]\bf\large{Kecepatan\ kereta\ 80\ km/jam} [/tex]

Penjelasan dengan langkah-langkah:

[tex]\sf\large{Diketahui: } [/tex]

Jarak dari Solo ke Jogja:

[tex]\sf\large{S=60\ km} [/tex]

Waktu tempuh kereta:

[tex]\sf\large{t=45\ menit=\frac{3}{4}\ jam} [/tex]

Ditanyakan besarnya kecepatan kereta

Jawab:

Gunakan rumus gerak lurus beraturan (GLB):

[tex]\sf\large{S=V×t} [/tex]

[tex]\sf\large{V}=\Large{\frac{S}{t}} [/tex]

[tex]\sf\large{V}=\Large{\frac{60}{\Large{\frac{3}{4}}}}\large{=60×}\Large{\frac{4}{3}} [/tex]

[tex]\sf\large{V=80\ km/jam} [/tex]

Jadi kecepatan kereta tersebut adalah [tex]\sf\large8\ km/jam [/tex]

23. kereta api ekonomi berjalan menempuh jarak 400km dengn kelajuan 40km/jam. kereta api eksekutif berjalan 2 jam kemudian dan tuba di tempat dan waktu yang bersamaan dengan kereta api ekonomi.kelajuan kereta api eksekutif tersebut adalah​


Jawaban:

50 km/jam

Penjelasan:

Waktu tempuh kereta ekonomi

t = 400/40 = 10 jam

Waktu tempuh kereta eksekutif

t = 10 - 2 = 8 jam

Kelajuan kereta eksekutif

V = 400/8 = 50 km/jam

Jawaban:

Kereta Api Ekonomi

S = 400 KM

V = 40 KM/Jam

T = S÷V

= 400 ÷ 40 = 10 Jam

Kereta Api Exekutif

T = 10 - 2 = 8 Jam

S = 400 KM

V = S ÷ T

= 400 ÷ 8 = 50 Km/Jam

Utamakan Berterima Kasih •_•
