11. Hitung dan tentukan lokasi titik henti bila seseorang ingin membuat rest area antara Jakarta-Cilegon, jika diketahui Jakarta berpenduduk 10.500.000 jiwa dan Cilegon berpenduduk 420.000 jiwa. Jarak Jakarta-Cilegon = 102 km.


Penjelasan:

Rumus Titik Henti:

[tex] \frac{d}{1 + \sqrt{ \frac{p \: besar}{p \: kecil} } } [/tex]

Keterangan:

d = distance (jarak)

p = populasi (jumlah penduduk)

Catatan: titik henti harus lebih dekat dengan kota yang lebih sedikit penduduknya

[tex] \frac{102 \: km}{1 + \sqrt{ \frac{10.500.000}{420.000} } } = \frac{102 \: km}{1 + \sqrt{25} } = \frac{102 \: km}{1 + 5} = \frac{102 \: km}{6} = 17 \: km[/tex]

Jadi rest area dapat dibangun di titik 17 km dari Cilegon
