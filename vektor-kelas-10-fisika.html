Jika dua vektor
V₁ = V₂ = x dan α = 120°
tunjukkan R = x.

cos α = cos 120° = - ½

R² = V₁² + V₂² + 2 V₁ V₂ cos α
R² = x² + x² + 2 • x • x (-½)
R² = x² + x² - x²
R² = x²
R = √x² = x. ✓ tertunjuk

Metode Geometrik :

Kelebihan :

1. Penggambaran vektor sesuai dengan skala sebenarnya.

2. Arah vektor sesuai dengan arah anak panah vektor

3. Besar vektor sesuai dengan panjangnya.

Kekurangan :

Dapat menimbulkan kesalahan sistematis ketika menghitung dua vektor atau lebih.

Metode Analitik :

Kelebihan :

Nilai yang dihasilkan adalah exact (Tepat).

Kekurangan :

Harus hapal rumus dan memerlukan banyak waktu untuk menghitung rumusnya.

D. 50 N

Penjelasan:

F12 = F2 – F1

F12 = 60 N – 20 N

F12 = 40 N

Jadi, resultan gaya F12 adalah 40 N ke arah F2, 37° dari sumbu x positif.

Selanjutnya kita hitung resultan gaya F12 dengan F3.

R² = F12² + F3² + 2 F12 F3 cos θ

R² = (40 N)² + (30 N)² + 2 (40 N) (30 N) cos (37 + 53)°

R² = 1.600 N² + 900 N² + 2.400 N² cos 90°

R² = 2.500 N² + 2.400 N² x 0

R² = 2.500 N²

R = 50 N

Jadi, resultan ketiga gaya pada gambar adalah 50 N (D).

CMIIW
Besaran vektor adalah besaran yang memiliki besar/nilai dan arah
Besaran skalar adalah besaran yang hanya memilki besar/nilai

contoh
besaran skalar: massa, panjang, waktu, energi, daya
besaran vektor: kecepatan, percepatan, gaya, momentum, impuls
