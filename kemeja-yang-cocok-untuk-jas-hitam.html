15. 3. Di sebuah toko terdapat 48 jas hujan dan 12 di antaranya berwarna hitam. Jika 6 jas hujan diambil secara acak, maka peluang terambilnya tepat tiga jas hujan berwarna hitam adalah ....bantuan nya dong kak besok harus selesai​


Penjelasan dengan langkah-langkah:

Jawaban : 2.525.600 / 12.271.512

Perhatikan penjelasan berikut ya.

Ingat kembali:

1) rumus peluang

P(A) = n(A) / n(S)

dengan :

P(A) = peluang kejadian A

n(A) = banyak kejadian A

n(S) = banyak seluruh kejadian

2) rumus kombinasi

nCr = n! / [(n-r)! · r!]

dengan:

x = banyak kejadian diharapkan

C = kombinasi

r = banyak seluruh kejadian

Diketahui di sebuah toko terdapat 48 jas hujan dan 12 diantaranya berwarna hitam. Jika 6 jas hujan diambil secara acak, maka peluang terambilnya tepat tiga jas berwarna hitam dapat ditentukan sebagai berikut:

n(S) = 48C6 ----> dari 48 jas hujan akan diambil 6 jas hujan secara acak

n(S) = 48! / [(48-6)! · 6!]

n(S) = 48! / (42! · 6!)

n(S) = (48x47x46x45x44x43x42!) / (42!x6x5x4x3x2x1)

n(S) = (48x47x46x45x44x43) / (6x5x4x3x2x1)

n(S) = 8.835.488.640 / 720

n(S) = 12.271.512

n(A) = 12C3 x 42C3 ------> 12 jas hujan berwarna hitam akan diambil 3 yang berwarna hitam

n(A) = [12! / ((12-3)! · 6!)] x [42! / [(42-3)! · 3!]

n(A) = [12! / (9! · 6!)] x [42! / [39! · 3!]

n(A) = [(12x11x10x9!) / ((9!x3x2x1)] x [(42x41x40x39!)/ (39!x3x2x1)]

n(A) = [(12x11x10) / ((3x2x1)] x [(42x41x40)/ (3x2x1)]

n(A) = (1.320 / 6) x (68.880 / 6)

n(A) = 220 x 11.480

n(A) = 2.525.600

Sehingga:

P(A) = n(A) / n(S)

P(A) = 2.525.600 / 12.271.512

Jadi, peluang terambilnya tepat tiga jas berwarna hitam adalah 2.525.600 / 12.271.512.
