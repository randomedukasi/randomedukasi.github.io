9. Bagaimana cara menampilkan internet access di laptop


Jawaban:

cara pertama :

Klik Start, carilah Settings; Klik opsi Change PC Settings, klik Network; Pada kolom Wireless devices, nyalakan WiFi sampai muncul tanda On; Jangan lupa untuk mematikan Airplane Mode jika ia masih menyala.

cara kedua :

Masuk ke Control Panel; Buka Network and Internet, kemudian klik Network and Sharing Center; Klik Change adapter settings; Klik kanan pada WiFi yang sedang kamu gunakan, klik Enable

cara ketiga :

Pada taskbar, klik ikon network; Klik WiFi sampai boks berubah warna menjadi biru; Pilih network yang ingin kamu sambung, klik Connect; Setelah itu, kamu hanya perlu memasukkan password WiFi.

cara keempat:

menekan tombol function key atau (F8) atau  (F12)

Penjelasan:

Semua laptop keluaran baru pasti dilengkapi dengan adapter WiFi bawaan. Jika tidak, siapa yang mau membelinya mengingat saat ini WiFi adalah kebutuhan untuk mayoritas orang.

Laptop kita tak selalu tersambung dengan WiFi. Biasanya, kita harus menyalakannya secara manual untuk mengaktifkannya.

Bagaimana cara menginstall aplikasi pada laptop/komputer?

Jika telah memiliki file instalasi aplikasinya,

1. Buka File Explorer

2. Buka folder dimana file instalasi aplikasi berada (file umunya berekstensi .exe atau .msi)

3. Klik 2x (Double klik) pada file tersebut

4. Baca dan ikuti instruksi pada dialog instalasi yang muncul (misalnya setujui EULA, klik tombol Next, dll.)

5. Tunggu proses intalasi selesai, dan klik tombol Finish

6. Aplikasi telah terinstall

• Penjelas:

Untuk membuka aplikasi yang telah terinstall, dapat klik 2x pada shortcut aplikasi yang ada di Dekstop, atau dengan Klik tombol start > All program > Klik aplikasi yang telah diinstall.

21. pengambilan gambar dari layar komputer dengan menggunakan aplikasi rekam layar dan dapat ditambahkan penggunaan lensa yang terpasang pada laptop atau bahkan yang sengaja dipasang untuk perekaman gambar


Untuk pengambilan gambar pada komputer dengan menggunakan aplikasi dapat dilakukan dengan cara sebagai berikut.

- Menggunakan Snipping Tool
Snipping tool adalah aplikasi yang telah terinstall ketika kita menggunakan Windows. Dengan menggunakan Snipping Tool kita dapat hanya tinggal meng-klik menu "New" dan kemudian kita akan memasuki mode pengambilan gambar. Kemudian, memilih layar yang ingin diambil gambarnya. Terakhir, tentukan lokasi tempat gambar ingin disimpan. Untuk Snipping Tool sendiri, aplikasi ini bukanlah untuk melakukan perekaman video, melainkan hanya mengambil gambar pada layar atau biasa disebut dengan screen capturing.

- Fraps
Fraps adalah salah satu screen recording terkenal yang sering digunakan oleh banyak orang, karena pengaturan dan tampilan yang sangat userfriendly. Bahkan, untuk memulai screen recording pun kita hanya memerlukan satu buah tombol. Selain itu, kita dapat mengatur berapa frame yang diinginkan dalam satu detiknya. Fraps, untuk hasil akhirnya, memiliki kualitas gambar yang tinggi pula ketika kita melakukan penyimpanan file. Tetapi tingginya kualitas gambar menyebabkan Fraps memakan memori yang cukup terbilang besar untuk satu kali screen recording. Jadi, jangan lupa mempersiapkan komputer untuk jumlah memori yang besar. Selain itu, jika ingin menggunakan Fraps kita harus membeli aplikasi tersebut. 

- Bandicam
Tidak jauh berbeda dengan Fraps, Bandicam adalah alternatif bagi teman-teman yang suka melakukan screen recording. Kualitas yang tinggi, suara ouotput yang baik, menadikannya Bandicam menjadi raja di masa lalunya. Selain itu, Bandicam menjadi primadona bagi orang yang sedang ingin mempelajari screen recording karena terdapat versi gratis yang dapat kita unduh padad website dari Bandicam itu sendiri

Untuk mempelajari lebih banyak mengenai screen recording dan screen capturing dapat membuka:

- Pengertian screen recording https://brainly.co.id/tugas/5781445
- Perbedaan video processing dan screen recording https://brainly.co.id/tugas/5935587
