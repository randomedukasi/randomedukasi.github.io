8. f4) Panjang diameter sebuah kerucut 28 cm. Jika tingginya 48 cm, maka luas seluruh kerucut tersebut adalah ..​


Jawaban:

Luas kerucut adalah 2.816 cm²

Penjelasan dengan langkah-langkah:

Rumus volume kerucut adalah πr²t

Rumus luas kerucut adalah πr (r + s)

dengan s = √r² + √t²

Simak pembahasan soal berikut!

Diketahui :

Diameter kerucut = 28 cm

Tinggi kerucut = 48 cm

Ditanya :

Luas kerucut ?

Dijawab :

Menentukan nilai s

Jika diameter = 28 cm, maka jari - jari = x 28 = 14 cm

s = √r² + √t²

s = √14² + √48²

s = √196 + √2.304

s = √2.500

s = 50 cm

Setelah mengetahui nilai s, maka kita bisa mencari luas kerucut

Luas kerucut = πr (r + s)

Luas kerucut = 22/7 x 14 (14 + 50)

Luas kerucut = 22 x 2 x 64

Luas kerucut = 2.816 cm²
