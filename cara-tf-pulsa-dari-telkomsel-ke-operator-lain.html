22. tolong bantu nanti aku tf pulsa 20k​


Jawaban:

Jawabannya adalah suku pertama = -11/2, beda = 1/2 dan rumus suku ke-n adalah Un = n/2 - 6

Konsep barisan aritmatika :

Un = a + (n-1)b

Keterangan :

Un = suku ke-n

a = U1 = suku pertama

b = beda

Jawab :

suku ke−3=−4 1/2 -> a + 2b = - 9/2

suku ke - 8=−2 -> a + 7b = -2

Eliminasi kedua persamaan

a + 2b = -9/2

a + 7b = -2

---------------- (-)

___-5b = -5/2

b = -5/2 x 1/-5

b = 1/2

Substitusi b = 1/2 ke salah satu persamaan

a + 2(1/2) = -9/2

a + 1 = -9/2

a = -9/2 - 1

a = -9/2 - 2/2

a = -11/2

Rumus suku ke-n

Un = a + (n-1)b

= -11/2 + (n-1)1/2

= -11/2 + n/2 - 1/2

= n/2 - 12/2

= n/2 - 6

Jadi suku pertama = -11/2, beda = 1/2 dan rumus suku ke-n adalah Un = n/2 - 6
