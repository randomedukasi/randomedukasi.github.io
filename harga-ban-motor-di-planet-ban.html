asumsikan ban mobil = a dan ban motor = b, maka

3a + 4b = 2.050.000 (pers. 1)

a + 3b = 1.350.000 (pers. 2)

substitusi pers. 2 menjadi

a = 1.350.000 - 3b (pers. 3)

masukkan pers. 3 ke pers. 1

3a + 4b = 2.050.000

3 (1.350.000 - 3b) + 4b = 2.050.000

4.050.000 - 9b + 4b = 2.050.000

-5b = 2.050.000 - 4.050.000

-5b = -2000.000

b = -2000.000/-5

b = 400.000

masukkan nilai b ke pers. 2 untuk mencari nilai a

a + 3b = 1.350.000

a + 3.400.000= 1.350.000

a + 1.200.000 = 1.350.000

a = 1.350.000 - 1.200.000

a = 150.000

maka nilai

10a + 5b adalah

(10 x 150.000) + (5 x 400.000)

= 1.500.000 + 2.000.000

= 3.500.000
