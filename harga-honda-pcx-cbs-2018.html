Pada sebuah brosur sepeda motor tertulis 2 buah honda genio + 1 buah honda pcx = rp60.000.000. Jika 1 buah honda genio + 1 buah honda pcx = rp50.000.000. Setelah melihat brosur, Daffa tertarik dan membeli 1 buah honda genio, maka ia harus membayar Rp.10.000.000.

Penjelasan dengan langkah-langkah

Diketahui :

2 honda genio + 1 honda pcx = Rp.60.000.0001 honda genio + 1 honda pcx = Rp.50.000.000

Harga 1 buah honda genio ?

Jawab :

x = honda genioy = honda pcx

[tex] \rm 2x+y = Rp.60.000.000[/tex]...(1)

[tex] \rm x+y = Rp.50.000.000[/tex]...(2)

Eliminasi persamaan diatas dengan mengurangi persamaan (1) dengan (2) :

[tex] \rm 2x+y = Rp.60.000.000[/tex]

[tex] \rm x = Rp.10.000.000[/tex]

6. definisi PCX format gambar


Jawaban:

PCX adalah sebuah format gambar yang dikembangkan oleh ZSoft Corporation yang berasal dari Marietta, Georgia, Amerika Serikat. PCX adalah format gambar yang aslinya diciptakan untuk PC Paintbrush (PCX = "Personal Computer eXchange") dan telah menjadi salah satu format gambar DOS yang paling diterima, meskipun keberadaannya sejak DOS telah digantikan oleh format yang lebih canggih semisal GIF, JPG dan PNG.

Sebuah file PCX adalah format file berbentuk raster; kepala filenya menyimpan informasi tentang perangkat keras monitor (resolusi layar, kedalaman warna, informasi palet, jumlah bit dst.) secara terpisah dari gambarnya. Hal ini membuat sebuah file PCX dapat dipindahkan dan dibuka di perangkat komputer lain dengan perangkat keras yang berbeda. Sebuah file PCX biasanya dapat menyimpan gambar-gambar dengan indeks palet 2 hingga 4 bit dan 16 hingga 256 warna, namun sekarang telah ditambah untuk menyimpan gambar-gambar true color (24-bit).

20. Jumlah usia honda dan bambang adalah 93. Jika usia honda 3 tahun yang lalu sama dengan usia bambang 8 tahun dari sekarang, berapa usia honda sekarang? sepi...


Usia Honda sekarang adalah 52 tahun

PENJELASAN

Misal:

Usia Honda = xUsia Bambang = y

Diketahui:

Jumlah usia x dan y adalah 93Usia x - 3 = y + 8

Ditanyakan:

Usia x

Jawab:

Jumlah x dan y adalah 93

=> x + y = 93  (I)

x - 3 = y + 8 (II)

=> x = y + 8 + 3

=> x = y + 11 (III)

Substitusikan III ke I

x + y = 93

y + 11 + y = 93

2y + 11 = 93

2y = 93 - 11

2y = 82

y = 82/2

y = 41

Nilai y (usia Bambang) adalah 41

Substitusikan nilai y ke III

x = y + 11

x = 41 + 11

x = 52

Nilai x (usia Honda) adalah 52

PEMBUKTIAN

x + y = 93

52 + 41 = 93

93 = 93  (Benar)

x - 3 = y + 8

52 - 3 = 41 + 8

49 = 49  (Benar)

KESIMPULAN

Usia Honda sekarang adalah 52 tahun
