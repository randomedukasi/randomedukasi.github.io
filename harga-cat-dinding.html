5. Tania mendapatkan proyek membuat mural di sebuah dinding kantor hari pertama Tania membeli 2 kaleng cat latex dan 5 kaleng cat akrilik dengan harga rp420.000 keesokan harinya Tania membeli 1 kaleng cat latex dan 3 kaleng cat akrilik dengan harga rp230.000 harga 1 kaleng cat lateks yang dibeli Tani adalah​


Harga 1 kaleng cat latex yang dibeli Tania adalah Rp 110.000,00. Sedangkan harga 1 kaleng cat akrilik adalah Rp 40.000,00. Nilai tersebut diperoleh dari perhitungan sistem persamaan linier dengan metode eliminasi subtitusi. Persamaan linier dua variabel yang dimaksud adalah:

2x + 5y = 420.000

x + 3y = 230.000

dengan:

x = harga 1 kaleng cat latex

y = harga 1 kaleng cat akrilik

Simak pembahasan berikut.

Pembahasan

Diketahui:

Tania mendapatkan proyek membuat mural di sebuah dinding kantor

Hari pertama Tania membeli 2 kaleng cat latex dan 5 kaleng cat akrilik dengan harga Rp 420.000,00

Keesokan harinya Tania membeli 1 kaleng cat latex dan 3 kaleng cat akrilik dengan harga Rp 230.000,00

Ditanya: Harga 1 kaleng cat latex yang dibeli Tania

Jawab:

Misalkan:

x = harga 1 kaleng cat latex

y = harga 1 kaleng cat akrilik

Maka diperoleh sistem persamaan linier dua variabel sebagai berikut:

Tania membeli 2 kaleng cat latex dan 5 kaleng cat akrilik dengan harga Rp 420.000,00

2x + 5y = 420.000 ..........1)

Tania membeli 1 kaleng cat latex dan 3 kaleng cat akrilik dengan harga Rp 230.000,00

x + 3y = 230.000 .............2)

eliminasi persamaan 1) dan 2)

2x + 5y = 420.000 |× 1| 2x + 5y = 420.000

x + 3y = 230.000   |× 2| 2x + 6y = 460.000   -

                                                -y = -40.000

y = 40.000

Subtitusikan nilai y kedalam persamaan 2)

x + 3y = 230.000

x + 3(40.000) = 230.000

x + 120.000 = 230.000

x = 230.000 - 120.000

x = 110.000

Jadi harga 1 kaleng cat latex yang dibeli Tania adalah Rp 110.000,00. Sedangkan harga 1 kaleng cat akrilik adalah Rp 40.000,00

Harga satu kaleng cat kayu adalah Rp 50.000,00. Soal tersebut merupakan soal tentang persamaan linier.

Penjelasan dengan langkah-langkah

Soal di atas merupakan soal matematika yang membahas tentang persamaan linier. Persamaan linier merupakan suatu persamaan aljabar yang tiap sukunya terdapat suatu konstanta, atau bisa dibilang suatu perkalian konstanta dengan suatu variabel tunggal. Persamaan tersebut dapat dikatakan linier karena sebab hubungan matematis tersebut dapat digambarkan dalam koordinat Kartesius.

Untuk menyelesaikan soal tersebut kita harus mencari masing-masing nilai variabel.

Penyelesaian soal

Diketahui:

x = kaleng cat dindingy = kaleng cat kayu3x + 2y = 595.000 (persamaan i)4x + 7y = 1.010.000 (persamaan ii)

Ditanyakan:

Tentukan harga satu kaleng cat kayu!

Jawab:

3x + 2y = 595.000  dikalikan 44x + 7y = 1.010.000 dikalikan 3

Sehingga

12x + 8y = 2.380.00012x + 21y = 3.030.000

______________________ _

-13y = -650.00013y = 650.000y = 50.000

Jadi, harga satu kaleng cat kayu adalah Rp 50.000,00.
