Pendahuluan

Diketahui :

harga satu sandal dan satu sepatu = 81.000

harga satu sandal dan 2 sepatu = 91.000

Pembahasan

misalkan sandal = x dan sepatu = y

maka model matematika dengan sistem persamaan linear dua variabel untuk masalah di atas adalah

x + y = 81.000

x + 2y = 91.000


dengan matriks, maka model matematika dari permasalahan di atas adalah

[ 1   1  ]      [ x ]  = [ 81.000 ]

[ 1   2 ]      [ y ]     [ 91.000 ]


kita selesaikan SPLDV di atas dengan metode eliminasi

x + y = 81.000

x + 2y = 91.000   -

----------------------

-y = -10.000

y = 10.000

karena x + y = 81.000 maka x = 71.000

maka harga satu sandal = Rp. 71.000 dan harga satu sepatu = Rp. 10.000


Kesimpulan

Jadi, harga sebuah sandal dan harga sebuah sepatu dapat dinyatakan dengan matriks

[ x ]  = [ 71.000 ]

[ y ] [ 10.000 ]

15. Harga sebuah sandal dan sebuah sepatu rp81.000.00 sedangkan harga sebuah sandal dan dan 2 sepatu Rp99.000.00 harga sebuah sandal dan harga sebuah sepatu dapat di tanyakan dengan matriks.....


Harga sebuah sandal dan sebuah sepatu Rp 81.000.00 sedangkan harga sebuah sandal dan dan 2 sepatu Rp 99.000.00 harga sebuah sandal dan harga sebuah sepatu dapat di tanyakan dengan matriks .....

Pembahasan

Kita buat persamaan linearnya terlebih dahulu

Misal sandal = x dan sepatu = y

Harga sebuah sandal dan sebuah sepatu Rp 81.000.00

x + y = 81.000 .............. persamaan 1

Harga sebuah sandal dan dan 2 sepatu Rp 99.000.00

x + 2y = 99.000 .......... persamaan 2

Setelah mengetahui bentuk persamaan linearnya kita buat matriksnya

[tex]\left[\begin{array}{ccc}1&1\\1&2\end{array}\right] \left[\begin{array}{ccc}x\\y\end{array}\right] =\left[\begin{array}{ccc}81.000\\99.000\end{array}\right][/tex]

Penyelesaian

[tex]\left[\begin{array}{ccc}x\\y\end{array}\right] =\frac{1}{(2.1)-(1.1)} \left[\begin{array}{ccc}2&-1\\-1&1\end{array}\right] \left[\begin{array}{ccc}81.000\\99.000\end{array}\right][/tex]

       = [tex]\left[\begin{array}{ccc}2&-1\\-1&1\end{array}\right] \left[\begin{array}{ccc}81.000\\99.000\end{array}\right][/tex]

       = [tex]\left[\begin{array}{ccc}162.000-99.000\\-81.000+99.000\end{array}\right][/tex]

       = [tex]\left[\begin{array}{ccc}63.000\\18.000\end{array}\right][/tex]


Jadi harga sandal = Rp 63.000 dan harga sepatu = Rp 18.000

29. Jika harga 3 pasang sepatu dan 2 pasang sandal adalah Rp1.030.000,00 dan harga 2 pasang sepatu dan 3 pasang sandal adalah Rp920.000,00 maka harga sepasang sepatu dan sepasang sandal adalah​


Jawab:

Harga sepasang sepatu dan sepasang sandal adalah​ Rp. 390.000

Penjelasan dengan langkah-langkah:

Soal diatas membahas mengenai sistem persamaan linier dua variabel (SPLDV). Untuk mengerjaknnya kita bisa buat permisalan dahulu, dilanjutkan dengan pengerjaan melalui metode subtitusi, eliminasi atau campuran dari keduanya.

Penyelesaian Soal:

Diketahui: 3 sepatu + 2 sandal = 1.030.000

                 2 sepatu + 3 sandal = 920.000

Ditanya: 1 sepatu + 1 sandal = ??

Jawab:

Misal:

sepatu = x

sandal = y

maka:

3x+ 2y= 1.030.000

2x+ 3y= 920.000

Saya berikan contoh mengerjakan dengan metode eliminasi:

3x+ 2y= 1.030.000 (x3)⇒ 9x + 6y = 3.090.000

2x+ 3y= 920.000 (x2)⇒   4x + 6y =  1.840.000  -

                                                  5x =  1.250.000

                                                     x = 250.000

3x+ 2y= 1.030.000(x2)⇒ 6x + 4y = 2.060.000

2x+ 3y= 920.000  (x3)⇒ 6x + 9y = 2.760.000   -

                                               -5y =   -700.000

                                                  y =     140.000

Maka harga 1 sepatu dan 1 sendal adalah

x + y = 250.000 + 140.000 = 390.000

Jadi harga sepasang sepatu dan sepasang sandal adalah​ Rp. 390.000
