7. Untuk membuat masker Bu Siti membeli kain katun seluas 6,80 M2. Ia akan membuat masker sebanyak 34 buah berapa luas kain yang dibutuhkan untuk setiap masker?


Untuk membuat masker Bu Siti membeli kain katun seluas 6,80 m². Ia akan membuat masker sebanyak 34 buah. Luas kain yang dibutuhkan untuk setiap masker adalah 0,2 m² atau 2.000 cm².

Pendahuluan

Pecahan adalah bilangan yang dinyatakan dengan bentuk [tex]\displaystyle \frac{a}{b}[/tex] dimana b ≠ 0.

Untuk sembarang pecahan [tex]\displaystyle \frac{a}{b}[/tex] dan [tex]\displaystyle \frac{c}{d}[/tex] dengan b ≠ 0 dan d ≠ 0 berlaku [tex]\boxed {\frac{a}{b} : \frac{c}{d} = \frac{a}{b} \times \frac{d}{c}}[/tex].

Pembahasan

Diketahui :

Luas kain = 6,80 m²Banyak masker yang dibuat = 34 buah

Ditanya :

Luas kain setiap masker?

Penyelesaian :

Menentukan luas kain untuk setiap masker

Luas masker = luas kain : banyak masker

                     = 6,80 m² : 34

                     = [tex]\displaystyle \frac{680}{100} : \frac{34}{1}[/tex]

                     = [tex]\displaystyle \frac{680}{100} \times \frac{1}{34}[/tex]

                     = [tex]\displaystyle \frac{20}{100}[/tex]

                     = [tex]\displaystyle \frac{1}{5}[/tex] m²

                     = 0,2 m²

Kita akan ubah satuan m² ke cm²yaitu 1 m² = 10.000 cm², dengan demikian

0,2 m² = [tex]\displaystyle \frac{1}{5}[/tex] × 10.000 cm²

           = 2.000 cm²

Jadi luas kain yang dibutuhkan untuk setiap masker adalah 0,2 m² atau 2.000 cm².

24. Seorang reseller masker ingin menjual dua macam masker kain, masker kain untuk anak dan masker kainuntuk dewasa. Harga masker untuk anak dan dewasa yang dia peroleh berturut-turut Rp 50.000/box dan Rp65.000/box. Modal yang ia miliki adalah Rp 10.000.000. Ia menjual lagi masker masker tersebut secaraonline dengan harga Rp 100.000/box untuk masker anak dan Rp 120.000/box untuk masker dewasa.Tempat yang ia miliki hanya dapat menampung tidak lebih dari 100 box Agar mencapai keuntunganmaksimum, tentukanlah model matematika supaya dapat dicari banyaknya box masker kain anak dandewasa yang harus dibeli dengan asumsi banyaknya box masker kain untuk anak adalah x dan dewasa y​


Jawab:

keuntungan bersih=Rp, 9.140.000,00

Penjelasan dengan langkah-langkah:

anak awal+dewasa awal=harga total awal

50.000+65.000=115.000

[tex]\frac{modal}{harga total awal}=jumlah maks box yg dapat dibeli[/tex]

[tex]\frac{10.000.000}{115.000} =86/87 box[/tex]

anak akhir+dewasa akhir=harga total akhir

100.000+120.000=220.000

harga total akhir × jumlah maks box yg dpt dibeli=keuntungan kotor

220.000 × 87=19.140.000

keuntungan bersih=keuntungan kotor-modal

19.140.000-10.000.000=9.140.000

[tex]\tt {\bold {\underline {Penjumlahan\:Bilangan\:Bulat}}}[/tex]

Penjumlahan bilangan bulat dapat dilakukan dengan beberapa cara, antara lain menggunakan pasangan kancing positif ( + ), dan pasangan kancing negatif ( - )

➪Penjumlahan pada kancing ( + ) dan kancing ( - )

Catatan :

Pasangan ( + ) dan ( - ) menyatakan bilangan nol

Penjumlahan bilangan nol tidak berarti, artinya mengubah nilai

( tetap )

[tex]\tt {\bold {\underline{Pengurangan\:Bilangan\:Bulat}}}[/tex]

Pengurangan bilangan bulat juga dapat dilakukan dengan beberapa cara seperti pada penjumlahan bilangan bulat

➪Pengurangan pada kancing ( + ) dan ( - )

Sama seperti pada penjumlahan bilangaj bulat, simbol ( + )untuk bilangan positif ( - ) untuk bilangan negatif. Ingat kembali bahwa pasangan ( + ) dan ( - ) menyatakan bilangan nol

Contoh

5 - 3 = 2

4 - 5 = 4 + 0 - 5 = -1

➪Perkalian Bilangan Bulat

Cara mengerjakan bilangan bulat sama seperti perkalian pada bilangan cacah

3 × 2 = 6 ➪Dikurang 2

2 × 2 = 4 ✎Dikurang 2

1 × 2 = 2 ➪Dikurang 2

0 × 2 = 0 ✎Dikurang 2

-1 × 2 = -2 ➪Dikurang 2

-2 × 2= -4 ✎Dikurang 2

-3 × 2 = -6 ➪Dikurang 2

Hasil perkalian dua bilangan bulat bertanda sama adalah bilangan positif

Bilangan positif dikali bilangan positif sama dengan bilangan positif

Bilangan negatif dikali bilangan negatif sama dengan bilangan positif

Hasil perkalian dua bilangan bulat berbeda tanda adalah bilangan bulat negatif

Bilangan positif dikali bilangan negatif sama dengan bilangan negatif

Bilangan negatif dikali bilangan positif sama dengan bilangan negatif

Catatan Rumus

Positif × Positif = Positif

Negatif × Negatif = Positif

Positif × Negatif = Negatif

Negatif × Positif = Negatif

➪Pembagian Bilangan Bulat

Seperti pada perkalian bilangan bulat, hasil pembagian bilangan bulat dapat ditentukan dengan cara sama

Catatan Rumus

Positif ÷ Positif = Positif

Negatif ÷ Negatif = Positif

Positif ÷ Negatif = Negatif

Negatif ÷ Positif = Negatif
