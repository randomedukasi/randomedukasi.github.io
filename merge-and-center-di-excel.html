19. gambar ikon merge and center tidak dapat di gunakan solusi bagaimana?


shortcut : Alt + HMC

detect : coba pilih minimal 2 sel , dengan cara klik sel + shift + left arrow
lalu tekan Alt + H + M + C

problem : jika icon Mc " ( merge and center ) masih belum dapat digunakan ada dua kemungkinan ,
1. berubah fungsi
2. produk office "broken"
3. masa linsensi sudah habis ( biasanya gak ngaruh )

solution
1. klik menu file --> options --> Quick Asscess Toolbar --> all command --> M --> Merge and Center --> Input --> Save --> Ok

2. re-install kembali produk MSOFFICE atau upgrade lisensi

3. cari SN baru !
