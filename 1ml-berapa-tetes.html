12. Suatu larutan penyangga terdiri atas HCl dan NH4OH. agar ph nya berubah dari 10+log8 menjadi 9+log1,6 berapa tetes HCl 10M(1ml=20 tetes) yg harus ditambahkan pada 2L Nh4oh 0,04M?? mohon bantuannya, tolong dengan caranya, terimakasih


saya coba jawab 
yang perlu anda perhatikan adalah bahwa NH4OH mula2 adalah dia berdiri sendiri... jadi HCl belum ditambahkan. kemudian buffer akan terbentuk setelah HCl di tambahkan, paham?
oke kita jalan dari pH terlebih dahulu...
pH awal (sebelum ditambah HCl) = 10 + log 8, kerena pH bawa sehingga pOH awal = 14-pH = 4-log8 paham?
kemudian kita cari [OH-] awal dari pOH awal
so.. [OH-] awal = [tex] 8.10^{-4} [/tex] M

kemudian dg cara sama kita cari [OH-] akhir (buffer)
pH = 9+log 1,6.... pOH = 5-log 1,6...so [OH-] = [tex] 1,6.10^{-5} [/tex]

kemudian kita bisa hitung Kb basa lemahnya/..

[tex] Kb=\frac{ (OH-)^{2} }{M(NH4OH)} [/tex]
[tex]Kb = \frac{ (8.10^{-4}) ^{2} }{0,04} [/tex]
[tex]Kb = 1,6 .10^{-5} [/tex]

kemudian, dari [OH-] akhir, kita bisa mencari perbandingan mol asam : mol basa pada keadaan setelah reaksi (bufffer)

[OH-] akhir = Kb (mol asam sisa)/(mol garam)
[tex] 1,6.10^{-5} [/tex] = [tex] 1,6.10^{-5} [/tex] (mol asam/mol garam)
(mol asam)/(mol garam) = 1

karena perbandingan = 1, maka mol asam = mol  garam ketika keadaan setelah reaksi

kita masukan reaksi, kan kita g tahu HCl tuh maka misalkan ajah dengan x
mol NH4OH awal = M.V = 0,04 M x 2 L = 0,08 mol

______   NH4OH + HCl --> NH4Cl + H2O
mula2    0,08 _____ x
bereaksi x    ______x _____x______x
sisa       x   _______0_____x______x

dari sini ketahuan NH4OH kan...
NH4OH mula - reaksi = sisa
0,08 - x = x
2x = 0,08
jadi x=0,04 mol...

kan mol HCl awal juga x, shingga mol HCl awal = 0,04 mol
so... kita bisa cari volume HCl awal
volume = mol HCl awal/M HCl
Volume HCl awal = 0,04 mol/10 M
volume HCl awal = 0,004 liter = 4 mL

25. Jika setiap 4 tetes larutan jenuh Ag2Co3 mengandung 30.000 partikel ion ag+,hasil kali kelarutan Ag2co3 sebesar....(dianggap 1mL=20tetes)


Diketahui :
1 ml = 20 tetes
Jumlah ion Ag+ (x) = 30000 ion
L = Nomor Avogrado = 6,02 × 10²³ partikel/mol

Ditanya : Ksp ( Hasil kali Kelarutan ) Ag2CO3 ?

Jawab :

n Ag+ :
= x / L
= 30000 ion / 6,02 × 10²³ ion/mol
= 5 × 10^ -20 mol

Reaksi :
Ag2CO3 <-----> 2Ag+ + CO3²-

n Ag2CO3 :
= Koefisien Ag2CO3 / Koefisien Ag+ × n Ag+
= ½ × 5 × 10^-20 mol
= 2,5 × 10^-20 mol

• Mencari Kelarutan Ag2CO3 :

Volume dalam 4 tetes :
= 4 tetes × 1 ml / 20 tetes
= 0,2 ml
= 0,0002 L

Kelarutan Ag2CO3 :
= n / V
= 2,5 × 10^-20 mol / 0,0002 L
= 1,25 × 10^-16


• Ksp Ag2CO3 :

Reaksi :
Ag2CO3 <----> 2Ag+ + CO3²-
____s___________2s______s

Ksp :
= [Ag+] [CO3²-]
= (2s)².s
= 4s².s
= 4s³
= 4(1,25 × 10^-16)³
= 4(1,953125 × 10^-48)
= 7,8125 × 10^-48


Catatan :
( ^ ) : Pangkat

V2= 1liter + V1 = 1000 ml + V1 = 10³ ml + V1
M1= 1 M
M2=10⁻³ M
V1.M1 = V2.M2
V1 x 1 = (10³ + V1) x 10⁻³
V1 =  1 + 10⁻³V1
V1 - 10⁻³V1 = 1
0,999V1 = 1
V1 = 1ml
jumlah tetesan NaOH = 1/(1/20) = 20 tetes
