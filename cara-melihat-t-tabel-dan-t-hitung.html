1. Bila nilai t tabel 5%=0,562 dan nilai t hitung = 0,277.maka H1 diterima ataukah H0 diterima? Jelaskan alasan mu


Jawabannya Ho diterima dan H1 ditolak karena F hitung < F tabel (0,277 < 0,562)

Dalam pengambilan keputusan :

Jika t hitung  < t tabel maka Ho diterima dan HA ditolak, variabel bebas tidak berpengaruh terhadap variabel terikat
Jika t hitung > t tabel, maka Ho ditolak dan HA diterima, variabel bebas memiliki pengaruh terhadap variabel terikat

2. Lihat soal nomor 11 pada gambar. Hitung besarnya T₁ pada soal itu....


Dinamika Gaya
Fisika X

Dik
F = 20 N
m₁ = 4 kg
m₂ = 3 kg
m₃ = 2 kg
m₄ = 1 kg

Dit
Tegangan tali antara m₁ dan m₂ (T₁)

Jaw

Tegangan tali antara tiap benda adalah sama, selama sistim berada di atas permukaan licin (tanpa gesekan)
T₁ = T₂ = T₃ = T₄

Diagram benda bebas untuk sistim
[m₄]T₃→ ←T₃[m₃]T₂→ ←T₂[m₂]T₁→ ←T₁[m₁]F→ 

Step-1
Hitung percepatan sistim (a)

∑F = (∑m)(a)
F - T₁ + T₁ - T₂ + T₂ - T₃ + T₃ = (m₁ + m₂ + m₃ + m₄)(a)
20 = (4 + 3 + 2 + 1)(a)
Diperoleh percepatan sistim sebesar a = 2 m/s²

Step-2
Hitung tegangan T₁

Diagram benda bebas untuk sistim benda m₁
←T₁[m₁]F→ 

∑F = (m₁)(a)
F - T₁ = (m₁)(a)
T₁ = F - (m₁)(a)
T₁ = 20 - (4)(2)
T₁ = 12 N

Jadi besar tegangan T₁ = 12 N

13. hi, kak. bagaimana cara nya menghitung 100/(100+t)? di mana t adalah 11%. Terima kasih


Hi guys, gini cara hitungnya :)

t = 11%

11% = 11/100, maka :

= [tex]\frac{100}{100 + \frac{11}{100} }[/tex]

= [tex]\frac{100}{\frac{10011}{100} }[/tex]

bentuk diatas sama dengan :

= [tex]\frac{100}{1} : \frac{10011}{100}[/tex]

Secara konsep membagi dengan pecahan sama dengan mengalikan dengan kebalikan pembaginya. maka...

= [tex]\frac{100}{1} \times \frac{100}{10011}[/tex]

= [tex]\frac{10000}{10011}[/tex] ✅
