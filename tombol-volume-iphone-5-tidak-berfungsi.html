3. sebutkan 5 tombol fungsi​


Jawaban:

tombol apa dan fungsi bagian mana

1.Tombol Esc

Fungsi tombol keyboard yang letaknya di ujung kiri atas ini dapat digunakan untuk membatalkan sebuah perintah yang sudah dilakukan. Semua perintah yang terlanjur diberikan akan batal jika kamu menekan tombol Esc ini.

2. Tombol Caps Lock

Tombol Capslock yang berada disisi kiri ini memilik fungsi untuk membuat tulisan menjadi huruf besar atau huruf kapital. Saat kamu menekan tombol Caps Lock ini maka ada lampu yang menyala pada keyboard sebagai penanda bahwa Caps Lock sedang aktif.

3. Tombol Tab

Tombol yang berada sedikit dibawah tombol capslock ini memiliki fungsi untuk memindahkan kursor dalam satu jeda atau tabulasi ke arah kanan. Saat kamu mengetik dan menekan tombol Tab, maka kursor akan bergeser ke kanan dengan jarak satu tabulasi.

4. Tombol Backspace

Tombol Backspace yang berada di sisi kanan huruf alphabet ini berfungsi untuk menghapus satu karakter yang berada di depan kursor. Jika kamu menekan tombol Backspace bersamaan dengan tombol Ctrl maka akan menghapus satu kata yang berada di depan kursor.

5. Tombol Delete

Fungsi tombol keyboard yaitu tombol Delete berfungsi untuk menghapus satu karakter yang berada di belakang kursor. Jika kamu menekan tombol Delete dan tombol Ctrl secara bersamaan, maka satu kata akan terhapus.

8. sebutkan fungsi - fungsi dari tombol keyboard berikut tombol caps lock, tombol tab, tombol back pace, tombol delete, tombol entar, tombol esc, tombol home, tombol end, tombol Page up, tombol Page Down​


Jawaban:

Misalkan dipakai pada program pengolah data teks (seperti Ms. Word):

Caps Lock: untuk mengunci huruf kapital.

Tab: untuk memindah posisi kursor ke sebelah kanan 1 tabulasi.

Back space: untuk menghapus 1 karakter di sebelah kiri

Delete : untuk menghapus 1 karakter di sebelah kanan

Enter: untuk mengakhiri 1 baris dan mengawali pengetikan teks di baris baru (di bawahnya).

Esc: untuk menghilangkan tampilan menu yg muncul.

Home: untuk memindah posisi kursor ke awal baris teks.

End: untuk memindah posisi kursor ke akhir baris.

Page Up: untuk memindahkan posisi kursor 1 layar ke atas.

Page Down: untuk memindahkan posisi kursor 1 layar ke bawah.

Penjelasan:

Contoh pemakaian tombol Esc: misalkan mengklik kanan teks (dg mouse) lalu muncul suatu menu, maka tombol ini bisa dipakai untuk menghilangkan menu yg muncul tadi.

11. apa fungsi tombol f1 berikan 5 fungsi​


Jawaban:

Tombol F1 merupakan salah satu tombol fungsi pada keyboard komputer yang umumnya digunakan untuk membuka menu bantuan (help) pada aplikasi atau sistem operasi. Berikut adalah beberapa fungsi lain dari tombol F1:

Bantuan: Tombol F1 paling sering digunakan untuk membuka menu bantuan pada aplikasi atau sistem operasi. Saat tombol F1 ditekan, sistem akan membuka layar bantuan yang berisi informasi tentang cara menggunakan aplikasi atau sistem operasi tersebut.

Pengaturan: Pada beberapa aplikasi atau sistem operasi, tombol F1 juga dapat digunakan untuk membuka menu pengaturan atau konfigurasi. Saat tombol F1 ditekan, sistem akan membuka layar pengaturan yang memungkinkan pengguna untuk mengubah beberapa pengaturan aplikasi atau sistem operasi tersebut.

Fungsi tambahan: Pada beberapa keyboard, tombol F1 dapat berfungsi sebagai tombol fungsi tambahan seperti mengatur volume atau mengubah kecerahan layar.

Debugging: Tombol F1 juga dapat digunakan dalam debugging atau penyelesaian masalah. Saat tombol F1 ditekan, sistem akan membuka layar debugging yang memungkinkan pengguna untuk melihat pesan kesalahan atau log sistem.

Shortcut: Tombol F1 dapat digunakan sebagai shortcut untuk beberapa fungsi pada aplikasi atau sistem operasi tertentu. Misalnya, pada beberapa program pengolah kata, tombol F1 dapat digunakan untuk membuka menu "search and replace" atau "find and replace".

19. Tombol yang berfungsi mengubah fungsi tombol penunjuk menjadi tombol numerik adalah....


Jawaban:

Tombol NumLock

Penjelasan:

Perhatikan contoh gambar keyboard berikut ini (khususnya di bagian yg ditunjukkan tanda panah kuning):

- bila tombol NumLock dalam kondisi aktif, maka tombol penunjuk berubah menjadi tombol numerik.

- bila tombol NumLock dalam kondisi nonaktif, maka tombol numerik berubah menjadi tombol penunjuk.

Contohnya dapat diamati dengan seksama:

- tombol numeric 8 dapat berubah jadi tombol penunjuk panah atas

- tombol numeric 6 dapat berubah jadi tombol penunjuk panah kanan

- tombol numeric 4 dapat berubah jadi tombol penunjuk panah kiri

- tombol numeric 2 dapat berubah jadi tombol penunjuk panah bawah
