4. rumus mencari panjang awal ( Lo ) apa sih??buat belajar uts..


Rumus Panjang awal (Lo) adalah :
Lo = ΔL / (αxΔT)

Keterangan :
Lo = panjang awal (m)
ΔL = pertambahan panjang (m)
α = koefisien muai panjang (/C)
ΔT = perubahan suhu

9. sebatang baja (koefisien mulai panjang 10^-5/ derajat Celsius) panjangnya 100 ㎝ pada suhu 30°Celsius. Bila panjang baja itu 100,1 ㎝, maka suhunya adalah? Rumus : L=Lo(1+α x ΔT) KeteranganLo = panjang awal. L = panjang setelah memuai. ΔL = L - Lo = pertambahan panjang benda akibat pemuaian. α = koefisien muai panjang. ΔT = T - To = kenaikan suhu.


L₀ = 100 cm
T₀ = 30°C
α = 10⁻⁵
L₁ = 100,1 cm
 
 ΔL = L - Lo
 ΔL = Lo(1+α.ΔT)  - Lo
 ΔL = Lo(α.ΔT)  
0,1 = 100(10⁻⁵.ΔT)
 ΔT = 0,1/0,001
 ΔT = 100°C

T = ΔT + T₀
   = 100+30
   = 130°C
