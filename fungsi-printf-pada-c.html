Jawaban:3001 sampai 4999 (3001, 3002, 3003, 3004, ..., 4996, 4997, 4998, 4999).Penjelasan:if (harga >= 8000) printf("Apel");

Artinya: Jika input harga ≥8000 (lebih dari 8000 atau sama dengan 8000), maka program akan mencetak pesan "Apel".

else if (harga >= 5000) printf("Mangga");

Artinya: Jika input harga ≥5000 (lebih dari 50000 atau sama dengan 8000), maka program akan mencetak pesan "Mangga".

else printf("Jeruk");

Untuk else (only) di sini akan mengambil nilai yang tidak disebutkan pada kedua if di atas. Jadi, apabila bilangan mulai dari 5000 dan lebih dari 5000 sudah disebutkan, maka yang belum disebutkan adalah bilangan kurang dari 5000.

Artinya: Jika input harga <5000 (kurang dari 5000), maka program akan mencetak pesan "Jeruk".

if (harga > 3000) printf("Pisang");

Artinya: Jiika input harga >3000 (lebih dari 3000), maka program akan mencetak pesan "Pisang".

else printf("Wortel");

Artinya: Jika input harga ≤3000 (kurang dari atau sama dengan 3000), maka program akan mencetak pesan "Wortel".

Untuk mencetak pesan "JerukPisang", maka ambil harga untuk jeruk yaitu >3000 (3001, 3002, 3003, ...) dan harga untuk pisang yaitu <5000 (..., 4997, 4998, 4999), lalu gabungkan.

Sehingga, nilai harga yang harus diberikan/diinputkan agar program tersebut dapat mencetak pesan "JerukPisang" adalah nilai mulai dari 3001 sampai 4999 (3001, 3002, 3003, 3004, ..., 4996, 4997, 4998, 4999).

Format untuk sebuah fungsi yaitu void, lalu diikuti oleh nama dari fungsinya (misal "b"), diikuti oleh tanda kurung "()" dan kurung kurawal "{}". Jika fungsi tersebut ingin melakukan suatu proses (perhitungan, menampilkan kalimat, dll) maka perintah proses tersebut disimpan pada kurung kurawal.

Untuk memanggil fungsi tersebut maka cukup untuk memanggil namanya saja. Misal ada fungsi

"void e(){

return x ;

}"

lalu kita ingin memanggil fungsi tersebut, maka panggil saja "e();"

30. Apa saja perintah output dalam bahasa C,selain printf,puts,putchar?


Jawaban:

cout

Penjelasan:

Dalam C++, cout merupakan fungsi yang digunakan untuk mencetak output ke peralatan standar (screen) / layar console program. Untuk menggunakan fungsi cout, anda harus menyertakan file header iostream .

Dalam menggunakan fungsi cout, kita menggunakan operator <<. Sama halnya dengan operator >> yang digunakan di dalam fungsi cin (perintah masukan).

Contoh Program:

#include <iostream>  

using namespace std;  

int main ()

{

    int nilai1=100, nilai2=80;

    int jumlah = nilai1 + nilai2;

    // Perintah Keluaran / Output Menggunakan Cout

    //  "\n" Digunakan untuk ganti baris

    cout<<"Nilai 1 : "<<nilai1;

    cout<<"\nNilai 2 : "<<nilai2;

    cout<<"\n\nJumlah Kedua Nilai : "<<jumlah;

    return 0;

}
