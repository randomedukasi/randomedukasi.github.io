3. Panjang pipa tembaga 1 Meter dengan suhu 25°C Koefisien tembaga 0,000017/°C berapakah panjang pipa tembaga pada suhu 50°C ?


ΔL = Lo x α x ΔT

     = 1 x 0.000017 x 25

     = 4,25 x 10^-4 m

L akhir = Lo + ΔL

           = 1 + 4,25 x 10^-4

           = 1,000425 m


4. Sebuah pipa tembaga panjang 1m dengan suhu 15 C. Panjang pipa pada suhu 75 C adalah...(koefisien muai panjang tembaga 0,000017/ C)


Diketahui:

L₀ = 1 m

t₁ = 15°C

t₂ = 75°C

α = 0,000017


Ditanya:

Lt


Jawab:

Lt = L₀ + ΔL

   = L₀ + ( L₀ x α x Δt )

   = 1 + ( 1 x 0,000017 x ( 75 - 15 ))

   = 1 + ( 1 x 0,000017 x 60 )

   = 1 + 0,00102

   = 1,00102 m   =====> kalikan 100

   = 100,102 cm


Semoga membantu

14. Sebuah pipa tembaga mempunyai 1 m pada suhu 25°C.Panjang pipa tembaga pada suhu 175°C adalah TOLONG DIJAWAB SECEPATNYA YA


Lt = Lo (1 + α x Δt)
Lt = panjang akhir (m, cm)
Lo= panjang awal (m, cm)
α = koefisien muai panjang (/°C)
Δt = perbedaan suhu (°C)

Diket :
Lo = 1 m
α tembaga = 0,000017/°C
Δt = 175 - 25 = 150°C

Lt = 1 (1 + 0,000017 x 150)
Lt = 1 (1 + 0,00255)
Lt = 1 . 1,00255
Lt = 1,00255 m

jadi panjang pipa tembaga pada suhu 175°C adalah 1,00255 meter.

19. sebuah pipa tembaga panjang 1m dengan suhu 25derajatcelcius.panjang pipa pada suhu 50 derajatcelcius adalah ????? dengan caranya #PLEASE


Diketahui:
1. Koefesien muai panjang tembaga (α) = 0,000017/°C
2. Panjang mula mula (L₀) = 1 m
3. Suhu awal (t₁) = 25°C
4. Suhu akhir (t₂) = 50°C
Ditanya:
Panjang pipa pada suhu 50°C (L) = ... ?
Jawab:
Panjang seluruhnya (L) :
   L₀ + ΔL
= 1 m + (α × L₀ × (t₂ ₋ t₁)
= 1 m + (0,000017/°C ×1 m ×(50°C - 25°C))
= 1 m + (0,000017/°C × 1 m × 25°C)
= 1 m + 0,000425 m
= 1,000425 m

ΔL = Perubahan panjang
L0 = Panjang awal = 1 m
α = koefisien muai panjang (1,7 x 10-5/°C)
ΔT = Perubahan suhu (175 - 25 = 150°C)

ΔL = 255 x 10-5
