1. Bagaimana cara menulis mind map?


Cara membuat mind mapping adalah:

1. Tentukan tema, dan letakkan tema yang telah ditentukan di tengah.

2.Tentukan cabang utama yang berkaitan dengan tema yang telah ditentukan.

3.Tentukan bagian-bagian dari cabang utama.

4.Gunakan simbol, gambar atau kode. sebagai kreativitasmu.

5.Gunakan garis untuk menghubungkan tema, cabang utama, dan sub cabang (bagian-bagian cabang).

6.Kembangkan mind mapping dengan caramu sendiri. Agar kamu lebih paham, mungkin bisa dengan memberi kode pada bagian-bagian tertentu.

11. Cara menulis alamat dalam bahasa inggris


Cara menulis alamat dalam bahasa Inggris adalah

On the first line, write the name of the recipient include any necessary titles.On the second line, write the address number and name of the street.On the third line, name the town or city.On the fourth line, write the name of the county.On the last line, write the postal code.Pembahasan

How to write an address

1. On the first line, write the name of the recipient include any necessary titles. Example:

Hanif Fais

2. On the second line, write the address number and name of the street.

Example:

Hanif Fais

902 East Kusumanegara St.

3. On the third line, name the town or city.

Example:

Hanif Fais

902 East Kusumanegara St.

Bantul.

4. On the fourth line, write the name of the county.

Example:

Hanif Fais

902 East Kusumanegara St.

Bantul.

Yogyakarta, Indonesia.

5. On the last line, write the postal code.

Example:

Hanif Fais

902 East Kusumanegara St.

Bantul.

Yogyakarta, Indonesia.
