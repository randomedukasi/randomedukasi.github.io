2. Bagaimana cara memperbesar dan memperkecil tabel​


Jawaban:

Untuk memperbesar atau memperkecil tabel pada komputer, Anda dapat melakukan beberapa cara, tergantung pada jenis aplikasi atau program yang Anda gunakan untuk membuka tabel tersebut. Berikut adalah beberapa cara umum untuk memperbesar atau memperkecil tabel pada aplikasi komputer:

Menggunakan tombol zoom: Banyak aplikasi spreadsheet seperti Microsoft Excel atau Google Sheets memiliki tombol zoom yang dapat Anda gunakan untuk memperbesar atau memperkecil tampilan tabel. Anda dapat mencari tombol zoom di bagian bawah atau sudut kanan bawah jendela aplikasi atau pada toolbar. Dengan mengklik tombol zoom, Anda dapat memilih opsi untuk memperkecil atau memperbesar tabel.Menggunakan shortcut keyboard: Beberapa aplikasi spreadsheet memiliki shortcut keyboard yang dapat Anda gunakan untuk memperbesar atau memperkecil tabel. Contohnya, pada Microsoft Excel, Anda dapat menggunakan tombol "Ctrl" dan "+", untuk memperbesar tabel dan "Ctrl" dan "-", untuk memperkecil tabel.Menggunakan fitur scaling: Beberapa aplikasi seperti Microsoft Word memiliki fitur scaling yang memungkinkan Anda memperkecil atau memperbesar tabel dan isi halaman lainnya. Anda dapat mencari fitur scaling di menu "Page Setup" atau "Print Setup" dalam aplikasi tersebut.Mengubah ukuran kolom dan baris: Pada aplikasi spreadsheet, Anda juga dapat mengubah ukuran kolom dan baris untuk memperbesar atau memperkecil tabel. Dengan mengarahkan kursor pada batas antara kolom atau baris, Anda dapat menarik batas tersebut untuk mengubah ukurannya.

13. sebuah foto berukuran 50cm*80cm diperkecil 20%.perbandingan luas foto sebelum dan sesudah diperkecil adalah... Pakai cara


Sebuah foto berukuran 50cm*80cm diperkecil 20%.perbandingan luas foto sebelum dan sesudah diperkecil adalah...
Jawab : 
∵ 50cm + (20%50cm    = 60cm
∵ 80cm + (20%80cm)   = 96cm

∴ luas sebelum 50*80 = 4.000cm²
∴ luas sesudah 60*96 = 5.760cm²
∴ perbandingan 4.000 : 5.760 = 25 :36

26. sebuah foto berukuran 50 cm * 80 cm diperkecil 20% . perbandingan luas foto sebelum dan sesudah diperkecil adalah... Pakai cara:


bismillah
sebelum diperkecil :
50 x 80 =
4000 cm²

setelah diperkecil :
(50 - (50 x 20%) x (80 - (80 x 20%) =
(50 - 10) x (80 - 16) =
40 x 64 =
2560 cm²

perbandingan :
4000 : 2560 =
25 : 16
maaf kalau salah Mapel : Matematika
Bab : Perbandingan
Kelas : 7

Jawaban :

Panjang setelah diperkecil :

80 - ( 80 × 20/100 )
= 80 - 16
= 64 cm

Lebar setelah diperkecil :

50 - ( 50 × 20/100 )
= 50 - 10
= 40 cm

Perbandingan :

Sebelum diperkecil : Setelah diperkecil
p × l : p × l
80 × 50 : 64 × 40
4000 : 2560
25 : 16
