3. Quis 1. Paman melakukan perjalanan dari Surabaya ke Jember . Ayah berangkat pukul 07.00 mengedarai mobil dengan kecepatan rata-rata 60 km/jam. Jarak Surabaya – jember 120 km . Pukul berapa ayah sampai di Jember ?2. Kereta Api Sancaka berangkat dari Surabaya ke Jogjakarta berkecepatan rata-rata 80 km/jam .Jarak Surabaya – Jogjakarta 160 km. Kereta tersebut sampai pukul 09.00 . Pukul berapa kereta Sancaka berangkat dari Surabaya ?​


Jawab:

Penjelasan dengan langkah-langkah:

1. Diketahui:

  Jarak (s) =  120 km

  Kecepatan (v) = 60 km/jam

  Jam berangkat = 07.00

  Ditanya:

  Jam tiba ayah sampai Jember...........??

  Jawab:

   Menghitung waktu tempuh ayah dari Surabaya ke Jember dengan   menggunakan rumus sebagai berikut

   waktu tempuh (t) = [tex]\frac{s}{v}[/tex]

                                 = [tex]\frac{120 km}{60 km/jam}[/tex]

                                 = 2 jam

   Maka ayah akan sampai di Jember pukul 07.00 + 02.00  = 09.00

2. Diketahui:

   Jarak (s) =  160 km

   Kecepatan (v) = 80 km/jam

   Jam tiba = 09.00

   Ditanya:

   Jam kereta Sancaka berangkat...........??

   Jawab:

   Menghitung waktu tempuh kereta Sancaka dari Surabaya ke Jogja dengan   menggunakan rumus sebagai berikut

   waktu tempuh (t) = [tex]\frac{s}{v}[/tex]

                                 = [tex]\frac{160 km}{80 km/jam}[/tex]

                                 = 2 jam

Maka, kereta Sancaka berangkat dari Surabaya pukul 09.00 - 02.00  = 07.00

22. kereta api sancaka kelas eksekutif berangkat dari yogyakarta pukul 23.45 dan tiba di surabaya pukul 04.35. kereta api ekonomi berangkat dari yogyakarta pukul 06.30 dan tiba di surabaya pukul 11.27. selisih waktu tempuh kedua kereta api tersebut adalah?​


Kereta api sancaka kelas eksekutif berangkat dari yogyakarta pukul 23.45 dan tiba di surabaya pukul 04.35. kereta api ekonomi berangkat dari yogyakarta pukul 06.30 dan tiba di surabaya pukul 11.27. selisih waktu tempuh kedua kereta api tersebut adalah?

Penyelsaian  

Waktu merupakan interval atau jarak antara dua atau lebih kejadian atau keaadaan, atau juga bias dikatakan sebagai lamanya suatu kejadian atau keadaan berlangsung.

Diketahui KA Sancaka eksekutif brangkat pukul = 23.45 dan sampai 04.35 sehingga kereta tersebut menghabiskan waktu tempuh perjalanan sebesar 3 jam 50 menit. sedangkan kereta KA Sancaka ekonomi brangkat pukul = 06.30 dan sampai 11.27 sehingga kereta tersebut menghabiskan waktu tempuh perjalanan sebesar 4 jam 57 menit.

Dengan demikian selisish waktu tempuh kedua kereta tersebut adalah  3 jam 50 menit -  4 jam 57 menit =  1 jam 07 menit.

• Pengertian waktu

Waktu merupakan interval atau jarak antara dua atau lebih kejadian atau keaadaan, atau juga bias dikatakan sebagai lamanya suatu kejadian atau keadaan berlangsung.

• Satuan waktu

Berdasarkan pada satuan internasioanal (SI), satuan standar yang dimiliki oleh waktu yaitu detik atau sekon second.

• Konversi satuan waktu  

1 jam = 60 menit

1 menit = 60 detik

1 detik = 1000 milidetik

Demikian lah penjabar dari kakak, semoga dengan ilmu yang sedikiti ini dapat bermanfaat untuk adik adik semua.

29. Dua buah kereta api (KA sancaka dan KA bengawan)bergerak dari 2 stadium dalam arah yang berlawanan kereta Sancaka bergerak dari Stasiun Solo Balapan ke Pasar Senen dengan kelajuan rata-rata 100 km/jam dari arah berlawanan bergerak kereta api Bengawan dengan kelajuan rata-rata 120 km/jam. jika jarak kedua kereta api 550 km, kereta sancaka akan berpapasan dengan kereta bengawan setelah bergerak sejauh


Jawaban:

Kereta Sancaka akan berpapasan dengan kereta Bengawan setelah bergerak sejauh 250 km

Penjelasan:

Diketahui:

v₁ = 100 km/jam

v₂ = 120 km/jam

s = 550 km

Ditanyakan:

s₁ = ?

Penyelesaian:

Pada saat bertemu , maka berlaku:

[tex]s_1 + s_2 = s[/tex]

[tex]v_1t + v_2t = s[/tex]

[tex]100t + 120t = 550[/tex]

[tex]220t = 550[/tex]

[tex]t = 550 \div 220[/tex]

[tex]t = \boxed{\frac{5}{2} \texttt{ jam}}[/tex]

[tex]s_1 = v_1t[/tex]

[tex]s_1 = 100 \times \frac{5}{2}[/tex]

[tex]s_1 = \boxed{250 \texttt{ km}}[/tex]
