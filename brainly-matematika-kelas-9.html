2. Luas = (2xLuas Δ) + Luas Persegi Panjang

            = (2 x 1/2 x a x t) + (p x l)

            = (2 x 1/2 x 6cm x 8cm) + ((20cm - 6cm) x 8cm)

            = 48 cm² + 112 cm²

            = 160 cm²

3. Luas = (2xLuas Δ) + Luas Persegi Panjang

            = (2 x 1/2 x a x t) + (p x l)

            = (2 x 1/2 x 5cm x 12cm) + (18cm x 12cm)

            = 60 cm² + 216 cm²

            = 276 cm²

3. Luas = Luas Pesegi + Luas Persegi Panjang

            = (p x p) + (p x l)

            = (15cm x 15cm) + (30cm x 15cm)

            = 225 cm² + 450 cm²

            = 675 cm²
