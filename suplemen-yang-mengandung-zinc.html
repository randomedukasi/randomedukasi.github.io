5. Seorang pasien rumah sakit untuk pemulihan kesehatan membutuhkan suplemen Fix dan Fit yang mengandung vitamin A, dan vitamin C. Pasien setiap hari membutuhkan minimum 14 mg vitamin A, dan 64 mg vitamin C. Suplemen Fix mengandung 2 mg vitamin A, dan 4 mg vitamin C. Suplemen Fit mengandung 1 mg vitamin A, dan 8 mg vitamin C. Harga setiap kapsul suplemen Fix = Rp1.200,00 dan Fit = Rp1.800,00 jika banyak suplemen Fix yang dibeli = x kapsul dan fit = y kapsul maka pengeluaran minimum untuk membeli suplemen Fix dan Fit setiap hari adalah....


Diketahui: Seorang pasien rumah sakit untuk pemulihan kesehatan membutuhkan suplemen Fix dan Fit yang mengandung vitamin A, dan vitamin C.

Pasien setiap hari membutuhkan minimum 14 mg vitamin A, dan 64 mg vitamin C.

Suplemen Fix mengandung 2 mg vitamin A, dan 4 mg vitamin C.

Suplemen Fit mengandung 1 mg vitamin A, dan 8 mg vitamin C.

Harga setiap kapsul suplemen Fix = Rp1.200,00 dan Fit = Rp1.800,00

Ditanya : Pengeluaran minimum untuk membeli suplemen Fix dan Fit setiap hari.

Penyelesaian :

Misalkan Suplemen Fix = x

Suplemen Fit = y

Perhatikan tabel berikut :

                                    | Vit A    |  Vit B      |

--------------------------------------------------------

Suplemen Fix (x)        |      2      |      4       |

Suplemen Fit (y)         |      1      |       8      |
--------------------------------------------------------

Total                            |     14     |      64    |

 

Model matematikanya:

2x + y ≤ 14

4x + 8y ≤ 64

x ≥ 0, y ≥ 0

 

2x + y ≤ 14

Jika x = 0, maka y = 14 titiknya (0, 14)

Jika y = 0, maka x = 7 titiknya (7, 0)

4x + 8y ≤ 64

Jika x = 0, maka y = 8 titiknya (0, 8)

Jika y = 0, maka x = 16 titikya (16, 0)

Eliminasi kedua pertidaksamaan :

2x + y = 14

4x + 8y = 64   (dibagi 2)

----------------

2x + y = 14

2x + 4y = 32  -

----------------

-3y = -18y

y = 6, x = 4 titiknya (4, 6)

karena harga setiap kapsul suplemen Fix = Rp. 1.200, 00 dana harga setiap kapsul suplemen Fit = Rp. 1.800, 00, maka fungsi objektifnya adalah

z = 1.200 x + 1.800 y

diperoleh beberapa titik :

(7, 0) ----------> 1200 (7) + 0 = 8.400

(0, 14) --------> 0 + 1800 (14) = 28.200

(0, 8) ----------> 0 + 1800 (8) = 14.400

(16, 0) --------> 1200 (16) + 0 = 19.200

(4, 6) ----------> 1200 (4) + 1800 (6) = 15.600

Jadi, pengeluaran minimum untuk membeli suplemen Fix dan Fit setiap hari adalah Rp. 8.400, 00.

kita asumsikan

suplemen ke 1 itu x dan suplemen ke 2 itu y

jadi x = mengandung 5 Vit A dan 3 Vit B

y = mengandung 10 Vit A dan 1 Vit B

maka X = 5A + 3B dan Y = 10A + 1B
